## Summary

Keep open. Current `main` still hardcodes the MiniMax VLM fallback request to `AbortSignal.timeout(60_000)`, so the configured `tools.media.image.timeoutSeconds` / per-model `timeoutSeconds` path is not fully honored for this image-understanding route.

## Evidence
