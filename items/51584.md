## Summary

Keep open. Current `main` still resolves `sandbox explain` channels through the runtime plugin registry only, while the CLI startup path defaults to not preloading plugins. That leaves the reported `(unknown)` / `allowFrom` failure plausible on current `main`, and the existing tests do not cover the slash-session regression path.
