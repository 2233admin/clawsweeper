## Summary

Keep open. Current main still appears to have the gap this PR targets: `gateway:watch` runs the minimal build/runtime-postbuild path, but `runRuntimePostBuild` does not copy bundled hook `HOOK.md` manifests, while hook discovery requires those manifests to discover bundled hooks.

## Best Possible Solution

Keep the PR open and adapt or merge a focused fix that makes the gateway-watch runtime artifact path copy bundled hook `HOOK.md` manifests. Best implementation direction: share the existing copy logic with `scripts/copy-hook-metadata.ts` or call an equivalent helper from `runRuntimePostBuild`, then keep regression coverage in `test/scripts/runtime-postbuild.test.ts` and the build profile tests.
