## Summary

Keep open. Current `main` still classifies any HTML 403 as `auth_html_403` and shows a re-authentication message, so the reported Cloudflare-challenge misdiagnosis is not implemented away. The PR discussion also surfaced unresolved matcher-specificity risks that still need maintainer judgment before merge.
