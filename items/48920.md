## Summary

Keep open. The original March 2026 repro looks obsolete because `isolatedSession` is now implemented and documented on current `main` and in the latest release line, but the broader docs-vs-release drift report still appears unresolved: the docs publish workflow mirrors `docs/**` from `main` directly into the public docs repo, and the checked-in Mintlify config does not show any visible docs version selector or release pinning surface.
