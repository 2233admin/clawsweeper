## Summary

Keep open. Current `main` still writes `systemSent: true` into the persisted session entry during `ensureSkillSnapshot()` before any model response succeeds, and there is no corresponding deferred write in `agent-runner.ts`. The PR still addresses a real gap on `main`, although the late queued-followup review comment should still be verified before merge.
