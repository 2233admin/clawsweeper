## Summary

Keep open. Current main has secure preferred-temp helpers and a narrow raw-tmpdir guard, but it does not implement the requested temp artifact provenance/classification model, cleanup/janitor awareness, durable routing, or runtime-critical temp-backed path detection. This is also not a clean ClawHub close because the request asks for built-in safe defaults and core/tool/runtime guardrails, not just an optional skill workaround.
