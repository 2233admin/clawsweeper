## Summary

Keep this PR open. Current main still contains the Node v24 DEP0190-triggering spawn shape for Windows .cmd/.bat/.com launchers, so the PR is not obsolete. The submitted patch still needs revision because it folds args with a plain space join, adds no regression coverage, and changes security-sensitive shell command construction.

## Best Possible Solution

Keep this PR open and request a revision that routes scripts/ui.js through the existing pnpm runner or the same buildCmdExeCommandLine()-based cmd.exe wrapper. The fix should avoid shell:true with a separate args array, preserve Program Files paths and args with spaces, keep unsafe metacharacter rejection, and add regression tests for .cmd launchers and the final spawn/spawnSync invocation shape.
