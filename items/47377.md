## Summary

Keep open. Current `main` and the latest shipped release still contain the pre-fix behavior: when a tool event starts, `handleAgentEvent` appends the entire cumulative `chatStream` into `chatStreamSegments`, and the chat renderer later emits each segment as its own stream item. That matches the duplication described by the PR and means the proposed delta-based fix is not already implemented on `main`.
