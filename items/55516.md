## Summary

Keep open. Current `main` still routes hook-dispatched agent runs onto the shared `nested` lane, and there is still no `hooks.maxConcurrentRuns` config/schema/docs surface to tune that lane.

## Evidence

  - file: [src/gateway/server/hooks.ts:83](https://github.com/openclaw/openclaw/blob/d3d9b5738f83d92787f57db9186ddcebda60af9a/src/gateway/server/hooks.ts#L83)
