Keep PR #59396 open. Current main does not contain the PR's three production flag propagations or the auth-probe regression test, and the underlying runtime contract still depends on explicit `allowGatewaySubagentBinding` opt-in for embedded runs. This is a plausible focused core bug fix rather than stale cleanup or a duplicate of already-shipped behavior.

Then run the changed command/agent test lanes once dependencies are available.
