## Summary

Keep open. Current main has plugin commands, hooks, generic plugin approval handling, and channel approval capabilities, but it does not implement the requested broader Control UI contribution registry for chat modes, input guards, event classifiers, or status surfaces. This is a concrete missing core/plugin SDK seam, so a ClawHub close would be premature.
