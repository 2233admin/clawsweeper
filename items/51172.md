## Summary

Keep open. Current `main` does not already implement this PR’s change or otherwise fix the reported behavior: `probeGateway()` still disables device identity for unauthenticated loopback probes, unresolved gateway auth SecretRefs still fall back to `{}` plus a warning, and status still treats the gateway as reachable only when `probe.ok === true`.

## Evidence
