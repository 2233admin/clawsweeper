## Summary

Keep open. Current `main` still does not expose the five active-run steer/abort helpers on `plugin-sdk/agent-runtime`, so this PR's requested SDK surface is not implemented there. `main` now has a separate experimental `plugin-sdk/agent-harness` surface, but that is documented for bundled/trusted harness plugins and only re-exports two aliased helpers, not the full five-function channel/plugin-facing API proposed here.
