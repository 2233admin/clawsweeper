## Summary

Keep open. Current `main` still does not add proxy handling to Feishu's Axios-backed HTTP client, so the underlying HTTPS proxy bug from #41820 is not implemented on `main`. The submitted patch is also not ready to auto-close as stale cleanup: review comments identified a real `NO_PROXY` regression and a proxy-cache-key gap, and the PR is now dirty against a substantially changed `extensions/feishu/src/client.ts`.
