## Summary

Keep open. Current `main` has partial mitigations for oversized tool outputs and an opt-in browser `efficient` snapshot mode, but the browser tool still defaults normal AI snapshots to a 40,000-character cap rather than a browser-specific compact/task-aware default. That is not strong evidence that #41949 is already implemented or obsolete.
