## Summary

Keep open. Current main still has the ordinary `before_tool_call` hook and plugin-owned tool registration, but it does not expose the requested trusted pre-plugin tool-policy stage or first-class plugin tool metadata/decorating API. The linked RFC PR #71731 is still open and explicitly docs-only, so it does not supersede or implement this runtime SDK work.
