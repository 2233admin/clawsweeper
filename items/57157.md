## Summary

Current `main` no longer hardcodes `qwen3.3-30B` for command/summary model selection. The shared model-resolution paths now read `cfg.agents.defaults.model` (and agent-specific overrides where applicable) and only fall back to the product default `openai/gpt-5.5`. The same implementation is already present in release `v2026.4.23`.
