## Summary

Current `main` only implements part of this PR's goal. `openclaw doctor` already emits a critical warning for an exposed unauthenticated gateway, but startup logging still lacks the requested warning and the doctor CLI still exits `0`, so this PR is not implemented on `main` and should remain open.
