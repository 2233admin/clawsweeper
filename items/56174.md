## Summary

Keep open. Current `main` still scopes the terminal/onboarding path to macOS, Linux, and Windows via WSL2, while Android is documented only as a companion node app and explicitly not a Gateway host. The only Termux-specific material on `main` is optional auth-helper scripting, which does not satisfy the issue's request for actual OpenClaw availability on Termux.
