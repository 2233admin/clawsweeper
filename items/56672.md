## Summary

Keep open. Current `main` already emits `session_end` on generic session deletion, which covers successful session-mode subagent cleanup, but it still suppresses that hook when a failed spawn already emitted `subagent_ended`, and cron isolated runs still do not emit `session_end` on their exit paths. The PR discussion also surfaced unresolved correctness gaps in the proposed implementation.

## Evidence
