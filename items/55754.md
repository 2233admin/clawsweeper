## Summary

Current `main` already contains targeted fixes for both halves of #55754: builtin memory indexing on macOS + Node 25 now watches the `memory/` directory directly, and QMD now recreates stale managed collections instead of leaving root memory stuck/unbound when collection repair hits an `already exists` conflict.
