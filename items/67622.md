## Summary

Keep open. Current `main` has Control UI attachment plumbing, but it is image-only and does not satisfy this feature request for browser-side file/document upload. The chat picker, paste, and drag/drop paths only accept image MIME types; the preview UI is image-only; and outbound `chat.send` attachments are serialized only as `type: "image"`. The Control UI docs also do not list generic file/document upload as a current capability.
