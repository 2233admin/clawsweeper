## Summary

Keep open. Current main contains the core last-only stripping helper and uses it for selected provider replay policies, but the issue asks for a general/configurable thinking-history policy. Main still defaults `dropThinkingBlocks` to false, applies stripping only when replay policy enables it, explicitly preserves thinking blocks for newer Claude models, and has no `thinkingHistory` config key.
