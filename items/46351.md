## Summary

Current `main` does not implement the requested per-agent Discord webhook override. The agent config surface only exposes generic `identity` fields, while Discord webhook persona delivery is limited to existing Discord thread-binding flows that already have a stored `webhookId`/`webhookToken` on the binding record.

## Evidence
