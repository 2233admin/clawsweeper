## Summary

Current main already covers this PR’s intended fixes: bundled runtime-dependency installs use an OpenClaw-owned npm cache/project-local env, and disabled plugin/channel configs are still honored before default-enabled fallback. The PR’s channel change would also regress the explicit-disable sentinel already preserved on main.
