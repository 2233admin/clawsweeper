## Summary

Keep open. Current main still contains the malformed export HTML script placeholders reported in #43620, while the export generator still performs exact literal replacements for contiguous `{{MARKED_JS}}`, `{{HIGHLIGHT_JS}}`, and `{{JS}}` tokens. A read-only replacement check against current `template.html` leaves the `MARKED_JS`, `HIGHLIGHT_JS`, and `JS` identifiers behind, so the reported export failure remains plausible on main and in the latest release template inspected locally.
