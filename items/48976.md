## Summary

Keep open. Current `main` still has the Feishu quoted-message mention gap this PR describes, so it is not implemented on main or obsolete. The PR may still contain useful work, but the requested behavior is not present in `extensions/feishu/src/send.ts` today and the current tests do not cover quoted-message `@_user_N` normalization.
