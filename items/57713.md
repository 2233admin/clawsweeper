## Summary

Keep open. Current `main` still has a live Docker sandbox fallback that auto-tags raw `debian:bookworm-slim` as `openclaw-sandbox:bookworm-slim`; that image does not include Python, while the sandbox file-mutation helper explicitly requires `python3` or `python` and exits 127 otherwise. The intended build path now installs `python3`, but the missing-image fallback remains inconsistent with that requirement.
