## Summary

Keep #44571 open. Current main has partial mitigation for missing subagent auth through main-agent store merging/inheritance, but the reported onboarding distribution gap is still present for Anthropic setup-token/plugin auth: setup writes profiles only to the selected agentDir, and stale same-profile entries in subagent stores can still override the main store. The linked PR #44644 is a relevant fix attempt, but it is open/unmerged in the provided GitHub context and current main still lacks the requested sync for this path.
