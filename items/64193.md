## Summary

Keep open. Current main still has the original bundled StepFun implementation from #60032, not the onboarding UX refinements in #64193. The PR remains potentially useful because main still lacks paired StepFun onboarding refresh, stale managed model cleanup, the updated auth copy, and the added regression coverage.

## Best Possible Solution
