## Summary

Keep open. Current main still uses one shared `credentials/github-copilot.token.json` cache for GitHub Copilot API tokens and returns any valid cached token without checking which GitHub PAT/auth profile produced it. The related PRs #43659 and #46229 are useful context, but their referenced fix commits are not ancestors of current main, and #43659 remains open while #46229 was closed unmerged.
