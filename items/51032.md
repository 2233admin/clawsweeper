## Summary

Keep open. Current `main` still feeds Feishu group sessions the raw `chat_id` via `GroupSubject`, does not set `ConversationLabel` on the inbound Feishu path, and the label resolver would therefore still show the raw ID. This PR remains actionable rather than auto-closeable.
