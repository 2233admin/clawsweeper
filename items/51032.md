## Summary

Current `main` does not implement this Feishu session-label feature yet. Group chats still populate `GroupSubject` with the raw `chat_id`, and the shared label resolver will surface that value when no `ConversationLabel` is provided, so this PR remains useful cleanup rather than a close candidate.
