## Summary

Keep open. Current `main` still rejects plugin source trees whose `node_modules/openclaw` or `node_modules/.bin/openclaw` symlinks resolve back to the trusted host checkout, while this PR’s narrow allowlist and regression coverage are not on `main`. One CI-timeout tweak from the PR already landed separately, but the plugin-scanner fix did not.
