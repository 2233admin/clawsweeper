## Summary

Keep this PR open. Current main still maps `gemma-*` forward-compat models to the Gemini 3 Flash template without a Gemma-specific context-window patch, so a Gemma model can still inherit the template's 1,048,576-token metadata. The PR is a narrow Google-provider fix with a matching regression test and no observed supply-chain or execution-path expansion.

## Best Possible Solution

Do not auto-close this PR.
