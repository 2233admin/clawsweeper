## Summary

Keep open. Current `main` does not implement the requested global `agents.defaults.reasoningDefault` behavior: the global defaults type/schema omit the key, runtime resolution only honors session and per-agent reasoning defaults, and docs only describe `agents.list[].reasoningDefault`. The PR also has unresolved maintainer feedback to rebase and clarify precedence relative to per-agent defaults.

## Evidence
