## Summary

Current `main` does not implement `agents.defaults.reasoningDefault`. The runtime still resolves reasoning from directive/session/per-agent state only, the global defaults config type and Zod schema do not define the key, and docs still describe only `agents.list[].reasoningDefault`. This PR should stay open unless replaced by another concrete implementation.
