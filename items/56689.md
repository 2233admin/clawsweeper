## Summary

Keep open. Current `main` does not implement LaTeX math rendering in the dashboard chat: the Control UI markdown pipeline uses `markdown-it` plus task-list handling only, the UI package has no KaTeX/math-rendering dependency, and the markdown tests cover general GFM behavior but not math rendering.
