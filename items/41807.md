## Summary

Keep open. Current `main` does not implement `agents.defaults.systemPromptFile`, and the closest existing surfaces are different features: `agents.defaults.systemPromptOverride` replaces the whole prompt with inline text, while CLI backend `systemPromptFileConfig*` only controls how an already-built prompt is handed to specific CLIs.
