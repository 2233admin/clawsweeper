## Summary

Keep open. Current main documents and partially improves the memory model, but it does not implement the requested unified `memory.type` modes or force one consistent write/search behavior across builtin, QMD, and disabled/misconfigured memory. The remaining request is a real bundled memory UX/config design issue, not a stale report or a clear duplicate of the narrower `memory_add` proposal in #70049.

## Evidence
