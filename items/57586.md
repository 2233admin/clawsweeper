## Summary

Keep open. Current `main` still has the reported gateway/app protocol mismatch: the gateway-side node exec path sends `system.run.prepare`, while the macOS Remote node command enum, advertised command list, and invoke dispatcher still only expose/handle `system.run` and not `system.run.prepare`. The latest release `v2026.4.23` matches `main` on the relevant files.
