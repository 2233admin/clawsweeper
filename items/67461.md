## Summary

Keep open. Current `main` still sends Anthropic Vertex traffic through the same upstream streaming path without any OpenClaw-side finalization or abandoned-stream cleanup, and the new tests only cover option shaping and lazy-loading rather than socket/stream lifecycle regression proof.
