## Summary

Close as ClawHub scope. AssemblyAI audio transcription is an optional service-specific media-understanding provider, and current main already has the plugin API, manifest contract, and install/publish path needed for this to live as an external ClawHub/npm plugin. The PR’s extension-shaped implementation is useful as a standalone plugin starting point, but there is no concrete missing core API or protected maintainer signal requiring it to remain open as a bundled core PR.
