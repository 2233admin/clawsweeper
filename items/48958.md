## Summary

Keep open. Current `main` still prepends `<relevant-memories>` context into `effectivePrompt` and then scans that full prompt for image references without stripping memory blocks first, so the phantom-image path described in the PR is still present. The proposed fix is not implemented on `main`, and current tests do not cover this regression.
