## Summary

Keep open. Current `main` does not implement file-backed persistence for the ACP gateway session store that this PR changes, and the submitted PR still carries unresolved high-severity review findings plus unrelated compaction/shared-context scope.

## Evidence

- **ACP gateway session store is still memory-only on main:** `src/acp/session.ts` only exposes `createInMemorySessionStore()` and `defaultAcpSessionStore = createInMemorySessionStore()`. There is no file-backed ACP gateway session store on current `main`.
  - file: [src/acp/session.ts:24](https://github.com/openclaw/openclaw/blob/7e49cc87f9e9cdad47d0572331d7a7a47281a83e/src/acp/session.ts#L24)
  - command: `nl -ba src/acp/session.ts | sed -n '1,210p'`
  - sha: [7e49cc87f9e9](https://github.com/openclaw/openclaw/commit/7e49cc87f9e9cdad47d0572331d7a7a47281a83e)
- **Translator still depends on the in-memory ACP session store:** `AcpGatewayAgent` still injects `defaultAcpSessionStore`, and both `newSession()` and `loadSession()` resolve through that store. That means the ACP protocol-facing session-id to OpenClaw session mapping is still process-local in the code path this PR targets.
  - file: [src/acp/translator.ts:449](https://github.com/openclaw/openclaw/blob/7e49cc87f9e9cdad47d0572331d7a7a47281a83e/src/acp/translator.ts#L449)
  - command: `nl -ba src/acp/translator.ts | sed -n '438,570p'`
  - sha: [7e49cc87f9e9](https://github.com/openclaw/openclaw/commit/7e49cc87f9e9cdad47d0572331d7a7a47281a83e)
- **Main has a different persisted ACP control plane, not the translator persistence proposed here:** Current `main` persists ACP metadata through `readAcpSessionEntry` / `listAcpSessionEntries` / `upsertAcpSessionMeta`, and `AcpSessionManager` resumes persistent ACP runtimes via `resumeSessionId`. That is real persistence, but it lives in the control-plane/session-meta path rather than the ACP gateway session store touched by this PR.
  - file: [src/acp/runtime/session-meta.ts:65](https://github.com/openclaw/openclaw/blob/7e49cc87f9e9cdad47d0572331d7a7a47281a83e/src/acp/runtime/session-meta.ts#L65)

  - file: [docs/tools/acp-agents.md:370](https://github.com/openclaw/openclaw/blob/7e49cc87f9e9cdad47d0572331d7a7a47281a83e/docs/tools/acp-agents.md#L370)
