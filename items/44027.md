## Summary

Keep open. Current `main` still suppresses text-only tool summaries in non-forum group chats regardless of session verbose state, so this PR is not already implemented and the report still matches shipped behavior. The existing tests intentionally preserve that suppression except for forum topics and exec-approval payloads, and the linked DingTalk card PR shows a real downstream consumer for `kind="tool"` delivery.
