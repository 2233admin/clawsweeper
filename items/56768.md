## Summary

Current `main` does not implement this PR's `gateway status` tokenized dashboard URL or JSON redaction, and recent `dashboard` security tests now explicitly forbid logging `#token=` fragments to terminal output. This is not safely auto-closable; it needs maintainer judgment on the security/UX tradeoff.
