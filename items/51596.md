## Summary

Keep open. Current `main` still has the docs inconsistency this PR tries to fix, so it is not stale or implemented already; however, the proposed `hyphen-case`/"keep folder name aligned" wording does not match current `main` behavior, which still accepts underscore skill names and does not enforce folder/frontmatter alignment. Conservative cleanup should not auto-close this PR.
