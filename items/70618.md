## Summary

Keep open. Current `main` has partial mitigations for large bootstrap context, tool-result overflow, and session-store backup rotation, but the exact reported overflow path still exists and can still terminate a run after bounded retries. That is not strong evidence that issue #70618 is already fixed or obsolete.
