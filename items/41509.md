## Summary

Close as obsolete. Current `main` and the latest release use a different chat composer structure (`agent-chat__input` / `agent-chat__toolbar-right`), and the PR’s one-line change targets a legacy `.chat-compose__actions` selector that is no longer referenced by current UI markup.
