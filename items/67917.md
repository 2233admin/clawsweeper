## Summary

Keep open. Current `main` still hard-blocks `openclaw models auth login --provider openai-codex` on TLS-cert preflight failures, which matches the reported `UNABLE_TO_GET_ISSUER_CERT_LOCALLY` symptom, and the latest release still contains that behavior. That is not strong enough evidence to close as implemented or obsolete.
