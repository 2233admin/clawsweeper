## Summary

Keep open. Current main already narrows the skills watcher to `SKILL.md` targets, so that half of the PR is effectively superseded, but the memory side is not implemented on main: `extensions/memory-core` still watches recursive `**/*.md` and multimodal globs without a depth cap, and a second QMD memory watcher also remains unbounded. The PR is therefore stale in paths but still points at a real, unresolved cleanup area.
