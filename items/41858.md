## Summary

Keep open. Current `main` partially supersedes the skills half of the PR by narrowing that watcher to `SKILL.md` globs, but the memory watcher still recursively watches `memory/` and `extraPaths` without any chokidar `depth` limit. That means the PR's core request is not implemented on `main`, and the branch is also stale because the memory watcher now lives under `extensions/memory-core/` rather than the old `src/memory/` path.
