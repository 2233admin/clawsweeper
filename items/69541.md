## Summary

Keep open. Current `main` still relies on per-consumer hardcoded stripping for plugin-injected prompt content, and there is no manifest-level `injectedContextTags`-style contract or shared registry implementing this RFC. The request also fits OpenClaw core/plugin-SDK scope rather than ClawHub-only scope.
