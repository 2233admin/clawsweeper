## Summary

Current `main` already routes `skills` command output through stdout, and command-level tests explicitly cover `skills list --json`, `skills info --json`, and `skills check --json` using stdout rather than `defaultRuntime.log`. The reported stderr routing bug is therefore already resolved on `main`.
