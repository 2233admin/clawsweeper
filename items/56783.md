## Summary

Keep open. Current `main` still truncates Control UI chat history by a fixed 200-message window in `ui/src/ui/chat/build-chat-items.ts` and does not implement the PR's character-budget guard. Large tool-result payloads are still rendered from raw `content` strings, so the reported freeze path remains plausible.

## Evidence

- **History logic moved out of `views/chat.ts`:** `renderChat` now delegates history assembly to `buildChatItems`, so the PR's original patch target is stale against current `main`.
  - file: [ui/src/ui/views/chat.ts:799](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/ui/src/ui/views/chat.ts#L799)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Current main is still count-capped only:** `buildChatItems` computes `historyStart` as `Math.max(0, history.length - CHAT_HISTORY_RENDER_LIMIT)` and shows a fixed `Showing last 200 messages ...` notice. There is no character-budget accounting here.
  - file: [ui/src/ui/chat/build-chat-items.ts:201](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/ui/src/ui/chat/build-chat-items.ts#L201)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Tool messages are filtered after window selection:** When `showToolCalls` is false, tool-result messages are skipped only after the raw history window is chosen, so the selection step still does not reason about visible payload size.
  - file: [ui/src/ui/chat/build-chat-items.ts:231](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/ui/src/ui/chat/build-chat-items.ts#L231)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Rendered tool payloads still come from raw `content`:** `extractToolText` falls back to `item.content`, so large tool-result payload strings can still reach the DOM/rendering path.
  - file: [ui/src/ui/chat/tool-cards.ts:44](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/ui/src/ui/chat/tool-cards.ts#L44)
