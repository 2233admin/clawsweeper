## Summary

Keep open. Current main still appears to have the reported false positive: the audit's known-command set is built from per-platform allowlists only, while dangerous-but-valid node commands like `camera.snap`, `screen.record`, `calendar.add`, `contacts.add`, and `sms.send` are defined separately and are not part of those default allowlists unless explicitly added to `gateway.nodes.allowCommands`. That means `gateway.nodes.denyCommands` can still flag valid dangerous node commands as unknown.
