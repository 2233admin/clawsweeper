## Summary

Keep open. Current `main` does not contain the PR's `memory-lancedb` fix: `extensions/memory-lancedb/index.ts` still sends embeddings requests without `encoding_format`, and the corresponding test still asserts a payload with only `model`, `input`, and optional `dimensions`.
