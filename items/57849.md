## Summary

Keep open. Current `main` still sends `agent_end` and `before_compaction` hook payloads from in-memory session state or omits `messages` entirely, so the provenance-preserving behavior described in PR #57849 is not implemented on `main`. Only the `before_reset` path already reads transcript messages.
