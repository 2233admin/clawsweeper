## Summary

Keep open. Current `main` has partial supply-chain/security work already landed, but the specific OpenSSF/tooling request in #26137 is not implemented end-to-end: `SECURITY.md`, CodeQL, dependency audit, and npm provenance exist, while repo-visible pieces like Scorecard, Dependency Review, SECURITY-INSIGHTS, Allstar, SHA-pinned actions across workflows, and the requested README badges are still absent.
