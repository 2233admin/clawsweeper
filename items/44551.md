## Summary

Keep open. Current main already has broad Control UI Chinese localization, including zh-CN/zh-TW loading, browser-locale detection, a Gateway Access language picker, translated navigation, and tests for shipped locale alignment. However, the issue specifically calls out Audit/Event Log operation names such as exec/read/cron/subagents/browser, and current main still renders gateway event names directly from raw event identifiers with no localized operation-label map. That remaining requested capability is meaningful and no narrower canonical follow-up was provided.
