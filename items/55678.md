## Summary

Current `main` still does not implement DashScope/Qwen provider-usage support. The provider-usage allowlist and wire types exclude `qwen`/DashScope, the bundled Qwen provider does not register usage hooks, and the docs still list usage-window support only for Anthropic, GitHub Copilot, Gemini CLI, OpenAI Codex, MiniMax, Xiaomi, and z.ai. This remains an actionable repo issue rather than a close candidate.
