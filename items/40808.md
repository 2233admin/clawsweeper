Keep PR #40808 open. Current main does not implement or document `sessions_send_concurrent`, and the gateway still configures only Cron/Main/Subagent lane concurrency, not `CommandLane.Nested`. The PR is not merged, not superseded by the provided related items, and targets a plausible core agents/gateway capability rather than a ClawHub-only plugin. It still needs maintainer review and stronger validation before merge.

## Best Possible Solution
