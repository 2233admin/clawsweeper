## Summary

Keep open. Current `main` still does not thread `sessionKey` into the system-prompt runtime info or `Runtime:` line, so this PR is not already implemented. There is overlapping current behavior because session tools already accept `sessionKey="current"`, but that only partially covers the motivation and is not strong enough evidence to auto-close the PR as obsolete.
