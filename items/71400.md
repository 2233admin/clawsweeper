## Summary

Current `main` does not implement the proposed `session.reset.mode: "adaptive"` behavior. The repo still supports only `daily` and `idle`, documents OR semantics when both daily and idle are configured, and the PR discussion surfaced unresolved follow-up points (missing symmetric test coverage and misleading help text). This needs maintainer review rather than conservative auto-close.
