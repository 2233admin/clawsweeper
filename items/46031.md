## Summary

Keep open. Current `main` appears to have fixed the runtime model-auth path by routing provider selection through `resolveAuthProfileOrder`, but the implicit GitHub Copilot provider-discovery path still walks raw stored profile order and a current test explicitly expects "first profile wins" behavior. That is too close to the reported bug to close conservatively as implemented.
