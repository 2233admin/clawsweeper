## Summary

Current `main` does not contain the PR’s proposed change, and the available code/tests do not give strong enough evidence that the PR is obsolete. `GatewayClient` still defaults omitted scopes to `[`operator.admin`]`, while gateway authorization already treats `operator.admin` as sufficient for read-class methods, so this looks like a small semantic cleanup rather than something already implemented or clearly invalid.
