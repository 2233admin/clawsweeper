## Summary

Keep open. Current `main` still hard-codes user-visible incomplete-turn fallback text and does not expose a config knob to silently swallow those responses. The newly landed main-branch changes only suppress the fallback when user-visible delivery already happened; they do not implement the requested configurable empty-turn delivery policy. A maintainer comment on April 25, 2026 explicitly confirms that distinction and says this feature request remains valid.
