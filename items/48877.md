## Summary

Keep open. Current `main` at `01bc49c88c0cef9807b15df4d7486a0c5fe4afbd` still supports Telegram `customCommands` only as flat BotFather menu entries (`command` + `description`). The PR’s requested `menus`/`routes` config surface, custom menu command handlers, and callback-based menu navigation are not present on `main`.
