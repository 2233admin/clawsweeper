## Summary

Keep open. Current `main` still only accepts legacy Anthropic setup tokens with the `sk-ant-oat01-` prefix and minimum length 80, so the underlying request is not implemented. This PR targets a real gap, but the proposed condition would also let too-short legacy-prefixed tokens pass validation and it does not add coverage for the new token format.
