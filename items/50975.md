## Summary

Keep open. Current `main` does not implement this PR’s requested behavior: Anthropic setup tokens still must start with `sk-ant-oat01-` and be at least 80 characters, including in the latest release code. The PR is also stale as written because the validation logic now lives in `src/plugins/provider-auth-token.ts`, and the proposed condition would regress validation by allowing too-short legacy-prefixed tokens.
