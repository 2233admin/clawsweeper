## Summary

Current main already carries the Docker release pipeline changes that remove the v2026.3.13-1 Blacksmith-based amd64 publish path and add manual version backfills. That implementation is present in shipped release v2026.3.22, so the reported non-AVX2 Docker regression is already addressed on main well enough to close.
