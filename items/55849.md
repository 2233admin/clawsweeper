## Summary

Keep open. Current `main` still writes session transcript headers without `sessionKey` or `agentId`, the docs still describe headers as only carrying `id`/`cwd`/`timestamp`/optional `parentSession`, and the only linked implementation attempt (#56082) was closed unmerged.
