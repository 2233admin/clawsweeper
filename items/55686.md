## Summary

Keep open. Current `main` still implements `/reasoning` status as plain text (`Current reasoning level ...` plus `Options: ...`), and the Feishu plugin still degrades presentation blocks into markdown text instead of interactive card controls. I found no Feishu command-channel UI builders analogous to Telegram’s clickable command/model UI hooks.
