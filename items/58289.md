## Summary

Keep open. Current main still returns early on the empty-provider `skip` plan without creating or bootstrapping `models.json`, `agents.create` still does not create the file, and the linked fix work is not merged onto main.

## Best Possible Solution

Keep the issue open and land a focused fix in `src/agents/models-config.ts`: when `planOpenClawModelsJson` returns `skip` for a missing non-main agent `models.json`, either bootstrap once from the main agent's `models.json` with the guard rails described in PR #58391/#58373, or fail early with actionable guidance. Add regression coverage for missing target, existing target preservation, missing/empty main source, idempotency, and write/read error handling.
