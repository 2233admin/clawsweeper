## Summary

Keep open. Current `main` does not implement configurable Control UI tab titles: `gateway.controlUi.title` is absent from the gateway config type/schema, absent from the Control UI bootstrap contract/server payload, and the client bootstrap loader never updates `document.title` from config.

## Evidence

- **Gateway config type has no `title` field:** `GatewayControlUiConfig` defines `enabled`, `basePath`, `root`, and later security/embed options, but no `title` property on current `main`.
