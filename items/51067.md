## Summary

Keep open. Current `main` still hardcodes the Control UI tab title and does not expose any `gateway.controlUi.title` config or bootstrap field, so the requested feature is not implemented on `main`. The PR remains directionally useful, but the prior review note about generated schema drift is still valid: current schema/help artifacts do not include this option either.
