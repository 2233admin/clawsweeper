## Summary

Keep open. Current `main` does not implement a per-message print action in Control UI: assistant message actions are limited to expand/copy, the shared action helper is clipboard-only, and a repo search found no `window.print` or equivalent chat-print flow. This also reads as core Control UI ergonomics work rather than a ClawHub/plugin-scope item.
