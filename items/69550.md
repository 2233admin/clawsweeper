## Summary

Keep open. Current `main` still renders assistant chat bubbles without the `has-copy` class, so the existing CSS that reserves space for copy/expand actions is still dead code. The issue report matches the older shipped code path, and the linked PR #69655 proposed this exact fix but that change is not present on `main`.

## Evidence
