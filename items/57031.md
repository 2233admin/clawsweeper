## Summary

Keep open. Current `main` still silently swallows both queued writer setup and append failures, and the latest release `v2026.4.23` ships the same behavior. The report’s severity is lower than first described because callers are diagnostic-only, but the underlying issue is still present and not implemented on `main`.
