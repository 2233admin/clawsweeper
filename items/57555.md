## Summary

Keep open. Current `main` still ships the bundled `summarize` skill as brew-only and not OS-scoped, so the Linux/Docker support problem remains. The exact silent-toggle symptom looks partially improved in current UI code, but not enough to close the issue because the packaged skill/install experience is still broken for this report's environment.

## Evidence

- **Bundled summarize skill is still brew-only:** `skills/summarize/SKILL.md` still requires the `summarize` binary and exposes exactly one installer: Homebrew formula `steipete/tap/summarize`. There is no Linux installer and no `os` restriction marking the skill as macOS-only.
