## Summary

Keep #57555 open. Current main still ships the bundled `summarize` skill with a required `summarize` binary and only the `steipete/tap/summarize` Homebrew installer, with no Linux/apt/download alternative and no macOS-only OS gate. The UI now has some status/detail plumbing for missing requirements and install failures, but enabling a skill only writes `skills.entries.<skill>.enabled`; it does not validate that the required binary is present, so the specific bundled-skill/platform problem remains. The related #14593 remains open for the broader Docker/Homebrew skill-install class, but this report adds a concrete `summarize` formula/removal and Control UI feedback case rather than being fully superseded.

## Best Possible Solution

Keep the issue open and fix the bundled skill surface directly: update or remove `skills/summarize/SKILL.md` so it no longer advertises a deleted brew formula on unsupported hosts, add an explicit OS gate or maintained Linux/download installer if available, and make the Control UI enable/install flow surface missing required binaries clearly when a skill remains ineligible after enabling.
