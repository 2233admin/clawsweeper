## Summary

Keep open. Current `main` still maps adaptive Anthropic `minimal`/`low` reasoning to `output_config.effort: "low"` for Claude Opus 4.7, and current failover/auth-profile code still turns 400/422 schema rejections into profile-wide cooldowns rather than a model-scoped or non-auth cooldown path.
