## Summary

Keep open. Current main still ignores per-agent `agents.list[].reasoningDefault` in the Telegram and Feishu channel preview paths, while the config contract and docs already expose that per-agent setting. PR #71817 is a focused, still-relevant fix with tests and active review discussion, not conservative cleanup material.

## Best Possible Solution

Keep #71817 open for maintainer and channel-owner review. The best path is a focused merge or small update that makes Telegram and Feishu preview resolution fall back to `agents.list[].reasoningDefault`, preserves explicit session overrides, keeps Feishu stream-only preview semantics, and leaves broader global/per-model reasoning defaults to their canonical follow-up issues.
