## Summary

Keep open. Current `main` still has no Control UI `tasks` tab, `/tasks` route, or `task-board` view, so this PR is not implemented on `main`. At the same time, current docs and CLI now treat Background Tasks and Task Flow as first-class product concepts, which makes the area still relevant rather than obsolete. The PR review thread also contains unresolved correctness concerns, so conservative cleanup should not auto-close it.
