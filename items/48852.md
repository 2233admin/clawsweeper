## Summary

Keep open. Current `main` does not implement remote Chrome DevTools MCP via configurable MCP URL. The browser stack intentionally separates `existing-session` Chrome MCP from remote browser access: `existing-session` is local-or-node-host auto-connect only, while remote browser access still uses raw CDP or a node host.
