## Summary

Keep open. Current main still has the ACP override gap this PR describes: the `runtime="acp"` `sessions_spawn` path does not forward `requesterAgentIdOverride`, and ACP requester/heartbeat resolution still derives the requester only from the parent session key. This is not implemented on `main` at `d3d9b5738f83d92787f57db9186ddcebda60af9a`.
