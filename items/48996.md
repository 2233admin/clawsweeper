## Summary

Keep open. Current `main` does not implement Mermaid rendering in WebChat/Control UI chat. Message markdown still flows through the generic markdown renderers, which render fenced blocks as plain code/JSON only, and repo-wide `mermaid` matches are docs-only. The issue also has an active linked PR head (`refs/pull/68061/head`), so this looks like unfinished work rather than something already shipped.
