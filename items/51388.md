## Summary

Keep open. Current `main` at `42514156e03c53dec6be6c581069d8c5b2d4cd42` still drops subagent `tools.alsoAllow` entries before optional plugin tools are resolved, so the PR addresses a live bug rather than an already-shipped fix.

  - file: [src/agents/pi-tools.policy.ts:81](https://github.com/openclaw/openclaw/blob/42514156e03c53dec6be6c581069d8c5b2d4cd42/src/agents/pi-tools.policy.ts#L81)
