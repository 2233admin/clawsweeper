## Summary

## Close Comment

Closing from a Codex review: current `main` already covers the behavior this PR was trying to add. The gateway entrypoint now runs all handlers inside `withPluginRuntimeGatewayRequestScope(...)` in `src/gateway/server-methods.ts:162-174`, and the `agent` path starts its detached `agentCommandFromIngress(...)` work from inside that scoped handler in `src/gateway/server-methods/agent.ts:229-263`.
