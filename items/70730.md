## Summary

Keep open. Current `main` does not already implement this PR, and the proposed `scanDir` reorder conflicts with existing bundled-plugin override contracts that current tests and channel-loader plumbing explicitly enforce. The RSS report may still be real, but this specific patch needs maintainer judgment or a different fix rather than conservative cleanup closure.
