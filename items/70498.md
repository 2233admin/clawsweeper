## Summary

Keep PR #70498 open. Current main still lacks the Zalo Personal channel documentation for `ZALOUSER_PROFILE` and `ZCA_PROFILE`, while the bundled plugin manifest, implementation, and tests all show those env vars are real supported behavior. The earlier PR #69643 was closed unmerged, so it does not supersede this PR.

## Best Possible Solution

Keep the PR open and land or reapply the focused docs addition to `docs/channels/zalouser.md`, preferably documenting profile names, resolution order, and the multi-account caveat against the public `/channels/zalouser` docs page.
