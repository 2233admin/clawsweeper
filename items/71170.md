## Summary

Keep open. Current `main` still hardcodes the reminder-guard note around core cron state only, and it exposes no `agents.defaults.reminderGuard` or per-agent `reminderGuard` config seam. The PR is recent, concrete, and addresses a gap that is not already implemented on `main`.
