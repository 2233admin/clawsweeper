## Summary

Keep open. Current `main` does not implement the core exec fixes from this PR: child exec env sanitization still does not scrub provider auth env vars, two-phase approvals still expire immediately when no approver route exists, and approval-pending copy still only points users at `/approve` instead of Control UI guidance. The PR also bundles unrelated `test/test-env.ts` changes that match the existing review concerns, so there is no strong basis to auto-close it as already implemented or obsolete.
