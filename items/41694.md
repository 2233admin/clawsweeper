## Summary

Keep open. Current `main` does not contain the proposed `defaultTargetKind` API or the related prefix/session-routing fixes, and the existing docs already position `inferTargetChatType` as the plugin seam for pre-directory target-kind decisions. This looks like plausible useful work, but it needs maintainer judgment on API shape rather than conservative auto-close.
