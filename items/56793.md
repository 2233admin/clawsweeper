## Summary

Keep open. Current `main` still documents the managed `openclaw` profile as isolated from the personal browser, supports `existing-session` attach via `userDataDir`, and exposes only list/start/stop/reset/create/delete profile mutations. I found no explicit import/clone CLI command, API route, or matching history proving this request was implemented.
