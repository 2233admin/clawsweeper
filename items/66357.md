## Summary

Keep open. Current `main` already uses `web_search`/`web_fetch` instead of the reported `ollama_web_*` names, including in the latest release tag `v2026.4.23`, but the zh-CN completion request is still not satisfied: several Control UI surfaces named in the report still contain hardcoded English strings outside the locale tables.

## Evidence

- **Web tool names already renamed on current main:** The Control UI fallback tool catalog exposes `web_search` and `web_fetch`, not `ollama_web_search` / `ollama_web_fetch`. `git show v2026.4.23:ui/src/ui/views/agents-utils.ts` shows the same names in the latest shipped release.
  - file: [ui/src/ui/views/agents-utils.ts:58](https://github.com/openclaw/openclaw/blob/9a0b26cafca76d267dd231b5eaceadb071ec749d/ui/src/ui/views/agents-utils.ts#L58)
