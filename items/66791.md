## Summary

Keep this issue open. Current main at edbab0e2db7f450f4d63f9700289ae1ba838a419 still only exposes Discord scheduled-event list/create; the requested event-edit and event-delete lifecycle actions are absent from the shared message-tool contract, Discord discovery, Discord action parsing, runtime dispatch, REST helpers, CLI surface, docs, and tests. The reporter's follow-up accurately identifies the existing bundled Discord implementation path and the missing pieces.

## Best Possible Solution

Keep this issue open and implement the missing lifecycle actions as a coordinated shared-contract plus bundled Discord plugin change: add event-edit and event-delete to the shared action list, schema, display metadata, outbound target metadata, CLI surface, and Discord discovery; add Discord API mapping and runtime actions; add REST helpers for scheduled-event fetch/update/delete; merge existing event state safely before PATCH; add focused Discord unit tests; and update the public Discord docs at https://docs.openclaw.ai/channels/discord.
