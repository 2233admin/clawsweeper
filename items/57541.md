## Summary

Keep open. Current main has not implemented the requested Feishu docs cleanup: the English Feishu doc still shows `defaultAccount: "main"` with `accounts.main`, while the same page and runtime default say the default account id is `default`. The PR is still useful, but it needs rework for current main because `docs/zh-CN/channels/feishu.md` no longer exists and repo docs policy says localized docs are generated outside this repo.

## Best Possible Solution

Keep the PR open, but request a current-main update: change the English Feishu source doc so the account-key example and `defaultAccount` reference are consistent with `DEFAULT_ACCOUNT_ID = "default"`, keep or add the focused schema test for arbitrary account keys when `defaultAccount` is omitted, and drop the generated/localized `docs/zh-CN` edits.
