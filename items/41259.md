## Summary

Keep open. Current `main` still swallows `mkdirSync(..., { recursive: true })` errors in the memory host `ensureDir`, the latest release still ships that code, and the memory DB open path still depends on this helper before opening sqlite. I did not find evidence that the PR’s behavior is already implemented or obsolete.
