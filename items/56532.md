Keep this PR open. Current main still lacks configurable memory-lancedb embedding timeout/retry settings, and this PR remains the active implementation candidate. It should not be closed as implemented or obsolete, but it needs maintainer review for current-main drift and the security concern around configurable embedding base URLs before merge.
