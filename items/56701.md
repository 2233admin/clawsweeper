## Summary

Current `main` does not implement per-agent TTS configuration. The agent config schema rejects `agents.list[].tts`, the TTS runtime resolves only `cfg.messages.tts`, and the `tts` tool exposes no per-call voice override fields, so the reported limitation still exists.

## Evidence

- **Agent schema has no per-agent TTS field:** `AgentEntrySchema` defines per-agent keys such as `model`, `skills`, `identity`, `tools`, and `runtime`, then ends with `.strict()` without any `tts` property.
  - file: [src/config/zod-schema.agent-runtime.ts:812](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/config/zod-schema.agent-runtime.ts#L812)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **TTS runtime reads only global messages.tts:** `resolveTtsConfig` builds the effective TTS config from `cfg.messages?.tts ?? {}` and returns provider settings from that global block only.
  - file: [extensions/speech-core/src/tts.ts:313](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/extensions/speech-core/src/tts.ts#L313)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Agent tts tool has no voice override input:** The built-in `tts` tool schema accepts only `text`, optional `channel`, and optional `timeoutMs`; execution passes the config through to `textToSpeech` without any voice or provider override argument.
  - file: [src/agents/tools/tts-tool.ts:10](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/src/agents/tools/tts-tool.ts#L10)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **Docs still document TTS as global messages.tts:** The gateway config docs describe TTS under `messages.tts` and the per-agent `agents.list` section does not document any `tts` override.
  - file: [docs/gateway/config-agents.md:1245](https://github.com/openclaw/openclaw/blob/6b618f063516aa9ba85903aa4565c5a204c055b2/docs/gateway/config-agents.md#L1245)
  - sha: [6b618f063516](https://github.com/openclaw/openclaw/commit/6b618f063516aa9ba85903aa4565c5a204c055b2)
- **No matching implementation surfaced in current history search:** A repository history search for commit messages mentioning `per-agent TTS` / `Support per-agent TTS voice overrides` returned no matching commits on the checked-out `main` history.
  - command: `git log --oneline --all --grep='add per-agent TTS voice configuration\|Support per-agent TTS voice overrides\|per-agent TTS'`
