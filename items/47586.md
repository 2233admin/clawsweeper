## Summary

Keep open. Current `main` still matches the report: the reminder guard regex still treats bare `remember` as a reminder commitment, the suppression path still keys off any enabled cron in the same session rather than a cron created in the current turn, and the shipped `v2026.4.22` file is unchanged in this area.
