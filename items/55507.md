## Summary

Keep open. Current `main` still carries `memorySearch.outputDimensionality` through config resolution, but the live OpenAI memory-embedding path does not forward that value into the actual embeddings request body. The PR’s core bug is not already implemented on `main`, and the branch likely needs a refresh because the touched code moved since the PR was opened.
