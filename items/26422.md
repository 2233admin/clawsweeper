## Summary

Keep open. Current `main` no longer matches the issue's broadest claim because generic outbound delivery and Telegram reply delivery now run `message_sending`, but the hook still is not universal: WhatsApp, Slack, Signal, and iMessage reply paths continue to send directly without invoking the hook, while docs still describe `message_sending` as an outbound message hook.
