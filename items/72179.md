## Summary

Keep #72179 open. Current main still lacks the public `cron_lifecycle` plugin hook, hook runner method, gateway cron start/finish hook emission, and docs. The PR is a focused implementation for the remaining #8209 use case, while the older #8855 is closed unmerged and superseded by this PR.

## Best Possible Solution

Keep #72179 open as the canonical implementation path for the #8209 cron audit/telemetry use case. Maintainers should review the typed `cron_lifecycle` plugin contract, ensure gateway emission covers `started` and `finished` for both `agentTurn` and `systemEvent`, keep the focused tests and public docs, then merge or request targeted changes.
