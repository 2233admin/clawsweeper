## Summary

Keep open. Current main already ships substantial browser-control functionality, including a bundled `browser` plugin with an OpenClaw-managed Chromium profile, Chrome MCP attach mode, gateway/CLI/tool integration, and cookie/storage/session surfaces. But the repo still explicitly documents WebChat as having "no embedded browser," so the Atlas/Comet-style native integrated browser shell requested in #24996 is not fully implemented. The request also fits current core direction in `VISION.md` (computer-use and companion apps), so this is not a clear ClawHub-only close.
