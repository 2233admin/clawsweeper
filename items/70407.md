## Summary

Keep open. Current main still requires `plugins.entries.memory-core.config.dreaming.storage` to be an object, and there is no runtime or doctor compatibility migration in-tree for legacy string values. Read-only flows have been softened, but the underlying upgraded config remains invalid for strict commands/runtime loads.
