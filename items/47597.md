## Summary

Keep open. Current main still rejects `sessions_spawn({ runtime: "subagent", streamTo: "parent" })` by design, has an explicit regression test for that rejection, and documents `streamTo` only for ACP. I did not find evidence on the checked-out `main` (0e7250f37bf347c72790abf2a6692d5ec84e5ea1) that subagent parent-streaming was implemented.
