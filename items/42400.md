## Summary

Keep open. Current `main` does not implement the PR’s `neverReply` behavior, and the plugin-based alternative discussed on the PR is not yet sufficient on `main` to make this obsolete. The shipped behavior is still mention-gating plus optional silent ingest on some channels, not unconditional group observation with guaranteed no reply across channels.

## Evidence
