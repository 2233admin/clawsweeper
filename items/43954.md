## Summary

Keep open. Current `main` still does not ship a `feishu_card` tool, and the PR as posted has an unresolved config-contract bug: it adds `tools.card` in the runtime type/defaults but not in the strict Feishu config schema, so a user trying `channels.feishu.tools.card: false` would fail validation.
