## Summary

Keep open. Current `main` still writes raw assistant/user transcript text from the session-memory hook back into memory without any artifact sanitization, so the reported regression is not implemented on `main`. The PR is also not a safe cleanup close: review comments identify concrete contract mismatches with current OpenClaw directive/media syntax and missing coverage that need maintainer judgment rather than auto-closure.
