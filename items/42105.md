## Summary

Current `main` does not implement this PR’s `typingTtlSeconds` feature. The reply path still hardcodes the controller TTL to the internal 2-minute default, and the config/docs/help surfaces still only expose `typingIntervalSeconds` and `typingMode`. This is not strong-close material; the PR should stay open.
