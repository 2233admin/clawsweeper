## Summary

Current `main` does not implement the requested fix. The gateway still auto-adds `opus`/`sonnet` aliases for these Anthropic models, `models.list` still returns alias-decorated allowed-catalog rows, and the Control UI still prefers `entry.alias` over `entry.name` for dropdown labels, so the reported `opus`/`sonnet` labels remain the current code path rather than an obsolete bug.

## Evidence
