## Summary

Keep open: current `main` still starts and probes the bundled `acpx` runtime during gateway startup, still ships `@mariozechner/pi-coding-agent` `0.70.2` with optional `@mariozechner/clipboard` `0.3.3`, and does not expose an OpenClaw-side clipboard/headless guard. The discussion’s `TERMUX_VERSION=1` workaround is an upstream package escape hatch, not an OpenClaw fix, so there is not strong evidence this issue is implemented or obsolete on `main`.
