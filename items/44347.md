## Summary

Keep open. Current main appears to have implemented the threaded outbound reply portion for Google Chat, but the issue's second request, optional receipt of all space messages without @mentions via user OAuth or equivalent event subscription support, is still not implemented on main. The related PR #68967 is useful context, but its own body says it resolves only the first of the two requests in #44347.
