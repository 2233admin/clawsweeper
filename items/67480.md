## Summary

Keep open. Current `main` still has the unconditional nested-skills redirect that the PR is trying to fix, and the mixed `dir/*/SKILL.md` plus `dir/skills/*/SKILL.md` regression case is not covered on `main`. This PR addresses a still-live bug and is not superseded by current code or the latest release in this checkout.
