## Summary

Current `main` already supersedes PR #41622 with a broader proxy-preserving implementation for embedded Codex runs: it bootstraps an `EnvHttpProxyAgent` before timeout tuning, applies the timeout hardening without downgrading that dispatcher, and includes regression coverage for both the proxy bootstrap and embedded runner wiring.
