## Summary

Keep open. Current main already has the tweakcn custom theme importer and it fetches `https://tweakcn.com/r/themes/{id}` directly from the browser, but the Gateway Control UI CSP still limits `connect-src` to `'self' ws: wss:`. The PR's one-line CSP change plus regression assertion targets the real remaining breakage and is not superseded by the broader configurable-CSP issue.

## Best Possible Solution

Keep the PR open and either merge it after normal maintainer validation or apply the same narrow fix on main: add the exact `https://tweakcn.com` origin to the Control UI `connect-src` directive and keep the regression assertion in the Gateway HTTP/CSP test. The broader configurable-CSP issue can remain separate because it does not replace this first-party tweakcn importer default.
