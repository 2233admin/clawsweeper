## Summary

Keep open. Current `main` and the latest published release still ship `node-llama-cpp` as an optional peer dependency rather than an `optionalDependency`, and npm documents that optional peer dependencies are not auto-installed. The linked fix PR #66586 proposed the right change but was closed unmerged, so the report still matches current behavior.
