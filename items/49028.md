## Summary

Current `main` does not implement Vietnamese UI locale support, so this PR is not closable as already shipped. The feature remains absent in the locale type, lazy registry, navigator fallback mapping, and registry tests. The PR is stale relative to current i18n workflow because non-English locale bundles are now generated, but that is a reason to refresh/rework it, not to close it conservatively.
