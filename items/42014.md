## Summary

Keep open. Current `main` does not implement the session sync observability this PR adds, and the PR still contains useful work. The main risk is staleness: its patch targets pre-pluginization paths under `src/`, while current memory CLI/manager code now lives under `extensions/memory-core/`, so it needs a port/rebase rather than closure.
