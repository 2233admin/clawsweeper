## Summary

Current `main` does not implement this PR’s opt-in rescue-watchdog onboarding flow. The onboarding CLI, option types, and interactive finalize path still only handle normal Gateway setup, while the docs point users to the existing manual `--profile rescue` multi-gateway workaround instead of a one-click rescue watchdog.
