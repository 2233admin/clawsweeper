## Summary

Keep open. Current main still converts a `doctor.memory.status` transport timeout into a checked, not-ready gateway memory probe, and the explicit `memorySearch.provider = "local"` doctor path still emits the warning when that probe is not ready. The linked fix PR #49314 was closed unmerged and its intended suppression is not present on main.
