## Summary

Keep open. Current main still has no per-agent `tools.selfDeny` or equivalent delegate-only tool policy. The strict config schema only accepts allow/alsoAllow/deny/profile-style tool policy fields, subagent tool policy is global under `tools.subagents.tools`, and the policy pipeline still applies each layer as a further filter, matching the documented limitation that denied tools cannot be granted back later.
