## Summary

Keep open: current `main` still lacks explicit schema label metadata for `agents.defaults.timeoutSeconds`, so this PR addresses a real gap. The gap is partially masked in the UI by fallback humanization, but it is not implemented on `main`, and the existing review comment correctly notes the proposed label text should likely be aligned with existing timeout label style before merge.
