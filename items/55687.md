## Summary

Keep open. Current `main` adds richer `embedded_run_agent_end` error metadata, but it still logs only sanitized error previews/hashes rather than a stack trace, and I did not find a Bedrock-specific fix or regression coverage for the reported pre-request `.replace` crash.
