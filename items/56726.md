## Summary

Current `main` still omits `--exclude-labels` from the Gmail watcher args, and that arg builder is what the gateway uses to spawn `gog gmail watch serve`, so this PR is not implemented on `main`. I could not verify the reporter's exact history claim that the flag existed in `v2026.2.24` because the public source tag I inspected shows the same omission there too; combined with missing regression coverage, that makes this unsuitable for auto-close.
