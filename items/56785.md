## Summary

Keep open. Current `main` does not implement this PR’s behavior end-to-end: the local pinned mutation helper now falls back to `python` and emits a clearer dependency error, but the SSH remote sandbox fs bridge still invokes `python3` directly and remains on the raw failure path this PR was trying to improve.
