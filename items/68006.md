## Summary

Keep open. Current `main` does not implement Anthropic advisor-tool support: the Anthropic transport still normalizes every tool into the standard `{name, description, input_schema}` shape, request headers only add the existing beta flags, and the user-facing Anthropic docs/tests only cover features like prompt caching, fast mode, and `context1m`. The issue also already has a related open PR (#64064), so there is active in-flight work rather than a stale or already-shipped fix.
