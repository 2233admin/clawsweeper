## Summary

Keep open. Current `main` still deliberately treats `qwen3.6-plus` as unsupported on Qwen Coding Plan endpoints in the shipped catalog, runtime normalization, explicit-model suppression, tests, and user docs. The PR only updates the catalog helper and one test file, so it does not fully implement the behavior it claims to fix and should not be conservatively closed.
