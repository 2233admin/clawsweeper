## Summary

Keep this PR open. Current main still has a no-argument ContextEngineFactory contract and the main run, compaction, and subagent cleanup paths still cannot forward config, agentDir, or workspaceDir into context engine factories. The PR remains a focused additive plugin SDK/API change, not obsolete or duplicated.

## Best Possible Solution

Keep the PR open for maintainer review. The best path is to rebase it onto current main, preserve the additive ContextEngineFactoryContext API for config, agentDir, and workspaceDir, fix or explicitly resolve the fallback context isolation review, update the current docs locations, regenerate plugin SDK API baselines, and run the changed context-engine/plugin SDK gates before merge.
