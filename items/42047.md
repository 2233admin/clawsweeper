## Summary

Current main already supersedes this PR. The skills watcher now watches only explicit `SKILL.md` targets instead of recursively watching whole skill trees, which addresses the FD-exhaustion case more directly than adding `depth: 2`. The memory watcher on main also keeps recursive `extraPaths` coverage while ignoring heavy directories like `node_modules` and `.git`, matching the follow-up concern noted on the PR.
