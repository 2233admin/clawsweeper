## Summary

Current `main` does not implement this PR's `agent_end.finalLlmOutcome` contract. The live hook type still exposes only `messages`, `success`, `error`, and `durationMs`, and the embedded runner still emits only those fields, so the requested behavior is not already on `main` and should remain open.
