## Summary

Keep open. Current `main` still hardcodes hook-dispatched agent runs to `sessionTarget: "isolated"` and does not expose hook `sessionTarget` config/request handling, so the behavior described by PR #51337 is not implemented on `main`. The PR discussion also shows the author addressed the main review findings, so this is still actionable work rather than a stale or incoherent PR.
