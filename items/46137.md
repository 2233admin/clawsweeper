## Summary

Keep open. Current main only repairs renderer-swap `targetId` changes for `/navigate`; the `/act` paths still return the pre-action `tab.targetId`, and the tab-selection layer still rejects stale explicit target IDs with `tab not found`. That matches the reported post-submit failure mode rather than proving it fixed.
