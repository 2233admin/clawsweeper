## Summary

Keep open. Current `main` has basic multi-session primitives in Control UI/WebChat, but not the requested ChatGPT-style multi-chat/tabbed sidebar UX. Session switching is still a header dropdown, prior chats are browsed from the separate Sessions page, and the left sidebar still renders only static navigation groups rather than a chat-session list.
