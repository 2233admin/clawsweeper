## Summary

Keep open. Current `main` still resolves normal provider auth profile-first, so stale per-agent `auth-profiles.json` API keys continue to shadow rotated env/config keys. The repo now has manual mitigations (`secrets audit`, `secrets configure`, ref-backed onboarding), but not an auth-sync command, automatic multi-agent reconciliation, or normal runtime env-first fallback for this case.
