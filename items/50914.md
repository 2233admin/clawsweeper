## Summary

Keep open. Current `main` does not strip inline image blocks on the session JSONL write path, and the run-start image pruner only mutates in-memory history for older completed turns. Based on the provided PR metadata, the PR branch also ended up as a no-op hook plus unrelated test-runner changes, so there is not strong evidence to close this as already implemented or obsolete.
