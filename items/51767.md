## Summary

Keep open. Current `main` still limits `cron.sessionRetention` to cron-run session keys, and the wired reaper path does not sweep subagent or ACP sessions. The separate global `session.maintenance` path is a different config surface with default `warn` mode, so it does not make this report obsolete.
