## Summary

Keep open. Current `main` has a newer parameter-validation path, but it does not implement this PR's tool-specific example strings. The PR is also stale against current tool contracts: `edit` now requires an `edits` array and legacy aliases are intentionally rejected, so the proposed examples/labels no longer match `main` well enough to close as already implemented.
