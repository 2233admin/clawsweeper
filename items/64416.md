## Summary

Keep open. Current main still lacks several central pieces from this PR: ACP turn capability-normalization guidance, task-level spawnedBy metadata, parent-spawned ACP silent-wake delivery, and a shared completion delivery gate. The PR is also paired with overlapping open PR #65242 by the same author, so it should not be auto-closed while that work remains unmerged.

## Best Possible Solution

Keep this PR open for maintainer review or author revision. The best path is to rebase against current main, preserve the already-shipped latest-assistant-output selection, decide whether ACP normalization belongs in the gateway prompt path, add parent wake routing only with durable task metadata or a shared delivery gate, and resolve the untrusted completion-text/persistence concerns. If #65242 becomes the canonical gate-based fix, consolidate the remaining useful pieces there instead of closing this PR prematurely.
