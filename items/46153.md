## Summary

Keep open. I could not verify a fix on current `main`, and I could not reasonably reproduce or dismiss the report from this Linux/non-launchd checkout. The current macOS LaunchAgent path still starts the normal CLI entrypoint, startup still enables Node compile cache by default, and the existing launchd integration coverage does not boot a real gateway under launchd.
