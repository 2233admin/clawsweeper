## Summary

Keep open. Current main does show newer TTS gating/processing for `tagged` mode, but the issue is not strongly closable: there is no read-only evidence that the reported intermittent cross-channel TTS delivery failure was fixed, and the WhatsApp voice-note caption behavior reported later in the thread is still present in current main and covered by tests.
