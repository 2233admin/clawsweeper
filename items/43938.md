## Summary

Keep open. Current `main` still plans channel config hot reloads as full channel restarts, and the account-scoped restart shape proposed in PR #43938 is not present in code or tests. The PR addresses a real gap with existing lifecycle support rather than duplicating an implementation already on `main`.
