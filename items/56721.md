## Summary

Keep open. Current `main` still allows hook-injected bootstrap files to accumulate per session, and the proposed path-dedup logic is not present in the shipped code. Related follow-up PR #57587 existed, but public GitHub state shows it was closed unmerged, so there is no evidence this landed on `main`.

## Evidence

- **Per-session bootstrap cache reuses the same mutable array:** `getOrLoadBootstrapFiles()` stores a `WorkspaceBootstrapFile[]` in a module `Map` and returns the cached array on later calls for the same session key. That means later hook mutations can persist across gateway restarts/session reloads.
  - file: [src/agents/bootstrap-cache.ts:3](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/agents/bootstrap-cache.ts#L3)
  - command: `nl -ba src/agents/bootstrap-cache.ts | sed -n '1,40p'`
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)
- **Bootstrap hooks receive and return that shared array directly:** `applyBootstrapHookOverrides()` passes `params.files` into `event.context.bootstrapFiles`, triggers the hook, then returns the mutated `bootstrapFiles` array from the event context. A hook using `event.context.bootstrapFiles.push(...)` will therefore append onto the cached array instead of a fresh copy.
  - file: [src/agents/bootstrap-hooks.ts:19](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/agents/bootstrap-hooks.ts#L19)
  - command: `nl -ba src/agents/bootstrap-hooks.ts | sed -n '1,40p'`
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)
- **Current main still builds project context without path deduplication:** `buildAgentSystemPrompt()` filters and sorts `contextFiles`, then passes them straight into `buildProjectContextSection()`. That section renders every file entry with `lines.push(\`## ${file.path}\`, ..., file.content, ...)`; there is no Map/set dedup by path on current `main`.
  - file: [src/agents/system-prompt.ts:121](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/src/agents/system-prompt.ts#L121)
