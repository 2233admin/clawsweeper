## Summary

Keep open. Current `main` does not implement `/restore`, does not load any restore command handler, does not document the command, and still omits the `sessionKey` header plumbing the PR adds for archive filtering. This PR is not implemented on `main` and is not obsolete from the checked-out code alone.
