## Summary

Keep #70372 open. Current main does not implement the PR's thinking-aware delivery state, and the related open work covers adjacent commentary/tool-use leaks without conclusively superseding this Anthropic/Bedrock unphased-thinking path. The PR also still needs maintainer review because its fallback can affect providers whose phase metadata arrives later in the stream.

## Best Possible Solution

Keep the PR open for maintainer review. The best path is a narrowly scoped fix with a regression test that simulates Anthropic/Bedrock native thinking events followed by unphased text, proves channel block replies and `assistantTexts` only contain the intended visible answer, and avoids affecting OpenAI/OpenAI-compatible transports whose phase metadata is delayed until `text_end`. If maintainers prefer the broader #70344 approach, explicitly fold this Anthropic/Bedrock thinking case into that canonical PR before closing #70372 as superseded.
