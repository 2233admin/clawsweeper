## Summary

Keep this issue open. Current main ships the env/file/exec SecretRef foundation and documents 1Password through an exec-provider recipe, but native keyring/Keychain, native 1Password, and a plugin SecretRef provider/source seam are still absent. Same-author PR #72548 is an active implementation candidate, so this issue should remain open until that PR is reviewed, landed, or declined.

## Best Possible Solution

Review #72548 as the active implementation candidate. If it lands, close this issue as implemented or split a focused native 1Password follow-up if maintainers still want it bundled. If it is declined, narrow this issue to the accepted native keyring/Keychain, native 1Password, or plugin SecretRef provider API work, with broader cloud-provider tracking left to #13610 or dedicated provider issues.
