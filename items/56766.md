## Summary

Keep open. Current `main` still enqueues Slack block-action and modal lifecycle system events without requesting a heartbeat wake for the routed session, and the shared queue helper does not wake on its own. The PR's proposed fix is not already implemented on `main`, so there is not strong evidence to close this as implemented or obsolete.

## Evidence
