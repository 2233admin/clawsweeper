## Summary

Keep open. Current `main` still routes isolated heartbeat runs through the base outbound session key, so the PR appears to fix a real unfixed bug rather than duplicate shipped behavior.

## Evidence

  - file: [src/infra/heartbeat-runner.ts:867](https://github.com/openclaw/openclaw/blob/7e52223d327555c611e8c91ee2500897e9eab0bf/src/infra/heartbeat-runner.ts#L867)
