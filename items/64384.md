## Summary

Current main still preserves the bug this PR targets: fresh persisted token totals return from `runPreflightCompactionIfNeeded` before token-threshold evaluation unless the newer transcript-byte guard is active. The PR remains a plausible focused fix candidate and is paired with the still-open same-author bug report; overlapping work such as #66716 is still open rather than shipped.

## Best Possible Solution

Keep this PR open until maintainers either land a minimal token-threshold fix here or explicitly choose another open implementation such as #66716. The fix belongs in `src/auto-reply/reply/agent-runner-memory.ts`, with regression coverage in `src/auto-reply/reply/agent-runner-memory.test.ts` for fresh persisted totals above and below the token threshold while transcript-byte compaction is inactive.

## Evidence
