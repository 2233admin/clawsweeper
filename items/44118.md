## Summary

Keep open. Current `main` still drops unmentioned Feishu group messages before command probing runs, and the broadcast path still picks an active agent only when the bot was mentioned or mentions are disabled. The PR remains relevant, and the discussion shows the contributor addressed the main review concern rather than the work becoming obsolete.
