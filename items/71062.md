## Summary

Keep open. Current `main` at `cc0992564bc565791810e6a560d12b430a8544ee` still lacks the PR’s core behavior changes: client-tool handoff in the Pi adapter does not terminate the loop, the gateway does not emit dedicated `response.function_call_arguments.{delta,done}` events, and the Responses stream parser still has no `.done` handler. The PR discussion also shows earlier review findings were addressed in follow-up commits, so this remains live, useful work rather than something already superseded on `main`.
