## Summary

Keep open. Current main routes both auto-reply channel turns and command/dashboard turns through the same skill-snapshot machinery, and the bundled `notion` skill itself has no Telegram-specific restriction. That weakens the original report, but read-only inspection did not produce high-confidence proof of a fix, a non-repro, or an alternative root cause.

## Evidence

- **Telegram/auto-reply skill snapshot path:** Inbound reply sessions resolve the session agent id, merge any skill filter, and build/persist a workspace skill snapshot for the run.
