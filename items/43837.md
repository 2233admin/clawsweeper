## Summary

Current `main` does not implement `/think <level> <body>` as a one-shot override. The PR is still addressing a real gap, especially for leading slash-command routing and Telegram multiline handling, so conservative cleanup should keep it open.

## Evidence

- **Leading `/think ...` is still classified as a control command on `main`:** `hasControlCommand()` treats any registered command alias with whitespace-separated args as a control command, and current `main` has no one-shot `/think` exemption. That means `/think high explain this` still matches the normal command path rather than a dedicated one-shot path.
