## Summary

PR #72319 should remain open. Current `main` at `43a003b8` does not implement the PR's Control UI tool-payload secret redaction: tool start args are emitted raw, `sanitizeToolResult` only strips images/truncates text, and exec command output still derives from the raw result details. The PR is an active, focused fix for open security issue #72283 and is not superseded by a merged implementation on main.

## Best Possible Solution

Keep the PR open for maintainer security review. The likely best path is to merge or refine this PR so Control UI tool start args, text result blocks, exec aggregated stdout, and patch/derived summaries pass through the existing redaction patterns before websocket emission, with regression tests retained. After landing, close #72283 through the normal linked-issue flow.
