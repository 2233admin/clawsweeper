## Summary

Keep open. Current `main` does not implement `gateway.handshakeTimeoutMs`, and the PR discussion identified unresolved blockers in the proposed patch: the strict gateway config schema and help/label metadata were not updated, so the new config key would not be safely shippable as submitted.
