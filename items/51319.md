## Summary

Keep open. Current `main` still only treats Linux zombies as dead, so the reported macOS stale-lock behavior is not implemented. The PR is directionally useful, but it still needs maintainer follow-up because the macOS path is not covered by a real regression test and the PR also includes unrelated runtime log artifacts with local/operator data.
