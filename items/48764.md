## Summary

Keep open. Current `main` still intentionally rejects broad default `workspace-*` media roots and instead allows only the active agent’s concrete workspace plus documented host-local cases. That means this PR is not implemented on `main`, and it is not obsolete enough for `cannot_reproduce`. The proposed glob-based widening also still has unresolved security-review concerns in the PR context, so there is not strong evidence for a conservative close.
