## Summary

Keep open. Current `main` does not implement the Feishu card-parsing expansion described in PR #45936, so this is not `implemented_on_main`. The PR also carries unrelated and likely problematic changes (`src/context-engine/registry.ts` and a generated `docs/.generated/config-baseline.json`) that should be split or dropped before maintainers consider landing it.
