## Summary

Keep open. Current main partially addresses the thinking-default display path, but it still does not implement the requested reasoning-default behavior: `chat.history` returns an effective `thinkingLevel` only, Control UI reasoning display still reads only the persisted session row and falls back to `off`, and `agents.defaults.reasoningDefault` is not a supported global config key. Related #66098 remains an unmerged partial implementation candidate in the provided GitHub context.

## Best Possible Solution

Keep this issue open. Review and land #66098 or a rebased successor for effective chat-default plumbing, then finish the missing reasoning path by returning an effective `reasoningLevel` through `chat.history`, storing/rendering it in the Control UI, adding gateway and UI regression tests, and explicitly deciding whether `agents.defaults.reasoningDefault` should be supported globally or documented as per-agent only. If global support is accepted, update the config schema/docs and the `config.patch` path together.
