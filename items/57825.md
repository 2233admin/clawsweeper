## Summary

Current `main` does not contain this PR's preserve-by-default session persistence change. The shared persistence helper still deletes omitted cleared fields unconditionally, the agent wrapper still always routes override fields through that delete-on-omit path, and the regression coverage described in the PR is not present on `main`.
