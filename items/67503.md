## Summary

Keep this issue open. Current main still enqueues truncated 160-character inbound body previews into trusted system events for Slack, MS Teams, and Mattermost while also delivering the full body through the normal inbound envelope. The same-author implementation candidate, PR #67761, is open and unmerged with closing syntax, so the maintainer path is to review and either land or close that PR rather than close this issue early.

## Best Possible Solution

Keep this issue open and review PR #67761. The likely fix belongs in the Slack, MS Teams, and Mattermost channel handlers: keep full inbound content in the normal agent envelope, remove message-body previews from notification-only system events, and add focused regression tests for long inbound messages.
