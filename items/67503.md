## Summary

Keep open. Current `main` still injects a 160-character inbound preview into trusted `System:` events for Slack, Teams, and Mattermost, while separately passing the full inbound body into the agent/user envelope. The prompt builder prepends those system-event lines ahead of the full body, so the duplicate truncated-header path described in #67503 still exists on `main` and in the latest released code I checked.
