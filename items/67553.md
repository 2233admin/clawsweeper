## Summary

Keep open. Current main does read `HEARTBEAT.md` generically and supports structured `tasks:` parsing with due-only selection, but it still does not dynamically derive heartbeat checks from arbitrary `HEARTBEAT.md` sections/headings. The shipped AGENTS template also still contains a static example checklist, so the requested behavior is not clearly implemented end to end.
