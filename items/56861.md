## Summary

Keep open. Current `main` still lacks the PR's Signal behaviors: outbound media is still sent as local file paths only, group mention gating is still regex-only, and quoted Signal context still carries text/sender only rather than quoted attachments. I did not find evidence of an equivalent merge on `main`.
