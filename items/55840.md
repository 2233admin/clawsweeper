## Summary

Keep open. Current `main` intentionally removed the Chrome extension relay in `2026.3.22`, still rejects `driver: "extension"`, and documents `existing-session` as a constrained migration path rather than restored feature parity. That means the request is neither implemented nor obsolete, and there is not enough scope evidence to auto-close it as a ClawHub/plugin-only item.
