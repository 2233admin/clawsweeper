## Summary

Keep open. Current `main` does not implement an `agent:turn:end` internal hook, and this PR still has unresolved contract/runtime concerns called out in review: the proposed payload drops `errorCode` despite the PR description promising it, and the emission path uses the unbounded `fireAndForgetHook` on top of sequential hook execution.

## Evidence
