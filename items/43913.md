## Summary

Keep open. Current `main` does not implement ACP session support in the `subagents` tool, and the proposed PR still has unresolved correctness/scope issues called out in review, so there is not strong evidence for any conservative auto-close reason.

## Evidence

The description and all three action branches operate on `listControlledSubagentRuns(...)` / `resolveControlledSubagentTarget(...)`; there is no ACP-session branch.
