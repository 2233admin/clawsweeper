## Summary

Current `main` does not implement this PR's feature. Context pruning still converts images to the fixed `[image removed during context pruning]` marker, hard-clear still collapses tool results to a generic placeholder, there is no `cacheMedia` config surface on `main`, and the tests still assert permanent stripping behavior. Conservative cleanup should keep the PR open for maintainer review/rebase rather than auto-close it.
