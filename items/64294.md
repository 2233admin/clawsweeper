## Summary

Keep open. Current main does not ship the proposed `api.runtime.llm.complete()` injected runtime API or the matching `ContextEngineRuntimeContext.llm` capability. The PR is a coherent plugin-SDK/core-runtime feature, related to open issue #45865, and should remain available for maintainer review rather than be auto-closed.

## Best Possible Solution

Keep the PR open for maintainer review or fold it into the canonical #45865 work. The best implementation path is a narrow injected `api.runtime.llm.complete()` facade with active-agent-aware model/auth resolution, context-engine capability injection where appropriate, public SDK docs/API-baseline alignment, and tests covering plugin runtime construction plus context-engine `compact`/`afterTurn`/`maintain` access.
