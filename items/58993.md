## Summary

Keep PR #58993 open. Current main still classifies Google Chat messages using only the legacy `space.type` field, while the Google Chat Space resource now exposes `spaceType` and `singleUserBotDm` and marks `type` deprecated. The PR addresses a real remaining gap tied to #58514, but should gain focused regression coverage before merge. ([developers.google.com](https://developers.google.com/workspace/chat/api/reference/rest/v1/spaces))

## Best Possible Solution

Keep the PR open for maintainer follow-up. The right path is to add a small tested classification helper or equivalent regression tests, then merge the targeted Google Chat fix so current `spaceType`/`singleUserBotDm` payloads route to the correct direct or group peer.
