## Summary

Keep open. Current main only partially addresses the reported disk bloat: stale `.deleted.*` and `.reset.*` archives can be cleaned during session-store maintenance, but normal maintenance still has no orphaned `*.tmp` sweep and no general cleanup for transcript `.jsonl.bak.*` files. The linked PR #47635 is relevant and directly targets this issue, but the provided GitHub context says it is still open/unmerged, so the issue should remain open until a fix lands.
