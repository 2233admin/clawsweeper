## Summary

Keep open. Current `main` does not show this PR's runtime-auth guard or an equivalent landed fix. `google-vertex` ADC still resolves to the non-secret marker `gcp-vertex-credentials`, and the embedded runner still writes the resolved credential into runtime auth storage on the no-exchange path, so there is not strong evidence that the reported bug is already implemented on `main`.
