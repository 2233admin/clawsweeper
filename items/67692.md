## Summary

Keep open. Current `main` still does not lazy-initialize ACP for TUI sessions when `agents.list[].runtime.type = "acp"`, even though config/docs expose that per-agent runtime contract. The PR also has unresolved review concerns in the CLI path, so this is not a conservative auto-close candidate.
