Keep PR #71786 open. Current main already has the non-mutating sort change, but the paged-select part is still a real type cleanup and the PR's join-to-string approach is not the right fix for the TUI component contract.

Keep the PR open for maintainer or contributor revision, but do not merge it as-is.
