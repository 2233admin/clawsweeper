## Summary

Current `main` still applies `toolsAllow` before bundled MCP/LSP tools are materialized, then appends bundled tools without reapplying that allowlist. This PR still addresses a live gap on the checked-out `main`, so it should remain open rather than be auto-closed as already implemented or obsolete.
