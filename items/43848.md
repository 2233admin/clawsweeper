## Summary

Keep open. Current main partially fixes Telegram topic routing by preserving threadId in announce target resolution and final send, but the report's remaining core regressions are still present: ANNOUNCE_SKIP suppresses the visible handoff instead of falling back to the target reply, and the generic A2A ping-pong path can still inject inter_session turns back into the requester transcript.
