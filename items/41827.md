## Summary

Keep open. Current main has docs and tests for custom-provider selection and local OpenAI-compatible compatibility knobs, but I did not find strong evidence that #41827’s specific silent-fallback and empty-subagent behavior is fixed on main. The report is still recent and concrete, and the latest-release-to-main diff does not show a clear implementation fix for this path.
