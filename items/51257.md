## Summary

Keep open. Current `main` still does not surface session `label`/`displayName` in `/status` or the TUI status summary, so the request is not implemented. The PR is also not a clear close candidate because current OpenClaw surfaces prefer `label` over `displayName` when humans need a usable session selector, and terminal-safe sanitization is required before printing untrusted labels/display names.
