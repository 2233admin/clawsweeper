## Summary

Keep open. Current `main` does not implement the PR's startup secret-resolution backoff: gateway startup still performs a single `prepareRuntimeSecretsSnapshot(...)` attempt and immediately throws on startup failure. The latest shipped release `v2026.4.23` appears unchanged in this area as well.
