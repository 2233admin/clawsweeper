## Summary

Keep open. Current `main` still silently accepts `openclaw config set agents.defaults.model...` with only the generic restart message, while the docs and runtime tests confirm that `agents.list[].model` overrides take precedence. The PR is still addressing a real current-main DX gap, and the provided review thread shows the author already handled the bracket-notation and string-shape review feedback on the PR branch.
