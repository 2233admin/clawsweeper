## Summary

Keep PR #57789 open. Current main still has the default Docker sandbox image fallback that can tag plain Debian as `openclaw-sandbox:bookworm-slim`, while the mutation helper still requires Python inside the sandbox. The PR is a plausible focused bug fix for #57713, and the related open PR #56785 only covers part of the same recovery-message surface.

## Best Possible Solution

Keep the PR open for maintainer review or replace it with an equivalent core fix: remove the default-image Debian pull/tag fallback, surface clear rebuild/recreate guidance for incompatible existing Docker sandboxes, and keep the focused regression coverage/docs updates. Reconcile the overlapping #56785 guidance work as a partial duplicate only after one canonical fix is chosen.
