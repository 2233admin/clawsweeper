## Summary

Keep open. Current `main` only forwards Signal quote text/sender/quote-flag; it still does not forward `ReplyToId`, and the Signal quote schema/sender resolution still ignore `quote.id` and `quote.authorNumber`, so the PR's requested behavior is not already implemented on `main`.
