## Summary

Current `main` does not contain the PR's proposed Ollama no-config fix. The auth resolver still returns `null` when `models.providers.ollama` is absent, the Ollama plugin only mints synthetic auth for meaningful explicit config, and current docs/changelog explicitly preserve missing-config failures rather than treating no-config Ollama as auth-free.
