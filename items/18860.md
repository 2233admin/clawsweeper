## Summary

Keep PR #18860 open. Current main at 1252da325f2b5afa354cfb8ff94ae61783005060 still does not expose `after_tools_resolved`, a hook-runner dispatch for resolved tool metadata, or an equivalent plugin API for reading the finalized post-policy tool list with schemas. The linked issue #21330 is closed and points back to this PR, and the latest maintainer/Codex comment also explicitly kept this PR open.

Keep the PR open as useful design and implementation context, but extract or land only a narrow maintainer-approved plugin/runtime seam.
