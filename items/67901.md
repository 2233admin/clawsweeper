## Summary

Keep open. Current `main` still emits the `plugins.allow excludes "subagents"` path for unknown primary commands, but `subagents` is not actually registered as a root CLI command on `main`; docs only describe `/subagents` as a chat slash command. This means the report still reflects a real current UX/behavior gap rather than something already implemented or safely closable.
