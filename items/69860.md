## Summary

Current `main` does not implement Feishu-native exec approval clients. Feishu is wired only for approver authorization plus general interactive cards, while native exec-approval config/runtime/docs currently exist for other channels and do not list Feishu. The request remains coherent and in scope for the bundled Feishu channel, so it should stay open.

## Evidence

- **Feishu channel exposes only auth capability:** The bundled Feishu channel sets `approvalCapability: feishuApprovalAuth`, with no Feishu-specific native approval runtime wiring in the channel entrypoint.
  - file: [extensions/feishu/src/channel.ts:666](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/extensions/feishu/src/channel.ts#L666)
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Feishu approval capability is auth-only:** `feishuApprovalAuth` is created with `createResolvedApproverActionAuthAdapter(...)` and only resolves approvers from `allowFrom`; it does not define `nativeRuntime`, delivery targets, or exec-approval setup guidance.
  - file: [extensions/feishu/src/approval-auth.ts:15](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/extensions/feishu/src/approval-auth.ts#L15)
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Feishu config schema has no execApprovals surface:** The Feishu shared/account/top-level config schemas define DM/group/streaming/actions fields but no `execApprovals` key, matching the reporter's claim that `channels.feishu.execApprovals` is absent on `main`.
  - file: [extensions/feishu/src/config-schema.ts:156](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/extensions/feishu/src/config-schema.ts#L156)

- **Docs list native exec approval clients, not Feishu:** The exec-approvals advanced docs say native clients are controlled by `channels.<channel>.execApprovals` and then enumerate Discord, Slack, and Telegram. Feishu is not listed there.
  - file: [docs/tools/exec-approvals-advanced.md:267](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/docs/tools/exec-approvals-advanced.md#L267)
