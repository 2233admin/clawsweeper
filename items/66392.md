## Summary

Keep open. Current main still ships only `claw`, `knot`, `dash`, and one browser-local `custom` tweakcn token slot; Matrix and Cyberpunk are not implemented as Control UI theme families. The linked Matrix PR is closed unmerged, and the existing plugin/API surfaces do not provide a supported Control UI theme registration path that would justify a ClawHub close.

## Best Possible Solution

Leave this issue open for maintainer product/design judgment. If accepted as core UI work, add Matrix and Cyberpunk as first-class Control UI theme families across the theme model, base CSS, full and quick settings selectors, and focused accessibility/reduced-motion tests. If maintainers prefer external distribution, first define and document a supported theme-pack or Control UI theme registration API; the current tweakcn custom slot is too narrow for the requested animated effects.
