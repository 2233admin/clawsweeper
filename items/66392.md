## Summary

Keep open. Current `main` does not ship Matrix or Cyberpunk as built-in Control UI themes. Instead, it now supports a single browser-local imported `custom` theme slot, while the built-in families remain `claw`, `knot`, and `dash`. That is a related alternative path, but not a high-confidence match for closing this feature proposal, especially with the linked PR thread still active in the issue discussion.
