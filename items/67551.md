## Summary

Keep open. Current `main` still does not implement Unicode smart-quote repair for tool-call argument JSON, so this PR's core fix is not already shipped or otherwise safely closable. The later openai-completions repair on `main` is narrower: it repairs leading/trailing junk around ASCII JSON, but it still fails when smart quotes are used as JSON delimiters.
