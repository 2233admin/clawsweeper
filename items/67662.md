## Summary

Keep open. Current `main` still omits `modelId` in the two incomplete-turn cooldown writes, and the auth-profile cooldown code uses `modelId` to preserve model-scoped rate-limit state. The requested behavior is not implemented on `main`, and the PR aligns those paths with existing model-aware failover writes.
