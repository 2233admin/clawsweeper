## Summary

Keep open. Current `main` still has a schema/transport mismatch: exec approvals allowlist entries can carry `source: "allow-always"`, the gateway snapshot path sends that file shape to Control UI, the UI saves the cloned file back unchanged, and `exec.approvals.set` validates against a schema that rejects unknown allowlist properties such as `source`. The issue’s cross-referenced PR #69732 is not reflected in the checked-out `main` SHA.
