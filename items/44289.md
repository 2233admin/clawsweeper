## Summary

Keep #44289 open. Current main derives the JSON matrix from the secret target registry, but the Markdown credential-surface reference is still a checked-in hand-maintained list guarded by a parity test. The central acceptance criteria still remain: both reference artifacts should be generated from one canonical source so registry metadata changes do not require manually syncing docs blocks.
