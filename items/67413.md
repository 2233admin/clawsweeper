## Summary

Keep open. Current `main` still treats dreaming as a single global memory-plugin configuration and resolves all configured agent workspaces for the sweep without any per-agent enable/disable or per-agent schedule override. The report is recent, concrete, and supported by discussion plus an open linked PR, so conservative cleanup should not close it.

## Evidence
