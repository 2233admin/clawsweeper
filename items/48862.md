## Summary

Keep open. Current `main` no longer matches the issue's exact `Node >=22.16.0` failure for `node 22.14.0`, but the requested feature is still not implemented in `openclaw update`: the updater only detects an unsupported Node runtime and tells the user to upgrade Node manually. Automatic Node installation exists only in the separate installer flow, so the feature request remains plausible and not strongly closable.
