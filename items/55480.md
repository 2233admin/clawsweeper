## Summary

Keep open. Current `main` does not include the non-ASCII display-name fallback, so the reported display bug is not implemented on `main`. The PR also includes unrelated `GroupSpace` changes that would change guild-policy lookup precedence from guild ID to guild slug, which conflicts with the current resolver and documented/configured ID-keyed guild entries.
