## Summary

Keep open. Current `main` still implements `tools.sessions.visibility` as one shared enum for all session-tool capabilities, with no structured per-capability object form. Cross-agent `sessions_send` still depends on the same global visibility dial used by `sessions_list` and `sessions_history`, so the requested split knobs are not implemented.
