## Summary

Keep open. Current `main` partially addresses the logging side by attaching raw provider-error diagnostics to lifecycle warning logs, but the operator-facing lifecycle/chat payloads still only carry the rewritten friendly error text and tests still assert that behavior. The reported Control UI/operator-surface gap is therefore not implemented on `main` yet.

## Evidence
