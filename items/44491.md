## Summary

Keep open. Current main has strong evidence that the title bug, Control UI chat history replay after restart/reconnect, and the heartbeat-label clobbering path have been addressed. However, the issue body also reports Dashboard localization inconsistency, and current main still has user-visible English strings in Sessions/Settings surfaces plus a static `<html lang="en">`. No provided canonical follow-up tracks that remaining localization work, so this should not be auto-closed as fully implemented.
