## Summary

Keep open. Current `main` still suppresses live chat/session fanout for non-control-UI-visible runs, so this request is not implemented on `main`. At the same time, the PR's proposed one-line gate removal would widen delivery of hidden external-channel content without a narrower authorization path, so there is no strong basis to close it as fixed or obsolete.
