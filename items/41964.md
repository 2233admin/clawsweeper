## Summary

Keep open. Current `main` still intentionally suppresses chat/session fanout for `isControlUiVisible: false` runs, so this PR is not implemented on `main`. The proposed one-line change also widens delivery to node session subscribers without any matching session-authorization change, so the privacy concern raised in discussion remains substantive rather than obsolete.
