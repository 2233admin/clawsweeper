## Summary

Keep open. Current main disproves the originally suspected field-alias root cause, but it does not provide strong evidence that the Volcengine Ark no-usage streaming case is fixed. The Volcengine provider still registers plain OpenAI-compatible completion models without a Volcengine-specific usage compat override, and the OpenAI completions stream only updates usage when a stream chunk actually contains usage data.
