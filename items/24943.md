## Summary

Keep open. Current main still sends iMessage outbound text through plain-text sanitization that only rewrites HTML, while the actual iMessage send path only converts Markdown tables and strips directive tags. The repo has a generic `stripMarkdown(...)` helper, but it is not wired into the iMessage outbound path.

## Evidence

- **iMessage outbound sanitizer is HTML-focused, not Markdown-focused:** The iMessage channel registers `sanitizeText: ({ text }) => sanitizeForPlainText(text)`, and outbound delivery applies that sanitizer before channel send. `sanitizeForPlainText` rewrites HTML tags like `<b>`, `<i>`, `<code>`, and `<br>`, but does not strip Markdown markers such as `**`, backticks, headings, or blockquotes.
  - file: [extensions/imessage/src/channel.ts:247](https://github.com/openclaw/openclaw/blob/d3f6783b16ecc10c67b101bf019ca4294453206a/extensions/imessage/src/channel.ts#L247)
  - command: `nl -ba extensions/imessage/src/channel.ts | sed -n '238,252p'; nl -ba src/infra/outbound/deliver.ts | sed -n '382,405p'; nl -ba src/infra/outbound/sanitize-text.ts | sed -n '1,80p'`
  - sha: [d3f6783b16ec](https://github.com/openclaw/openclaw/commit/d3f6783b16ecc10c67b101bf019ca4294453206a)
- **Direct iMessage send path does not strip Markdown:** `sendMessageIMessage` only runs `convertMarkdownTables(...)` and `stripInlineDirectiveTagsForDelivery(...)` before passing `text: message` to the `imsg` client. There is no Markdown-to-plain-text conversion in this path.
  - file: [extensions/imessage/src/send.ts:139](https://github.com/openclaw/openclaw/blob/d3f6783b16ecc10c67b101bf019ca4294453206a/extensions/imessage/src/send.ts#L139)

- **Reply delivery path also preserves Markdown aside from table conversion:** The monitor delivery flow sanitizes assistant-internal scaffolding, then calls `resolveSendableOutboundReplyParts` with `text: convertMarkdownTables(rawText, tableMode)`. It does not call `stripMarkdown(...)` or any equivalent plain-text Markdown conversion before invoking `sendMessageIMessage(...)`.
  - file: [extensions/imessage/src/monitor/deliver.ts:41](https://github.com/openclaw/openclaw/blob/d3f6783b16ecc10c67b101bf019ca4294453206a/extensions/imessage/src/monitor/deliver.ts#L41)

The visible references are the helper definition and TTS tests.
