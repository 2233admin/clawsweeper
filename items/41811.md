## Summary

Current `main` already ships the requested Slack thread opt-out behavior under `channels.slack.thread.requireExplicitMention` instead of the PR's proposed `thread.autoReplyOnParticipation`. The implementation is wired through config schema, monitor context/provider plumbing, mention gating, docs, and targeted tests, and it is present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
