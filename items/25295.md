## Summary

Current main still lacks the requested Mattermost message-tool edit/delete action surface. Related preview-streaming work landed underlying post update/delete helpers, but `extensions/mattermost/src/channel.ts` still advertises and handles only `send` and `react`, so PR #25295 remains useful rather than obsolete.

## Best Possible Solution

Keep PR #25295 open or replace it with a focused rebased PR that wires Mattermost `edit` and `delete` into `extensions/mattermost/src/channel.ts`, adds targeted tests in `extensions/mattermost/src/channel.test.ts`, and documents the user-facing action behavior if maintainers want this surface supported.
