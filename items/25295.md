## Summary

Keep open. Current `main` does not implement Mattermost `message` tool `edit`/`delete` actions. The bundled Mattermost plugin still advertises only `send` and optional `react`, and its action handler rejects any other action. `main` does have low-level post update/delete helpers for internal draft-streaming, but that is not the same user-facing feature this PR adds.
