## Summary

Keep PR #71573 open. Current main at 6d60b035b4e7c1739f038808e041567448f58060 still lacks the `update.run` restart-continuation producer path that this PR adds, while the restart continuation consumer and restart-action producer already exist. The PR is not obsolete or implemented on main.

## Best Possible Solution

Keep #71573 open as the active focused fix for the `update.run` continuation gap. The best path is to address or explicitly disposition the security-review concerns, keep the protocol schema, parser, handler, agent-tool forwarding, generated Swift models, and regression tests together, then land the producer-side fix. After landing, maintainers should decide whether #71178 closes or remains open for the broader graceful-drain/reset-bridge follow-ups.
