## Summary

Keep open. Current `main` still drops post-update agent continuations: `update.run` does not accept or forward `continuationMessage`, does not build `payload.continuation`, and the existing tests explicitly encode that missing behavior. The surrounding restart-continuation machinery already exists and is active for normal restarts, so this PR is addressing a real gap rather than obsolete behavior.
