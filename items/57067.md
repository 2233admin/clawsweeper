## Summary

Current `main` still uses a single dropdown on the Agents page, and the `Set Default` button only stages a config change until the separate Save action is used. That means the UX request is not implemented on `main`, and the reported refresh-loss behavior is still plausible if the user clicks `Set Default` without also saving.
