## Summary

Keep open. I could not verify an implemented fix on current `main` or the latest shipped tag. The current gateway session-row builder still falls back to `origin.label`, the WebChat selector still renders `label`/`displayName` verbatim, and the heartbeat sender resolver still emits the literal fallback `"heartbeat"` when no better sender is available. That matches the reported failure mode rather than disproving it.
