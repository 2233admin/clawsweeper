## Summary

Keep #44302 open. Current `origin/main` does not implement the requested opt-in `session.mirrorWebchatToLastExternalChannel` behavior: the config surface has no such key, the strict session schema has no corresponding field, and the reply-routing policy still explicitly suppresses external route delivery for ordinary internal webchat turns. The related #52068 PR attempted this feature but was closed unmerged, so it does not supersede the issue.

## Evidence
