## Summary

Keep open. Current `main` still persists `/exec` overrides into `SessionEntry`, resolves those persisted values ahead of agent/global exec config, and preserves them across session resets. That means the reported stale per-session exec behavior after global config changes is still present rather than implemented or obsolete.
