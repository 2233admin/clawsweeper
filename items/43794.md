## Summary

Keep #43794 open. Current main has useful SecretRef mitigations for supported credential surfaces, but it does not implement native config encryption at rest, a system-keychain-backed OpenClaw credential store, or a master-password workflow. The threat model still explicitly tracks plaintext token-at-rest risk as high and recommends token encryption at rest.
