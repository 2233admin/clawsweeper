## Summary

Current `main` does not implement per-account SOUL-file support. The `channels add` CLI still has no `--soul` flag, onboarding has no SOUL-file configuration step, and runtime bootstrap still reads the fixed workspace `SOUL.md` only. Some snapshot-compatibility plumbing touched by the PR is already present on `main`, but the requested feature itself is not shipped.

## Evidence
