## Summary

Keep open. Current `main` still synchronously reads and parses `subagents/runs.json` on non-test read paths, and `sessions.list` still reaches that path during session filtering/row building. I did not find evidence in the checked-out code that this issue is implemented or obsolete.

## Evidence
