## Summary

Current `main` does not contain this PR's fix. Remote memory embedding and batch HTTP requests still flow through `withRemoteHttpResponse`, and that helper still calls `fetchWithSsrFGuard` without `mode: GUARDED_FETCH_MODE.TRUSTED_ENV_PROXY`, so `HTTPS_PROXY`/`HTTP_PROXY` support for this path is not implemented on `main`.
