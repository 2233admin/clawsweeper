## Summary

Keep open. Current `main` only partially covers this PR: the user-relay exec prompt now embeds exec details, but the internal-only exec path still drops the event body, and the mixed-queue/deferred-reminder loss raised in review still matches current `main` queue consumption and duplicate-suppression behavior.
