## Summary

Keep #45134 open. Current main still lacks the requested stopReason-gated buffering for block replies and still has the cumulative assistantTexts final-payload path called out in the issue. The existing phase-aware commentary suppression helps only when messages carry commentary/final_answer metadata; unphased legacy/tool-call text can still be emitted before message_end/toolUse is known.
