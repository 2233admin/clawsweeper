## Summary

Keep open. Current main still has only binary bootstrap filtering for subagent/cron versus other sessions, no `agents.defaults.bootstrapTier` schema/docs/types, and no tier-aware hook wiring. The related model-aware AGENTS/SUBAGENTS stack is closed and scoped differently, while #22438 remains the open proposal this PR implements.

## Best Possible Solution

Keep #22439 open until a maintainer either lands tier-aware bootstrap loading or explicitly chooses a different product direction. If proceeding, the implementation should cover the tier resolver, `agents.defaults.bootstrapTier` schema/types/labels/help/docs, bootstrap-extra-files hook behavior, and tests for default and override behavior across main, subagent, and cron sessions.
