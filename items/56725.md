## Summary

Keep open. Current `main` does not implement path deduplication for bootstrap files, and the underlying accumulation mechanism is still plausible because cached bootstrap arrays are reused by reference and exposed to hooks as a mutable array. This specific PR also is not strong enough to close out conservatively because its proposed first-wins dedupe would preserve stale entries when hooks append refreshed content for an existing path, and it adds no regression test.
