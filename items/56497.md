## Summary

Keep open. Current `main` does not include the Volcengine session-cache feature from PR #56497, and the latest code shows the provider still ships on the older `openai-completions` path with no Volcengine-specific cache wrapper or docs. The request is coherent and still belongs in OpenClaw, but the implementation likely needs a rebase onto newer shared provider-stream hooks rather than cleanup closure.
