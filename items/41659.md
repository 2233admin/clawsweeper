## Summary

Keep open. Current `main` still serves Control UI `index.html` without rewriting favicon or apple-touch-icon links for `gateway.controlUi.basePath`, while the UI HTML still uses root-absolute favicon paths and docs still advertise `basePath` support. The PR’s review feedback about `$` replacement handling was addressed in its second commit per the provided discussion, so this remains a live, useful fix rather than a stale or already-implemented change.
