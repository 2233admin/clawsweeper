## Summary

Keep open. Current `main` does not implement configurable custom-term autocomplete/suggestions for the WebChat chat input. Existing suggestion UI is limited to fixed welcome chips and slash-command completion, and the WebChat config surface only exposes history truncation settings.

## Evidence

- **WebChat welcome suggestions are fixed starter prompts, not user-configurable terms:** `WELCOME_SUGGESTIONS` is a hardcoded array and is rendered only in the empty-state welcome UI.
  - file: [ui/src/ui/views/chat.ts:462](https://github.com/openclaw/openclaw/blob/3a64aa49a9014518fa1977fbc023f3e0b978ae5d/ui/src/ui/views/chat.ts#L462)
