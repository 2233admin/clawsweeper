## Summary

Keep open. Current `main` still has the mismatch this PR addresses: the workspace context loader always reports missing `BOOTSTRAP.md`, while the gateway file list already hides `BOOTSTRAP.md` after setup is complete. The downstream "commit your changes" reminder path is still present, and the relevant regression coverage is not on `main` yet.

There is no `isWorkspaceSetupCompleted`/post-onboarding suppression in this function.
