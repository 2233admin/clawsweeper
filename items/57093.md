## Summary

Current `main` already addresses the reported Feishu/Lark DM reset leak. The Feishu plugin now only replays channel-side history for group chats, while core `/new` and `/reset` handling rotates to a fresh session and archives the previous transcript. This appears to be present on unreleased `main` (`2026.4.24`), so the issue is best closed as implemented on `main` rather than as a shipped release fix.
