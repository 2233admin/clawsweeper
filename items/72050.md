## Summary

Keep PR #72050 open. Current main does not ship a `subclaw` command, and the linked request in #71832 is a plausible core CLI/session workflow rather than something clearly outside the repo. The submitted PR is not merge-ready as-is: the provided diff does not update the sub-CLI descriptor catalog, so the new command would likely be filtered out, and the branch bundles unrelated gateway/scripts/auth changes with the feature.

## Best Possible Solution

Keep the item open for maintainer review or contributor revision. The best path is to split unrelated changes out, keep #71832 as the canonical feature request, then land a focused `subclaw` PR that updates the descriptor catalog, wires `cwd` through `TuiOptions` and autocomplete, derives a stable per-directory session key, handles directory errors consistently, and adds targeted CLI/TUI tests plus user-facing docs if the command is accepted.
