## Summary

Keep #43564 open. Current main has ACP spawning and per-agent skill filtering for native OpenClaw runs, but it does not implement the requested ACP skill-context injection surface: `sessions_spawn` has no `skills` argument, ACP rejects native context forking, ACP attachments are unsupported, and the ACP manager/runtime turn contracts only carry plain text plus media attachments. The related context-injection items do not supersede this specific skill-injection request.
