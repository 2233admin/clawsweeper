## Summary

Current `main` has a bundled Codex harness and plugin seams that could support this work, but it does not implement computer-use-specific spawn/configuration or automatic GUI-task routing. The request also matches the repo vision's stated interest in better computer-use and agent-harness capabilities, so there is not strong enough evidence to conservatively close it as out of scope or already handled.
