## Summary

Keep open. Current `main` has OpenAI-compatible request code that includes top-level `tools`, but the current Ollama docs still explicitly warn that `/v1` OpenAI-compatible mode breaks or unreliably handles tool calling. With that mixed evidence, and no live repro run in this checkout, there is not enough confidence to auto-close conservatively.
