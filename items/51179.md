## Summary

Keep open. Current main still truncates `allowFrom` to two entries in the `channels status --probe` formatter and in `buildChannelSummary()`, and it still requests `allowFrom` data for non-admin status summaries. This PR still contains unfixed value on main rather than work that is already implemented there.
