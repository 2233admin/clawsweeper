## Summary

Current `main` already implements the substance of this PR in a different shape: package updates re-resolve the installed global package root after the update, return that verified root in `result.root`, and the gateway service refresh uses that updated root. The shipped `v2026.4.23` release already contains this behavior and test coverage, so this PR is superseded.

## Evidence
