## Summary

Keep open. Current `main` does not expose a per-turn raw-context or assembled model-input inspector in the Sessions UI, and the available session/history surfaces are adjacent diagnostics rather than the requested operator-facing turn inspector.

## Evidence

- **Sessions UI only exposes session list controls and compaction checkpoints:** The Sessions view renders a table of session metadata and per-session overrides, plus optional compaction checkpoint details. There is no turn/run drill-in, no raw prompt payload view, and no assembled model-input inspector.
