## Summary

Keep open. Current `main` still contains the race described by the PR, so this is not `implemented_on_main` and there is no strong basis to close it as obsolete. The Control UI still renders approval buttons that call `handleExecApprovalDecision(...)` without a captured approval id, while the queue entry can be pruned asynchronously before the click handler reads `execApprovalQueue[0]`.
