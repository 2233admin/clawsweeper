## Summary

The report targets an older Feishu streaming-footer path that is not present on current `main`. Current Feishu streaming cards use a direct note field, not a session-store footer metrics lookup, and multi-account dispatch is explicitly account-aware, so the reported non-default-account footer-metrics bug does not match current behavior.
