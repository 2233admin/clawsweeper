## Summary

Keep open. Current `main` already covers the older navigation-preservation part of this branch, but the selector-rerender fix described in PR #57077 is not present on `main`. The desktop and mobile session pickers still rely on `.value` plus `?selected` option attributes, the mobile picker still reuses the desktop session-control class, and current tests do not cover the fallback-option rerender case this PR targets.
