## Summary

Keep #44603 open. Current main still supports `pty` only as a per-call exec parameter, while the requested `tools.exec.pty` global/default config is absent from the config schema, config types, default merging, and `usePty` resolution. The related PR #45976 directly attempted this feature, but the provided GitHub context shows it was closed unmerged, so it does not supersede the open issue.
