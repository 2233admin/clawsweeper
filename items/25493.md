## Summary

Keep open. Current `main` still treats the core system prompt as OpenClaw-owned with fixed sections, including a hardcoded `## Safety` block. The only built-in end-user customization on `main` is full prompt replacement via `agents.defaults.systemPromptOverride` or adding injected workspace files like `AGENTS.md`/`SOUL.md`; there is no section-level template editing or `SAFETY.md`-style override matching the request.
