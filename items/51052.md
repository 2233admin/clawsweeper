## Summary

Keep open. Current `main` at `6944d7025d9c0ba4891cbe646ab5dab039d504ca` still hardcodes the Control UI tab title to `OpenClaw Control`, and the gateway config/schema/bootstrap contract do not expose any `gateway.controlUi.title` field. This feature request is not implemented on `main` yet.
