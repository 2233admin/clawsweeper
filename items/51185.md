## Summary

Keep open. Current `main` still contains the reported behavior: `ensureDockerImage()` retags `debian:bookworm-slim` to `openclaw-sandbox:bookworm-slim` whenever the default sandbox image is missing, while the actual default sandbox build recipe installs `python3`, `jq`, and `ripgrep`. The runtime path therefore still replaces the richer default image with plain Debian instead of rebuilding or surfacing a repair path.
