## Summary

Keep #69269 open. Current main still lacks the coordinated invariant fix: several compaction/reset hook paths do not carry or normalize `messageProvider`, subscribe-time compaction hooks omit it entirely, and the shared compaction accounting helper still ignores valid `tokensAfter: 0` outcomes. The open PR #69270 is the right related implementation, but it is not merged into the reviewed main SHA.

## Best Possible Solution

Keep #69269 open and use #69270, or an equivalent maintainer-owned patch, to add a shared hook message-provider resolver, thread normalized providers through all compaction/reset hook emitters, and refresh token/cost/cache accounting for `tokensAfter: 0` outcomes with regression coverage across command, gateway, and subscribe-time compaction paths.
