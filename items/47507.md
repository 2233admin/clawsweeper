## Summary

Keep open. Current `main` still implements Feishu as an app/bot-credential channel only; there is no user-scoped OAuth flow, no `user_access_token`/refresh-token storage, no config surface for user auth, and no exported Feishu calendar/profile tooling that would consume user tokens.
