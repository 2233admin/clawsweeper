## Summary

Keep open. Current `main` does not audit `SKILL.md` content in the deep security scan, so the requested behavior is not implemented. The issue’s `skill.md` + `index.js` example does not exactly match OpenClaw’s markdown-only skill model, but the underlying request is still coherent because skills are prompt instructions that can influence tool usage while the code-safety scanner only inspects script files.
