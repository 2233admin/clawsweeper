## Summary

Current `main` does not implement this PR’s requested fix. `collectExposureMatrixFindings()` still only audits `groupPolicy="open"`, so `dmPolicy="open"` and legacy `dm.policy="open"` remain uncovered for the `security.exposure.open_groups_with_elevated` and `security.exposure.open_groups_with_runtime_or_fs` findings. There is partial `dmPolicy` coverage elsewhere for exec exposure, but not for this exposure-matrix path.
