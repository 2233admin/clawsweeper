## Summary

Keep open. Current `main` still hardcodes the Control UI `Permissions-Policy` and CSP headers, and the gateway config/schema/docs expose no `gateway.controlUi` fields for CSP or permissions-policy overrides. The only partial change is that microphone is already allowed for same-origin by default, so that part of the original workaround is obsolete.
