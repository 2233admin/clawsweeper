## Summary

Keep open. Current main has newer generic Feishu conversation-binding plumbing, but the ACP block-delivery fix from #48834 is not on main. Feishu still drops block payloads, and ACP delivery still counts queued block sends as successful visible delivery, so the final-text fallback can be skipped even when nothing was actually sent to Feishu.
