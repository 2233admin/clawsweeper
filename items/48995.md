## Summary

Keep open. Current `main` does not implement the requested `thinking-events` capability or the routing/delta fixes described in the issue. Gateway caps still only define `tool-events`, reasoning streaming is still disabled unless `onReasoningStream` is present, reasoning deltas are still computed from formatted text, and non-tool agent events still go through the generic broadcast path.
