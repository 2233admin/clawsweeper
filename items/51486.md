## Summary

Keep open. Current `main` and the latest release still run the broad `schtasks /Query` preflight inside `readScheduledTaskRuntime()`, so the Windows status bug from #49187 is not implemented on main. The PR remains aligned with that gap and adds focused regression coverage, including the spawn-error fallback raised in review.
