## Summary

Current `main` does not implement this rename. The Control UI locale type, lazy-load registry, browser-locale resolution, persisted locale values, tests, and user docs still use `zh-CN` and `zh-TW`, so there is no strong basis to close as implemented or obsolete.

## Evidence
