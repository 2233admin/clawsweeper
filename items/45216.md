## Summary

Keep open. Current `main` still lacks the `detail`-payload handling described in the PR, so the linked raw-JSON leak remains plausible from code inspection. The PR contains the missing logic plus regression tests, but it is now out of date and marked non-mergeable/dirty, so this needs normal maintainer review rather than conservative auto-close.
