## Summary

Keep open. Current `main` still does not implement a gateway-only tolerant config validation path: both runtime startup and config snapshot reads use the strict plugin validator, and there is no `validateConfigObjectTolerant*` API on `main`. Part of the PR has also drifted because `main` already treats `google-gemini-cli-auth` as a removed legacy plugin.
