## Summary

Keep open. Current `main` still emits pi-embedded `lifecycle:end` without `aborted` or `stopReason`, and the latest shipped tag in this checkout (`v2026.4.23`) has the same omission. The issue discussion points at PR #66574/commented-as-fixed, but that behavior is not present in the current code.
