Current main still does not implement this PR's diagnostic-heartbeat skip in the transcript tail scanner, and the related open #66544 work is a different display-name fallback path. The PR should stay open for maintainer review rather than be closed as implemented, duplicate, or obsolete.

## Best Possible Solution
