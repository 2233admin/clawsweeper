## Summary

Keep open. Current `main` still limits ACP `streamTo="parent"` relays to assistant text summaries plus lifecycle notices; tool and thinking events exist in the agent event system but are not forwarded by `startAcpSpawnParentStreamRelay()` or covered by its tests/docs as visible parent-stream output.

## Evidence

- **Parent relay only handles assistant and lifecycle streams:** `startAcpSpawnParentStreamRelay()` processes `event.stream === "assistant"`, then returns early for anything except `"lifecycle"`. There is no branch for `"tool"`, `"thinking"`, `"command_output"`, or related streams.
