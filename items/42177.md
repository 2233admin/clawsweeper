## Summary

Keep open. Current main has generic Telegram/Feishu duplicate-delivery guards, but I did not find strong evidence that #42177's specific regression was fixed. The repo still only has structured ClawHub install paths for explicit `clawhub:` specs or CLI skill slugs, and I found no code/tests covering a raw `https://clawhub.ai/...` link like the reported `self-improving-agent` flow.
