## Summary

Keep open. Current `main` still mixes two different data scopes on the Usage page: the header uses `sessions.usage` totals aggregated across configured agents, while the daily chart still comes from `usage.cost`, which only loads the default agent transcript directory. The linked fix PR for this exact mismatch, #42063, is still open and unmerged.
