## Summary

Keep open. Current `main` still accepts role-less final assistant payloads, but the submitted fix is not closeable as-is: it broadens every missing-role message to assistant semantics, and current Control UI rendering now flows through `normalizeMessage()`/`grouped-render.ts` rather than relying only on `extractText()`. The PR needs rework, not cleanup-closing.
