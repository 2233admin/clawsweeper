## Summary

Keep open. Current main has adjacent plugin observation hooks (`agent_end` and `model_call_ended`) and lifecycle error events, but it does not implement the requested `agent:error` hook with structured error code/retryable context or recovery/resume semantics. The issue is recent, coherent, unprotected, and still describes a real missing contract decision.

## Evidence
