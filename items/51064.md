## Summary

Keep open. Current main still carries the alert-oriented heartbeat contract and docs that the issue is challenging, so there is not strong evidence that #51064 is implemented or obsolete. Main does contain a newer GPT-5-specific overlay that pushes heartbeats toward continued work, but that is only a partial/model-specific mitigation and does not replace the generic heartbeat framing.
