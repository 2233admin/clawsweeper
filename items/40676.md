## Summary

Keep PR #40676 open. Current main still lacks both central fixes: Discord voice auto-join is still wired only to READY handling, and the voice join path still calls `joinVoiceChannel` without account-scoped stale `@discordjs/voice` connection cleanup. The branch also cannot merge as-is because it edits the old `src/discord/**` layout and includes review-flagged session-key changes beyond the stated fix.

## Best Possible Solution

Keep the PR or its linked issue open for maintainer follow-through. The best path is a focused rebased Discord plugin patch in `extensions/discord/src/voice/manager.ts`, `extensions/discord/src/voice/manager.runtime.ts`, and `extensions/discord/src/monitor/provider.ts` that adds/registers a `DiscordVoiceResumedListener`, performs account-scoped stale connection cleanup before joining, and adds targeted tests for RESUMED registration plus stale cleanup. If this PR is abandoned, keep #40665 as the canonical bug until an updated fix lands.
