## Summary

Keep PR #70629 open. Current main still emits `thinkingBudget: 128` for `reasoning: "minimal"` on Gemini 2.5 Pro and Flash-family models, and the current tests do not cover the proposed 512 floor. The related #57683 cleanup fixed a different `thinkingBudget=0` failure mode, so this PR is not implemented or superseded.

## Best Possible Solution

Keep the PR open and review it as a focused Google provider bug fix. The best implementation path is a narrow update in `extensions/google/transport-stream.ts` to raise the default Gemini 2.5 minimal budget floor, plus regression coverage in `extensions/google/transport-stream.test.ts` for `gemini-2.5-flash-lite`, `gemini-2.5-flash`, and `gemini-2.5-pro`. If maintainers confirm the current Google API minimum, land the targeted fix and consider a small changelog entry because released builds still contain the 128 default.
