## Summary

Keep open. Current `main` still writes file-log JSONL records directly to disk without sink-level redaction, and the user-facing logging docs explicitly say redaction is console-only and does not alter file logs. The repo also has redaction coverage for diagnostic emission, which reinforces that this specific file-transport gap remains unresolved on `main`.

## Evidence
