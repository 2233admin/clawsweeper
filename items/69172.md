## Summary

Current main does not implement the Google/Gemini endpoint and apiType override behavior claimed by PR #69172, and the related PR #61714 is closed unmerged rather than a shipped canonical fix. The PR is not merge-ready because it bundles unrelated changes and has unresolved P1 review findings, but the underlying bug/feature request is concrete and still appears unaddressed on main, so none of the conservative close reasons fits.

## Best Possible Solution

Keep the PR open for now, but require a narrow replacement or substantial cleanup: limit the diff to the Google plugin/provider surfaces, add tested baseUrl/apiType resolution for web search, embeddings, image generation, and media only if maintainers want that behavior in the bundled Google plugin, preserve native Google defaults, remove unrelated files, and address the existing P1 review findings.
