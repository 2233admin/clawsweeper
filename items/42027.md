## Summary

Keep open. Current `main` does not contain this PR's three core changes: exec PATH probing is still gateway-only, browser status still lacks the layered diagnostics/HTML-error summarization proposed here, and manual `cron.run --force` still queues on `CommandLane.Cron` with tests asserting that behavior. Some adjacent browser/CDP diagnostics and docs landed separately, but not enough to treat this PR as implemented on `main`.
