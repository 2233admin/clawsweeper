## Summary

Keep #44134 open. Current main still rebuilds and provider-normalizes the tool catalog on each embedded attempt, and the Google Gemini CLI provider still uses the Gemini tool-schema normalization hook on that path. The issue has recent concrete follow-up evidence tying the same path to memory pressure, so this is not stale or safely auto-closeable.
