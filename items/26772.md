## Summary

Keep open. Current `main` still rebuilds the builtin memory index in a fresh temp SQLite file and only swaps it into place after the full build completes; if the build fails, the temp index is deleted. I found no checkpoint/resume path or related config on `main`, so the feature request is still unmet.

## Evidence
