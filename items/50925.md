## Summary

Issue #50925 proposes a standalone accounting platform that is outside OpenClaw core scope. Current main positions OpenClaw as a personal AI assistant and explicitly says optional capabilities should ship as plugins, with new skills published to ClawHub first.

## Close Comment

If you want to pursue SovereignLedger, the maintainable path is an external plugin/skill integration, not OpenClaw core.
