## Summary

Keep open. Current `main` does not contain the requested `skills/skill-creator/references/common-pitfalls.md` file or the SKILL.md link, so this PR is not implemented on main. But the PR’s lead rationale is now partly obsolete: current skill loading tolerates inline colons in `description`, while the packager/validator already fails invalid YAML in standard environments. That makes the PR salvageable but not closeable under the allowed reasons.
