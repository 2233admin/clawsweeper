## Summary

PR #63981 should stay open. On current `main`, `runGatewayLoop` still awaits `params.start()` directly with no startup deadline or late-cleanup path, so the proposed fail-fast behavior is not implemented. The linked PR discussion also shows the contributor addressed the review findings called out in-thread, which makes this still-actionable rather than obsolete.
