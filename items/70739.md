## Summary

Keep open. Current `main` does not implement SSE heartbeats for `/v1/chat/completions` or `/v1/responses`, while the PR addresses a plausible gateway bug with existing repo precedent. This is not implemented on `main`, not stale, and still has active review follow-up, so conservative cleanup should not close it.
