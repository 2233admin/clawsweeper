## Summary

Keep #69291 open. Current main has adjacent prompt guidance through `Execution Bias`, `Tool Call Style`, and a GPT-5-family behavior overlay, but it does not implement the requested dedicated, model-agnostic Agent Behavior Principles section for all channels/models. The related #71937 context is a closed design/RFC item that references #69291; it is not a shipped implementation or canonical open tracker for this specific prompt component.

## Best Possible Solution

Keep the issue open for maintainer product judgment. The best path is either to add a small core-owned behavior-principles section that explicitly covers the missing global scope-control and verification guidance, or to document that OpenClaw intentionally relies on compact `Execution Bias` plus model/plugin-specific overlays and close only after that product decision is made in a canonical issue/PR.
