## Summary

Keep open. Current main has adjacent prompt guidance in Tool Call Style, Execution Bias, Safety, and a GPT-5-specific behavior contract, but it does not implement the requested systematic all-model Agent Behavior Principles component or fully cover the proposed scope/surgical-operation guidance. The related #71937 is closed and broader, so it is not an open canonical tracker for this concrete request.

## Best Possible Solution

Have a maintainer decide the product direction: either add a small all-model behavior-principles section by extending the existing `Execution Bias`/`Tool Call Style` prompt surface, with prompt-composition tests and docs, or explicitly redirect this kind of optional behavior tuning to a plugin using `before_prompt_build`/system-prompt contribution hooks. Do not auto-close until that scope decision is made or a canonical follow-up exists.
