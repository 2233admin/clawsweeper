## Summary

Keep #69291 open. Current main has adjacent prompt guidance via `Execution Bias` and the GPT-5 behavior contract, but it does not implement the requested dedicated, global Agent Behavior Principles section across all models/channels. The related #71937 context is a closed design/RFC item, not a canonical implementation tracker that supersedes this narrower request.

## Best Possible Solution

Maintainer should decide whether the existing compact `Execution Bias` plus GPT-5 overlay is the preferred product direction, or add a small core prompt section that explicitly covers the missing global behavior-principle contract, especially surgical scope control. If kept external, document the recommended plugin hook or prompt-overlay path instead of silently closing the core request.
