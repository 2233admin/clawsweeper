## Summary

Keep open. Current main already has auto-compaction, memory flush, pruning, and overflow-retry mitigations, but that is not strong evidence that #47668 is fixed. The issue has fresh activity with an open follow-up PR (#70605), and the current prompt-context sanitization path still injects context file contents nearly verbatim rather than stripping struck-through/deprecated text.
