## Summary

Current main does not implement the doctor check requested in #49036. `openclaw doctor` still has only a session-lock health contribution and the session-lock implementation only scans agent session directories. Main has separately hardened gateway startup against stale gateway locks, which may make the original failure mode less likely, but that is not the same as adding gateway-lock detection/cleanup to doctor.
