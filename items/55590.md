## Summary

Current `main` only supports one memory-search embedding provider, one model override, and one fallback provider per agent. It does not implement the requested `primary + fallbacks` embedding stack, multiple embedding-model choices for memory search, or adaptive runtime switching. The issue is also not obsolete: per-agent `memorySearch` overrides already exist, but the comment's vector-compatibility concern is real because changing embedding dimensionality requires a reindex.
