## Summary

Keep open. Current `main` still runs the Feishu setup status probe through `probeFeishu()` during the wizard, and that probe still constructs `@larksuiteoapi/node-sdk`'s `Client` without overriding its logger. The bundled SDK version defaults to `LoggerLevel.info` and emits `client ready`, which matches the reported leak. The report is also fresh and already cross-referenced to an open fix PR (#67734), so there is no conservative basis to auto-close it.
