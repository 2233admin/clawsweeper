## Summary

Current `main` has explicit required-parameter validation for the built-in `read`/`write`/`edit` tools, but it does not implement the issue's broader request for automatic pre-call validation with configurable `tools.validation.enabled` / `autoPrompt`, and `exec` still does its own inline guard inside the tool body. Evidence is not strong enough to close this as implemented.
