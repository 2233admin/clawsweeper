## Summary

Keep open. Current `main` still emits unrenderable `{}` JSON Schema branches for both `meta.lastTouchedAt` and `sandbox.docker.setupCommand`, and the Control UI analyzer still treats those unions as unsupported. The PR appears to address a real bug, but it is neither implemented on `main` nor ready for conservative cleanup closure because review feedback remains unresolved.

## Evidence
