## Summary

Keep open. Current `main` no longer falls straight through to a zero-payload terminal turn in this path; it retries some empty replies and otherwise surfaces an explicit incomplete-turn error. But the codebase still intentionally does not synthesize a user-visible completion from successful tool runs when the final assistant text is empty, so the issue’s requested Anthropic/tool_result behavior is not clearly implemented.
