## Summary

Keep open. Current `main` does not implement a user-level per-skill `preferences.md`/`conventions.md` sidecar or any equivalent auto-loaded skill-scoped preference file. The documented/customization path is still full `SKILL.md` override plus global bootstrap notes like `TOOLS.md`, so the request remains a live core feature request rather than cleanup-close material.
