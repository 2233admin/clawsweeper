## Summary

Current `main` still has the reported schema/implementation mismatch. `src/agents/tools/nodes-tool.ts` keeps `node` optional in `NodesToolSchema`, but the `describe` branch still calls `readStringParam(params, "node", { required: true })`, so `action: "describe"` without `node` is still rejected on `main`.
