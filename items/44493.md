## Summary

Keep open. Current main has generic gateway auth and Control UI token handling, but it does not implement Paperclip `openclaw_gateway` token auto-configuration or document the Paperclip adapter setup path. The related #48566 Paperclip pairing/onboarding docs gap remains open, and #53090 covers a different gateway-side pairing/auth behavior rather than this token auto-population request.
