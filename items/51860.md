## Summary

Keep open. Current `main` still manages macOS gateway installs as a per-user LaunchAgent only, does not expose a system/LaunchDaemon install mode, and still builds service env with the host `TMPDIR`. `gateway status --deep` can discover LaunchDaemons as extra services, but the primary managed-service path remains LaunchAgent-based. That matches the issue report and the recent 2026-04-24 downstream confirmation in the provided discussion.
