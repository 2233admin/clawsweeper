## Summary

Current `main` has related protections for background/session routing, delivery-state accounting, and heartbeat exec-event prompting, but I could not prove this specific live Signal failure is fixed. The Signal path still treats a successful `signal-cli` RPC `send` as delivery, and I did not find a Signal-specific heartbeat/session regression test covering the reported case where the final text exists in the internal session transcript but does not reach the human in Signal.

## Evidence

- **Signal delivery still stops at RPC acceptance:** `deliverReplies()` forwards each reply chunk to `sendMessageSignal()` and only logs that delivery happened when the payload was non-empty; it does not reconcile a later user-visible receipt back into status.
  - file: [extensions/signal/src/monitor.ts:331](https://github.com/openclaw/openclaw/blob/0e9156d2053022d208c3eea1d0fa1e377ed0a154/extensions/signal/src/monitor.ts#L331)
  - sha: [0e9156d20530](https://github.com/openclaw/openclaw/commit/0e9156d2053022d208c3eea1d0fa1e377ed0a154)
- **Signal send success is derived from `signal-cli` send response only:** `sendMessageSignal()` calls the Signal JSON-RPC `send` method and returns success from the returned timestamp/message id. There is no second confirmation step that the DM actually surfaced to the handset/UI.
  - file: [extensions/signal/src/send.ts:211](https://github.com/openclaw/openclaw/blob/0e9156d2053022d208c3eea1d0fa1e377ed0a154/extensions/signal/src/send.ts#L211)

## Risks / Open Questions

- I could not run a live Signal end-to-end reproduction in this read-only review environment, so there is no direct proof that the reporter's failure path is gone.
- The current Signal send path still equates `signal-cli` RPC acceptance with delivery, which leaves room for 'internal transcript has final text, user never saw it' failures outside or after the RPC boundary.
- The issue is only two days old and has no logs, screenshots, or follow-up comments, so closing it would be too aggressive for a conservative maintainer cleanup pass.
