## Summary

Keep open. Current `main` has partial overlap: it supports streamed usage for specific built-in local provider IDs and for explicit `compat.supportsUsageInStreaming: true`, but it still intentionally leaves generic custom/local OpenAI-compatible endpoints disabled by default. That means this PR's address-based auto-detection for arbitrary local inference endpoints is not implemented on `main`.
