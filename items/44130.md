## Summary

Keep open. Current main does not show an implemented TUI viewport/auto-scroll control or scroll-anchor fix for the reported terminal chat behavior. The TUI still mounts ChatLog as a plain pi-tui Container, updates it on every streaming delta, and has no documented or tested scroll preference, jump-to-latest action, or bottom-follow policy for the chat log.
