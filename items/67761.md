## Summary

Keep open. Current `main` still injects the 160-character preview into inbound system events for Slack, MS Teams, and Mattermost, while separately delivering the full message body. That matches the PR’s reported bug, so this is neither implemented on `main` nor a good candidate for conservative auto-close.
