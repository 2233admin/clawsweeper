## Summary

Keep open. Current `main` still uses a module-local `SANDBOX_BACKEND_FACTORIES` map, while plugin backends are registered through the public plugin SDK and at least one shipped extension (`openshell`) depends on that registry. The proposed fix is not implemented on `main`, and current tests do not cover the reported cross-module-instance failure mode, so there is not strong evidence to close this PR conservatively.
