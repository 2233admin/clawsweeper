## Summary

Keep open. Current `main` still gates reasoning streaming on the presence of `onReasoningStream`, and the gateway/webchat path still does not guarantee that callback. The gateway fanout already broadcasts non-tool agent events, so the PR’s core claim remains plausible and is not already implemented on `main`.
