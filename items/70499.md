## Summary

Keep open. Current `main` still enforces a 2 MiB limit for local avatar files, but the docs on `main` do not document that limit in the active agent identity docs. The PR's intent is still valid, though one doc target is stale and likely needs a rebase onto the newer `docs/gateway/config-agents.md` page.
