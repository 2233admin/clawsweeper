## Summary

Keep open. Current `main` still persists `source: "allow-always"` in exec approval allowlist entries, while the gateway `exec.approvals.set` schema still omits `source` and forbids extra properties. The CLI forwards the file unchanged to the gateway, so the reported mismatch remains plausible on `main` and is also present in the latest stable tag `v2026.4.23`.
