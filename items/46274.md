## Summary

Current main treats Tencent/WeChat integrations as plugin territory, not OpenClaw core. The repo already documents WeChat as an external channel plugin, and the vision/docs explicitly route optional service integrations to community plugins/ClawHub/npm.

Because WeChat Official Account article publishing/message handling is a Tencent-specific integration surface, the right path is a community plugin/ClawHub package rather than expanding OpenClaw core here.
