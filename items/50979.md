## Summary

Keep open. Current `main` does not contain this PR's `messageToolHints` formatting guidance, and the shipped Slack formatting path on `main` appears materially different from the PR branch. `main` injects Slack formatting rules through `inboundFormattingHints`, while plain-text outbound replies still run through Markdown-to-mrkdwn conversion. That makes it unsafe to call this already implemented or obsolete.
