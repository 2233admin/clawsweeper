## Summary

Keep open. Current `main` still leaves `/bot-clear-storage` on the QQBot pre-dispatch path without `requireAuth`, while that command can delete files under `~/.openclaw/media/qqbot/downloads/<appId>/`. The PR is a narrow, evidence-backed security fix in the same class as the already-landed `/bot-approve` gating change.

## Evidence
