## Summary

Keep open. Current `main` still does not implement WhatsApp auto-reply `message_sending` / `message_sent` hook wiring, and the PR discussion identifies unresolved behavioral questions rather than an obsolete or incoherent change.

## Evidence

It does not import `openclaw/plugin-sdk/hook-runtime`, does not run `message_sending`, and does not emit `message_sent` or internal `message:sent` for auto-replies.
