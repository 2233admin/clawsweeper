## Summary

Keep open. Current `main` at `7ac35b4f69009490180a35bf404638d4280a4865` still rejects Windows drive-letter bind sources during both config validation and runtime sandbox validation, so the PR's reported bug is not implemented on `main` and remains actionable.

## Evidence
