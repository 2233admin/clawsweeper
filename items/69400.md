## Summary

Current main does not implement OPENCLAW_WRAPPER or a --wrapper install option. The issue is actionable core daemon/service behavior, and the related PR #69425 is still open rather than merged or shipped.

## Best Possible Solution

Keep #69400 open while maintainers review PR #69425. The best path is to land or revise that PR so resolveCliProgramArguments honors OPENCLAW_WRAPPER, then decide whether the same change should include a persisted --wrapper install option, service environment persistence, doctor/status visibility, docs, and regression tests for LaunchAgent reinstall/update behavior.
