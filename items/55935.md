## Summary

Current `main` still emits the locale-formatted `Current time:` line for heartbeat/cron-style prompts, and it still only injects a single line rather than a separate trigger timestamp. The only linked fix attempt was PR #55957, but it was closed unmerged due the author's active-PR limit, so the issue is not implemented on `main`.
