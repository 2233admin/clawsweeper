## Summary

Keep open. Current `main` still waits on all live descendant subagent runs during isolated cron delivery, without scoping that wait to the current cron run window, so the linked bug is not implemented on `main`. The PR also picked up an unrelated `scripts/restart-gateway.sh` change with unresolved review feedback, so it is not a conservative auto-close candidate.

## Evidence

- **Current follow-up wait is not window-scoped:** `waitForDescendantSubagentSummary` on current `main` has no `runStartedAt` parameter and filters active descendants only by `endedAt`, so any still-live descendant under the requester session can be awaited regardless of when it started.
  - file: [src/cron/isolated-agent/subagent-followup.ts:75](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/cron/isolated-agent/subagent-followup.ts#L75)
  - command: `nl -ba src/cron/isolated-agent/subagent-followup.ts | sed -n '75,106p'`
  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Current delivery path still counts all active descendants:** `dispatchCronDelivery` still calls `countActiveDescendantRuns(params.agentSessionKey)` before and after the follow-up wait. That count API has no run-window input, so stale-but-live descendants from older executions can still suppress or delay current isolated cron delivery.
  - file: [src/cron/isolated-agent/delivery-dispatch.ts:685](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/cron/isolated-agent/delivery-dispatch.ts#L685)

  - sha: [5fe333ada85a](https://github.com/openclaw/openclaw/commit/5fe333ada85afd0def9e85985ce57e36aa00108a)
- **Underlying descendant count API is session-wide, not run-window scoped:** `countActiveDescendantRunsFromRuns` increments for every live unended descendant found under the requester session graph. There is no timestamp/window filter in this helper.
  - file: [src/agents/subagent-registry-queries.ts:249](https://github.com/openclaw/openclaw/blob/5fe333ada85afd0def9e85985ce57e36aa00108a/src/agents/subagent-registry-queries.ts#L249)
