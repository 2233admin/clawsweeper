## Summary

Keep open. Current `main` still contains the linked regression path: config materialization writes empty `safeBinProfiles`/`safeBinTrustedDirs` back as `undefined`, the raw-snapshot safety check still rejects any non-exact round-trip, and the Control UI/docs/tests now explicitly force Form-only mode when `raw` is unavailable rather than restoring Raw editing.
