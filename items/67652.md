## Summary

Keep open. Current `main` and the latest shipped release still do not accept the loose outbound `media` alias in the plugin SDK reply-payload helpers, so this PR is addressing a real gap rather than obsolete work. The review discussion also shows follow-up commits for the reported edge cases, which argues against conservative auto-closure.
