## Summary

Keep open. Current main still implements the reported failure: `sessions_spawn` exposes `streamTo` in the shared schema, resolves the default runtime to `subagent`, and returns `streamTo is only supported for runtime=acp` before the subagent dispatch path. The existing test suite also still asserts the old rejecting behavior. The discussion points to open PR #68397 as the intended fix, but that fix is not present on current main.
