## Summary

Keep #72219 open. Current main still builds a `delivery.resolved` object for bare `delivery.mode: "none"` cron runs, so the PR is not obsolete and directly addresses the linked open bug #72210 with a focused code change and regression test.

## Best Possible Solution

Keep the PR open for maintainer review and land #72219 or an equivalent targeted patch that omits `delivery.resolved` only for intentionally disabled fallback delivery, while preserving explicit-target `delivery.mode: "none"` message-tool context and adding the regression test.
