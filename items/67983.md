## Summary

Keep open. Current `main` still does not expose tool or client-tool schemas on the `llm_input` hook, so the feature is not implemented on `main`, and this PR is a recent additive plugin-contract change with unresolved correctness/fidelity review concerns that need maintainer judgment rather than conservative auto-close.
