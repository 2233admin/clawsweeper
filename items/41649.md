## Summary

Keep open. Current `main` does not implement this PR’s model-change transcript archiving/handoff flow, and it also does not contain the simpler session-model reset path from the related discussion. Model-related config reloads still only restart the heartbeat while persisted session runtime model fields remain authoritative.
