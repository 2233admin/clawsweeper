## Summary

Current main already has partial model-fallback visibility, but it is not a full default user notification across surfaces. Auto-reply emits fallback lifecycle events and WebUI shows fallback toasts, yet plain agent replies stay silent when verbose mode is off, so the feature request is not clearly implemented well enough to close conservatively.
