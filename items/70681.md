## Summary

Keep open. Current `main` still uses the Unix `lsof` `c` field to decide whether a listener is an OpenClaw gateway, and the existing tests still encode that behavior. Nothing in the checked-out `main` shows this PR's `ps argv` verification landed, and the review discussion already identifies follow-up correctness fixes on the PR itself, so there is no conservative basis to auto-close it.
