## Summary

Keep #59662 open. The provided issue and comment describe a concrete Anthropic OAuth/Claude Max metadata leak, and current main still has no targeted filter for those usage-alert text blocks before assistant text is streamed, finalized, or delivered to channels.

## Best Possible Solution

Keep the issue open and implement a narrow Anthropic OAuth/Claude Max usage-alert sanitizer in the provider output or assistant-text normalization path, with regression coverage for standalone alert drops and embedded alert removal before both transcript persistence and channel delivery.
