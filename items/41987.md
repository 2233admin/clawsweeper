## Summary

Current `main` still ignores `agents.defaults.imageMaxDimensionPx` for `browser` image outputs. The browser plugin no longer lives under `src/agents`; it now lives under `extensions/browser`, and the current implementation still does not thread config into `createBrowserTool` or pass `imageSanitization` to `imageResultFromFile` for either `screenshot` or labeled `snapshot`. This PR should stay open in principle, but it needs a refresh for the current file layout and should also cover the labeled snapshot path.
