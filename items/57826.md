## Summary

Keep open. Current `main` still has a live schema/runtime mismatch around tool arguments: exec/process schemas require canonical `command`/`action`, read validation requires canonical `path`, and current tests on `main` explicitly reject alias-shaped arguments like `file_path` while other current tests/handlers still encounter alias-shaped tool calls. That matches the issue and its fresh April 24, 2026 reports, so there is not strong evidence for closing.
