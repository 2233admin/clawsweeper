## Summary

Keep open. Current `main` and release `v2026.4.22` contain multiple Telegram and config hardening changes that plausibly address parts of this report, but the issue bundles several Windows-specific symptoms and does not provide enough concrete reproduction data to prove the whole problem is fixed on current `main` with high confidence.
