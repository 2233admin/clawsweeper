## Summary

Keep open. Current `main` still has a `+`-icon `New session` button in the composer toolbar, separate from but adjacent to the `Attach file` paperclip flow, and the Control UI config schema does not expose any `gateway.controlUi.hideNewSession`-style toggle or similar reposition/hide option. The report is recent and still matches the shipped UI, so there is not strong evidence to close it.
