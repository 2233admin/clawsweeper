## Summary

Keep open. Current `main` still routes user-facing assistant text through `sanitizeUserFacingText`, but the sanitizer stack on `main` does not contain any handling for leaked plain-text `to=... {json}` tool-call prefixes, so this is not implemented on `main`. The submitted PR also has an unresolved false-positive concern from review, so there is not strong evidence to close it conservatively.
