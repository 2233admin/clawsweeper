## Summary

Keep PR #40756 open. Current main still appends the HEARTBEAT.md workspace-path hint based only on prompt text, including resolved user-configured heartbeat prompts, while the docs still describe custom heartbeat prompts as sent verbatim. The linked bug #40255 remains open and the related PRs in the supplied context were closed as duplicate children, not merged replacements.

## Best Possible Solution

Keep PR #40756 or the canonical bug #40255 open. The best maintainer path is a small heartbeat-runner fix that distinguishes user-configured heartbeat prompts from built-in/generated prompts, preserves the workspace-path hint for default/internal prompts where desired, keeps custom prompts consistent with the documented verbatim contract, and adds a focused regression test for a custom prompt mentioning HEARTBEAT.md.
