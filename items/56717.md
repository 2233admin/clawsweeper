## Summary

Current `main` no longer has the reported semantics. Bundled chat channels explicitly enabled under `channels.<id>.enabled` bypass `plugins.allow`, restrictive allowlists are auto-augmented for configured built-in channels, and the config help text now documents that behavior. The latest release tag `v2026.4.23` already contains the same implementation.
