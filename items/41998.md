## Summary

Current `main` already installs an env-proxy-aware undici global dispatcher during CLI startup, preserves that proxy dispatcher when applying long streaming timeouts, and has regression coverage for the proxy bootstrap/error-handling path. This makes PR #41998 redundant on current `main`.
