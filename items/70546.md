## Summary

Keep open: current `main` still hard-codes `generic_repeat` as warning-only, current tests explicitly preserve that behavior through `criticalThreshold`, and the runtime only blocks on `critical` results, so the report still matches shipped behavior rather than an already-fixed issue.
