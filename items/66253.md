## Summary

Keep open. I did not find strong evidence that current `main` already fixed this custom-provider image-drop path, and the issue discussion says the proposed fix PR was closed without merge. The current gateway logic still fails closed when the model is missing from the runtime catalog, and the nearby tests only cover Foundry/Claude shims rather than config-defined custom providers like `bigmodel/glm-5v-turbo`.
