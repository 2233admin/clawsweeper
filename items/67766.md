## Summary

Keep open. Current `main` does not contain this PR's gateway fallback, still returns `false` on catalog misses, and still suppresses `qwen3.6-plus` on Coding Plan endpoints before inline provider-model config is consulted. That is enough evidence that the PR is neither implemented on `main` nor safely closeable as obsolete.
