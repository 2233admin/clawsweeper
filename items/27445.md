## Summary

Keep open. Current `main` does not implement the requested `announceTarget` config/tool option, and top-level subagent completions still default to external delivery rather than a parent-only orchestration route. Current code does have partial wake-up behavior for active requester sessions and `sessions_yield`, but that is not the requested configurable parent-routing feature.
