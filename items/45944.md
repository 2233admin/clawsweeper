## Summary

Keep open. Current `main` does not implement Ollama batch embeddings via `/api/embed`; the bundled Ollama embedding provider still posts each text to `/api/embeddings` and `embedBatch` is still `Promise.all(texts.map(embedOne))`. The tests also still assert `/api/embeddings`, and a repo-wide search found no `/api/embed` usage in the Ollama provider.
