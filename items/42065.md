## Summary

Keep open. Current `main` has partial overlap with this PR's goal, but it does not already implement the requested global topbar locale switch, it does not contain the proposed `scripts/ui-i18n-report.ts` / `ui/src/i18n/README.md` workflow, and the PR's locale-switch handler still has a plausible async race against the current `i18n.setLocale()` contract.
