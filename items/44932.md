## Summary

Keep open. Current main still treats a timed-out or otherwise failed `doctor.memory.status` RPC as `checked: true, ready: false`, which is the same condition that causes doctor memory-search health to warn. The related PR #46576 described the right fix but was closed unmerged, and the current source/test coverage still matches the reported false-warning path rather than resolving it.
