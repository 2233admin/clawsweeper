## Summary

Keep open. Current main has a new OpenAI-completions transport with broader tool-call parsing coverage, but there is not strong enough evidence that issue #67745 is fixed end-to-end for the reported custom provider. A key part of the reporter's claim also still does not match current main: OpenClaw does not send `tool_choice` by default when tools are present.
