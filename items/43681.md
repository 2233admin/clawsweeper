## Summary

Keep open. Current main has auth-profile rotation/failover once multiple profiles exist, but it still does not solve the reported same-email OpenAI Codex OAuth login problem: Codex OAuth identity prefers the email as the profile name, `models auth login` has no alias/profile-id option, and the related alias PR (#43694) was closed unmerged.
