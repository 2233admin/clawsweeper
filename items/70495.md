## Summary

Keep #70495 open. Current main still has the confusing CLI wording that #65209 reports: `openclaw mcp list/show` read only `mcp.servers`, but the human-readable CLI output still says generic “MCP servers” and has no mcporter exclusion note. The linked issue is still open and asks for exactly this CLI/help/docs clarification. ([github.com](https://github.com/openclaw/openclaw/issues/65209))

## Best Possible Solution

Keep the PR open and refresh it against current main. The best path is to land the narrow CLI/help/docs clarification with tests: label `mcp list/show` as OpenClaw-managed, add the mcporter exclusion note to human-readable output and MCP docs, keep JSON output unchanged, then close #65209 when merged.
