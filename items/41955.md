## Summary

Keep open. Current `main` still does not add configured agents without existing sessions to the chat session dropdown, and the PR is stale against a refactor that moved this logic out of the file it edits. The underlying idea still looks useful, but it is not implemented on `main` and likely needs an updated patch plus handling for global session scope.
