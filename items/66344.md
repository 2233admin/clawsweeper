## Summary

Keep open. Current `main` does not implement table-aware block chunking: `EmbeddedBlockChunker` only protects fenced code blocks, there is no `table-spans` parser on `main`, docs still describe fence-only protection, and the PR discussion shows related duplicate work (#66568) was closed in favor of this PR rather than superseding it on `main`.
