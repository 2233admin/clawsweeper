## Summary

Keep open. Current main still contains both behaviors the PR is trying to change: Telegram draft streams can receive an implicit reply target, and the global assistant prompt still tells models to prefer `[[reply_to_current]]`. The PR is recent, focused, and the provided discussion says the intended Telegram change is narrow and covered by tests, so there is no conservative close reason here.
