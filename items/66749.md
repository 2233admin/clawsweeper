## Summary

Keep this PR open. Security review found no supply-chain or permissions-sensitive surface in the provided two-file TypeScript/test diff.

## Best Possible Solution

Keep this PR open for maintainer review. The likely best path is a narrow guard that skips requestHeartbeatNow for isSubagentSessionKey(sessionKey) in both exec completion paths while preserving enqueueSystemEvent, plus regression coverage for both explicit exec system events and background notify-on-exit. If accepted, landing this PR or an equivalent patch should close #66748.
