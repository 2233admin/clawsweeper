## Summary

Keep this PR open. Current main still lacks a distinct Agents overview control for editing the global default primary model at `agents.defaults.model`; the existing selector remains selected-agent scoped and writes `agents.list[].model`. The PR still addresses a real missing UI path, but it needs a rebase or fresh port because the overview now lives in `ui/src/ui/views/agents-panels-overview.ts`.

## Best Possible Solution

Keep this PR open for maintainer review, rebase, or a fresh port. The concrete path is to add a distinct global default primary-model selector in `ui/src/ui/views/agents-panels-overview.ts`, thread an `onDefaultModelChange` prop through `ui/src/ui/views/agents.ts` into `ui/src/ui/app-render.ts`, update `agents.defaults.model` without touching per-agent `agents.list[].model` overrides, preserve existing fallback lists in canonical `fallbacks` form, and keep focused unit coverage for default-vs-override model state resolution.
