## Summary

Keep this issue open. Current main has retired the old direct-resume ACP relay path, but the active implementation still relies on session-scoped system events plus heartbeat wake delivery. That remains a plausible match for the heartbeat-disabled reproduction in the discussion, and the newer thread comment broadens the same yielded-parent symptom to native spawn completion as well. The same-author PR #60402 is still an implementation candidate, not a reason to close the issue before review or merge.

## Best Possible Solution

Keep this issue open. Review, rebase, and either land or close PR #60402, or replace it with a shared yielded-parent completion wake primitive. The accepted fix should add regression coverage proving child completion wakes a yielded parent when periodic heartbeat is disabled or absent, and should also evaluate the newer native-spawn report so the solution is not narrowly ACP-only if the bug is shared.
