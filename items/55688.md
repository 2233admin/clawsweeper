## Summary

Current `main` does not implement this PR's fix. The shared Volc coding catalog is still text-only, both `volcengine-plan` and `byteplus-plan` build directly from that catalog at runtime, and the docs still advertise those coding models as `text` input only. This remains a plausible open bugfix PR, not a conservative close candidate.
