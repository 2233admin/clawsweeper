## Summary

Keep open. Current `main` still promotes assistant text from mixed assistant+tool-call messages into subagent output selection, then freezes that value into `frozenResultText`; the latest release shows the same behavior.

## Evidence

- **Mixed assistant messages still become `latestAssistantText`:** `summarizeSubagentOutputHistory()` counts assistant tool calls but still assigns any extracted assistant text to `latestAssistantText` and `assistantFragments` without excluding messages that also contain tool calls.
