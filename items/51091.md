## Summary

Keep open. Current `main` intentionally keeps mutating tool failures visible even when `messages.suppressToolErrors` is true, and that behavior is covered by tests and changelog text. This PR would reverse shipped safety behavior, but that does not fit an allowed auto-close reason.

## Evidence
