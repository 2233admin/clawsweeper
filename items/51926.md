## Summary

Keep open. Current `main` still has the reported schema/implementation mismatch: the `nodes` tool schema makes `node` optional, but the `describe` branch still hard-requires it and does not use the existing default-node resolver path. The PR appears directionally correct, but it is not already implemented on `main`, and it still lacks a regression test for `action: "describe"` without `node`.
