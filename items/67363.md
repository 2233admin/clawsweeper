## Summary

Keep open. Current `main` still writes `candidate.snippet` directly into `MEMORY.md` during deep promotion. There is some upstream cleanup in light-phase chunking (for example dropping generic day headings and merging nearby bullet lines), but there is still no separate distillation/summarization step before durable promotion, so the reported behavior remains materially valid on `main`.
