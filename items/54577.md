## Summary

Keep this issue open. Current main still has adjacent observability pieces, including sessions, tasks, subagents, `sessions.changed`, tool events, and per-session trajectory export, but it does not ship the requested built-in agent activity timeline surface: no `openclaw timeline` command, no `/timeline` chat command, no Control UI/TUI timeline view, and no persisted or JSON-exportable cross-agent lifecycle timeline. Related #60814 is an open draft Control UI dashboard candidate, not a merged implementation of the full requested scope.

## Best Possible Solution

Keep this issue open until maintainers scope and ship the built-in agent activity timeline product. The likely path is either to broaden and land #60814 with the missing CLI command, `/timeline` chat command, TUI/Control UI surface, lifecycle event persistence/export contract, filters, tests, and docs, or to split a narrower core/gateway timeline implementation over existing session lifecycle, task, subagent, agent-event, and trajectory data. This should remain a core observability UX item rather than ClawHub-only work because it asks for built-in OpenClaw surfaces over OpenClaw's own agent lifecycle.
