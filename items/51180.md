## Summary

Keep open. Current `main` still truncates `allowFrom` to two entries in the live `channels status --probe` formatter, and the config-only fallback path also truncates to two entries. The PR addresses a real current bug, but closing it would be incorrect; if anything, maintainer review should decide whether to merge it as-is or broaden it to cover the fallback path too.
