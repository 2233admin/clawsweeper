## Summary

Current `main` no longer has the tool-result guard behavior this PR is trying to make configurable. The guard now truncates individually oversized tool results and throws on aggregate overflow instead of compacting or rewriting older tool results, so the PR’s motivating reproduction is obsolete against `aa3a5f9ff7907c2a41d6259d24b529bda41f293a`.

## Evidence
