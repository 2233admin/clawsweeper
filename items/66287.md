## Summary

Keep open. Current `main` still intentionally requires repeated/diverse recall signals before deep promotion, and the test suite explicitly asserts that a one-off recall does not get written to `MEMORY.md`. Docs show manual overrides/workarounds, not an implemented automatic path for single-occurrence important events.
