Keep #72053 open. Current main still sends the subagent task in both the subagent system prompt and the first user message, so the linked #72019 bug is not implemented on main. #72053 is a focused active PR with tests and a follow-up for multiline task preservation; the related #72028 is still an alternate open design, and #72026 was closed unmerged, so neither cleanly supersedes this PR.

If maintainers prefer the user-message-canonical design, close this only after selecting or landing the canonical alternate PR.
