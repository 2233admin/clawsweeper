## Summary

Keep open. Current `main` does not implement transport-level proxy authentication for mobile gateway connections. The pairing/setup payload still only carries the gateway URL plus bootstrap token, Android opens the WebSocket without any extra HTTP headers, and iOS uses `webSocketTask(with: url)` with no request/header customization. The reported feature remains unimplemented.
