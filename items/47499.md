## Summary

Keep open. Current `main` has partial noVNC observer plumbing for the sandbox browser, but it does not implement the requested dashboard-embedded VNC viewer or an agent-facing full desktop takeover flow. The request also does not fit `clawhub`: repo vision explicitly keeps better computer-use and web frontend ergonomics in OpenClaw scope.
