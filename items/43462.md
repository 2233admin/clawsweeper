## Summary

Keep this PR open. Current main still performs case-sensitive `memory/` directory-prefix checks in the moved memory host SDK and temporal-decay code, while the PR’s latest revision addresses the earlier dated-path regression. The useful path is to rebase/port the fix to current files rather than close it as obsolete.

## Best Possible Solution
