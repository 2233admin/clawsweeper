## Summary

Keep open. Current `main` does not implement this docs addition, so it is not closable as `implemented_on_main`. At the same time, current gateway security/origin rules make the PR text incomplete for browser Control UI access through third-party tunnels, so this is not a conservative cleanup close; it needs maintainer rework or author updates.
