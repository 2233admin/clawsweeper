## Summary

Keep PR #58373 open. Current main still has the reported gap: `ensureOpenClawModelsJson` returns `wrote: false` on `plan.action === "skip"` without bootstrapping a missing non-main `models.json`, while `agents.create` only creates workspace/config state and does not create that file. The linked bug #58289 remains open and the PR is a focused fix rather than obsolete or superseded.

## Best Possible Solution

Keep the PR open and finish it as the focused core fix: in the `skip` branch, bootstrap a missing non-main agent `models.json` from the main agent exactly once, without overwriting existing per-agent catalogs, with atomic/create-if-missing behavior, cache invalidation that observes main `models.json`, and regression coverage for the linked #58289 scenario.
