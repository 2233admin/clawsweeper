## Summary

Keep open. Current main has partial effective-tool visibility (`tools.effective`, `/tools`, and the Control UI), but the requested strict inheritance mode and declared-vs-effective removal trace are still not implemented. Runtime filters such as subagent policy still intentionally reduce tools after agent/global tool policy, and the public effective-tools schema only returns the tools currently available, not declared tools, removed tools, or removal sources. Related #59898 is adjacent no-tools prompt cleanup and does not supersede this broader capability/trace request.
