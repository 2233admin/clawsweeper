## Summary

Current main does not include this PR’s change, and the underlying Pi embedded-extension seam still has no path that would make `sessionKey` visible to registered `ExtensionFactory` plugins. The PR remains potentially useful as a starting point, but it is not implemented on `main` and the 3-line patch alone would not resolve the stated behavior.
