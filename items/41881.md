## Summary

Current `main` already has official multi-arch Docker publishing for `linux/amd64` and `linux/arm64`, but I found no `linux/arm/v7` build or manifest path. Because the issue title and summary explicitly ask for both ARM64 and ARMv7, the request is only partially implemented and should stay open.
