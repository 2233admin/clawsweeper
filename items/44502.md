## Summary

Keep open. Current main at 496d90c3b5ee849dba2f5aae3c8275236fa2ebb6 still uses broad Discord mention detection over the full base text plus generic mention metadata, and it does not implement the reported attention-window, other-configured-agent, role-label, or cross-agent bot-sender handling.
