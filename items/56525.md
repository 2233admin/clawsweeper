## Summary

Current `main` does not expose a persistent browser viewport config. The browser config types/schema/docs still have no `browser.viewport` key, docs only show manual `openclaw browser resize` / `openclaw browser set viewport` commands, and the browser plugin still creates a Playwright context with `browser.newContext()` and no viewport options.
