## Summary

Current `main` still has the validation hole this PR addresses in `skills/skill-creator/scripts/quick_validate.py`, so there is not strong evidence to close it as already implemented or obsolete. The only notable caveat is that the PR body overstates one downstream effect: `package_skill.py` names archives from the directory name, not the frontmatter `name`.

## Evidence
