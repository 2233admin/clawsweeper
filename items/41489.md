## Summary

Keep open. Current `main` no longer has the in-core `src/media-understanding/providers/*` architecture this PR targets, but AssemblyAI is also not implemented on `main`; the contribution is still plausibly useful as a rewrite onto the plugin-based media-understanding provider path. The PR discussion also surfaced timeout-budget concerns in the polling flow, so there is not strong evidence for an auto-close.
