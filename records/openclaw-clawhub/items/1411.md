## Summary

Keep open. Current main does not implement owner-editable skill summaries, and this PR remains the active implementation path for the related user request, but the proposed branch needs maintainer/product revision before it is mergeable.

## What This Changes

The PR adds a public Convex `updateSummary` mutation plus skill-detail header editing state, inline form controls, and CSS so authorized users can edit a skill summary without publishing a new version.

## Best Possible Solution

Land a revised implementation that adds a validated summary override/update path for skill owners, publisher managers, and moderators; keeps detail, search, API, and digest surfaces consistent; and defines how manual edits interact with future publishes and moderation state.
