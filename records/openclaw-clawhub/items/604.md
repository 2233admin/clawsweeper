Keep open. This is a ClawHub issue, and the supplied repository policy only allows auto-closing ClawHub pull requests that are certainly implemented on main. Current main still supports the reported failure shape: public profile lookup hides deleted/deactivated users, the profile route renders “User not found.” for that null result, and OAuth sign-in blocks inactive accounts rather than restoring access. The related recovery discussion and PR #1133 remain open, so this needs maintainer support rather than automated cleanup.
