Keep this open. ClawHub cleanup policy for this review keeps issues open, and the current source does not prove the deleted-account OAuth/session report is resolved. Main still treats self-deleted accounts as irreversible, blocks deleted/deactivated users during OAuth completion, returns null for inactive users, and has a related deleted-account login-loop report still open in #886.

## Best Possible Solution
