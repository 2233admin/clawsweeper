## Summary

Keep this PR open. Current main still does not implement the PR’s intended indexed direct prefix lookup for skill slug/display-name recall, and the PR is still paired with open bug #1254 via closing syntax. The later `kling` -> `klingai` and `ScienceClaw` reports reinforce that this recall class is still active rather than obsolete.

## Best Possible Solution

Keep this PR open for maintainer review, or replace it with an equivalent patch that adds indexed direct prefix lookup for skill slug/display name, normalized digest fields and indexes, migration-safe backfill, and regression coverage for the Midscene case plus the reported `kling` -> `klingai` and case-stability examples. Close #1254 only after an implementation merges.
