## Summary

Keep this PR open. Current main still lacks the PR's intended indexed direct lexical prefix lookup for skill slug/display-name recall, and the same-author issue #1254 remains open as the canonical bug paired with this PR. Later reports around `kling` -> `klingai` and `scienceclaw`/`ScienceClaw` point to the same unresolved recall class rather than making the PR obsolete.

## Best Possible Solution

Keep this PR open for maintainer review, or replace it with an equivalent patch that adds indexed direct prefix lookup for skill slug/display name, normalized digest fields and indexes, migration-safe backfill, and regression coverage for the Midscene case plus the reported `kling` -> `klingai` and case-stability examples. Close the paired issue only after an implementation merges.
