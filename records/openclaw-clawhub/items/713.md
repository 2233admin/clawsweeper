## Summary

Keep open. Repository policy for openclaw/clawhub only permits auto-closing pull requests that are certainly implemented on current main, and this is an issue. The underlying moderation appeal is also still current: the live API reports the reported skill as suspicious for `suspicious.prompt_injection_instructions`, and current main still flags `system prompt:` wording unless a moderator applies the existing clean manual override or maintainers intentionally change the scanner policy.

## Best Possible Solution

Keep this issue open for moderator or maintainer review. A moderator should inspect the `suspicious.prompt_injection_instructions` evidence for `agent-memory-sr` and apply the existing clean manual override if it is benign.
