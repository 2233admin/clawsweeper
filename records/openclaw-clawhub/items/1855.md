## Summary

Keep open. Current main still does not implement this PR's intended inspect behavior: `clawhub inspect` does not include skill-level moderation in JSON output, plain output does not print skill moderation from the skill detail response, and the version response schema still omits `version.security`.

## Best Possible Solution

The final change should pass `skillResult.moderation` through inspect JSON, print a concise skill-level moderation summary in plain `clawhub inspect <slug>` without requiring a version fetch, type version `security` in the CLI schema, and retain focused regression tests for text and JSON output.
