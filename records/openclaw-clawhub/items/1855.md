## Summary

Keep this PR open. Current main still does not implement the PR's central behavior: `clawhub inspect` parses skill-detail moderation from the API schema, but does not include it in JSON output and does not print a plain-text moderation summary. The PR remains a relevant draft implementation candidate for #1483.

## Best Possible Solution

Keep this PR open and review/land it, or an equivalent change, as the implementation path for #1483. The main fix should pass `skillResult.moderation` through inspect JSON, print a concise plain-text skill moderation summary without requiring a version fetch, keep version `security` typed in the CLI schema, and preserve focused regression coverage.
