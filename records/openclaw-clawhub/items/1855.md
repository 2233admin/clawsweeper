## Summary

Keep this PR open. Current main still does not implement the PR's central behavior: `clawhub inspect` parses skill-detail moderation metadata but omits it from `--json` output and never prints a plain-text skill-level moderation summary. The PR is still a useful draft implementation candidate for the related open CLI security request.

## Best Possible Solution

Keep this PR open as the implementation candidate for the related CLI security/moderation request. The best path is to review and land this PR or an equivalent change that passes `skillResult.moderation` through inspect JSON, prints a concise plain-text moderation summary, preserves version-level security output, and adds focused regression coverage.
