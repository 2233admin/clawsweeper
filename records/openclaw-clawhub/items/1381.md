## What This Changes

The issue asks how a user can delete a published skill.

## Best Possible Solution

Keep this open for a maintainer support response. The response should point the reporter to `clawhub delete <slug> --yes` for owner soft-delete, `clawhub undelete <slug> --yes` for restore, and the v1 API equivalents; it should clarify that permanent hard deletion is admin-only.
