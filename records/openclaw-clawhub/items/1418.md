Keep this PR open. Current main still implements `clawhub list` as a lockfile-only command, so the PR's intended behavior of listing manually installed/untracked skill directories has not shipped. The related cleanup PR #1419 is closed unmerged and does not supersede this work.

## Best Possible Solution
