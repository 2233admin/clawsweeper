## Summary

## What This Changes

The PR adds manual-skill detection for untracked skill directories, displays those skills in `clawhub list`, updates the list command help text, and adds helper tests.

## Best Possible Solution

Implement the feature in the active `packages/clawhub` CLI: add a manual-skill scan helper, wire it into `cmdList`, keep tracked and manual output clearly separated, update CLI help/docs, and cover helper plus command behavior with focused tests.
