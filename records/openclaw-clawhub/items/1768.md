Keeping this issue open. Current main still lacks native per-skill pinning, and an open implementation PR (#1806) with closing syntax is the right place to finish or reject the work. ClawHub cleanup policy also says issues in this repository should not be auto-closed by this pass.

## Best Possible Solution
