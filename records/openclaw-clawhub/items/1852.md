Keep open. This is a ClawHub issue, and the repository policy only permits auto-closing ClawHub PRs that are certainly implemented on main. Current main has package registry trust controls, capability metadata, provenance/source fields, scans, and audit logs, but not the requested PAP-style principal hierarchy, transport-layer authorization, context minimization, inter-agent receipt model, or runtime trust boundary.
