## Summary

Keep this issue open. The linked `apple-notes-native` skill is already published externally, and current ClawHub main has the registry, metadata, public API, moderation, and install surfaces needed for this kind of macOS skill. The unresolved part is maintainer product/security judgment about whether to recommend or replace an integrated memo-based Apple Notes path. Under the ClawHub cleanup policy, issues should stay open unless a maintainer resolves that product decision.

## Best Possible Solution

Keep this issue open for maintainer product/security review. The published `apple-notes-native` ClawHub skill can remain the community path; maintainers should separately decide whether to document it as a recommended alternative or track an OpenClaw core/default-skill change after evaluating reliability, macOS Automation permissions, setup friction, and support burden.
