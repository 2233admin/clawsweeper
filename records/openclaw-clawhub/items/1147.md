## Summary

Keep open. This is an open ClawHub issue, and the repository-specific cleanup policy only permits auto-closing ClawHub pull requests that are certainly implemented on current main. ## Best Possible Solution

Keep this issue open for maintainer/security moderation review. A moderator should inspect the current `unraidclaw` source/package and the published `openclaw.plugin.json`; if reviewed safe, apply the existing manual clean override with an audit note. If LAN-IP placeholders are acceptable for plugin UI hints, narrow the static scanner so `uiHints` placeholders are not treated as install sources, or ask the publisher to change the placeholder format.
