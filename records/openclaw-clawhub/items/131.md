## Summary

Keep this issue open. The ClawHub cleanup policy for this review keeps issues open, and current main still has source-level evidence for the reported publish/update visibility failure: an existing active skill update can be patched to a hidden moderation state, and public search/list paths then drop it.

## Best Possible Solution

Keep this issue open. Fix the skill publish update path so publishing a clean new version of an already-active skill preserves public search/list/detail visibility while scanner review runs, except for static-malicious uploads, quality quarantine, removed/manual moderation locks, or publishers under moderation. Add regression coverage for active skill updates, public search/list/detail visibility, owner-visible moderation state, and slug ownership preservation; coordinate with #134 and decide whether to revive or replace the closed-unmerged #274 approach.
