## Summary

Keep open: current main still has the duplicated local helpers, so the PR is not implemented on main, and the proposed shared helper would silently change existing skills and souls pagination limit coercion from rounding to truncation.

## What This Changes

The PR adds a shared `clampInt` helper in `convex/lib/math.ts` with tests and replaces local helper definitions across 11 Convex modules with imports from it.

## Best Possible Solution

Land the deduplication only after preserving existing rounded behavior for skills and souls or explicitly standardizing truncation as the accepted integer coercion contract with focused tests for public pagination and maintenance batch-size behavior.
