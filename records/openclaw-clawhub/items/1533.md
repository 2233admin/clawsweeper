## Summary

Keep this PR open. Current main still has the duplicated `clampInt` helpers and does not contain the shared `convex/lib/math.ts` helper, while the PR still needs maintainer judgment because it changes `skills.ts` and `souls.ts` non-integer limit behavior from `Math.round` to `Math.trunc`.

## What This Changes

The PR branch adds a shared `convex/lib/math.ts` `clampInt` helper with tests and replaces local helper definitions across 11 Convex modules with imports.

## Best Possible Solution

Keep this PR open. The best path is to deduplicate `clampInt` into a shared helper while either preserving the existing `Math.round` behavior for `skills.ts` and `souls.ts` pagination limits or getting explicit maintainer approval for switching those call sites to `Math.trunc`, then rebase and run the standard checks.
