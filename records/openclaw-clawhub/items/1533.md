## Summary

Keep this PR open. Current main still has the duplicated local `clampInt` helpers, does not contain `convex/lib/math.ts` or its test, and the PR still carries an unresolved behavior change from `Math.round` to `Math.trunc` for float pagination limits in `skills.ts` and `souls.ts`.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to deduplicate `clampInt` into a shared helper while preserving existing `Math.round` behavior for `skills.ts` and `souls.ts`, or explicitly approve and document the switch to `Math.trunc` before merging.
