## Summary

Keep this PR open. Current main still has the generic report and moderation pipeline, but it does not include the PR’s intended bad-faith/trademark reporting guidance in the docs. Under the ClawHub policy, an open contributor PR should only be auto-closed when main already implements the intended change, and that is not true here.

## Best Possible Solution

If maintainers agree that trademark and bad-faith registration disputes should use the existing report plus staff moderation pipeline, land this docs/security.md change or equivalent guidance; otherwise revise it to document the preferred private legal/trademark intake path for #1591.
