## Summary

Keep open. ClawHub review policy only allows auto-closing PRs that are already implemented on main, and this is an issue. The requested PowerShell upload support is also still incomplete: current main has no explicit .ps1 allowlist entry, so browser, Convex, archive, GitHub import, and CLI paths can reject or skip PowerShell files unless a text MIME type happens to be supplied.

## Best Possible Solution

Keep this issue open and either consolidate it with open related issue #290 or implement explicit PowerShell support if maintainers accept the policy tradeoff. A complete implementation should add ps1 to the relevant allowlists, normalize or guess a text content type for PowerShell files, update web/server/archive/CLI/GitHub import tests, and document the allowed-file behavior.
