Keep open. Current main still intentionally rejects GitHub OAuth reauth for deactivated or legacy-deleted users, and the issue discussion plus related open reports show multiple affected users. ## What This Changes

The issue requests that GitHub OAuth sign-in recover, relink, or recreate a ClawHub account after the user deleted their previous ClawHub account.

Keep this as the canonical deleted-account GitHub OAuth recovery triage item.
