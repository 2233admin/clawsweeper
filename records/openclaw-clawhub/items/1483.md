## Summary

Keep open. This is an open ClawHub issue, and the repository review policy says ClawHub issues should remain open for maintainer triage. Current `main` also does not fully satisfy the request: the API/schema exposes skill-level moderation data and install/update have partial safeguards, but `clawhub inspect` still omits the skill-level `moderation` block from both text output and JSON output. An open draft PR, #1855, is the current implementation candidate and explicitly references this issue with closing syntax, so this issue should stay open until that PR is reviewed, merged, or closed.

## Best Possible Solution

Keep this issue open for maintainer triage. Review and either land or close PR #1855 as the implementation candidate for `clawhub inspect` text/JSON moderation visibility. If it lands, decide whether benign-only install/filtering should be included in the same resolution or split into a focused follow-up that wires existing `nonSuspiciousOnly` API support into CLI search/explore or install flows.
