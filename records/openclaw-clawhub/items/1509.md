## Summary

Keep open. This is an open ClawHub issue, and the repository-specific cleanup policy only allows automated closure for pull requests that current main definitely implements. Current main still has a plausible CLI failure path for `SKILL.md required`, and open draft PR #1856 is the active implementation candidate that should be reviewed, landed, or replaced before this issue closes.

## Best Possible Solution

Keep this issue open while maintainers review and land PR #1856 or an equivalent CLI fix. The likely fix is to ensure `clawhub publish` always preserves a root `SKILL.md` or `SKILLS.md` manifest even when ignore rules exclude other markdown files, add regression coverage for that case, and point affected users to `https://clawhub.ai/publish-skill` as the manual upload workaround until release.
