## Summary

Keep this issue open. EcoCompute is already available as a published ClawHub skill, but the request asks maintainers to promote it as recommended, official, built-in, or runtime GPU-monitoring behavior. Current main has skill publishing/discovery, staff-pick and official-badge curation, and package-catalog exposure for official skills, but it does not bundle EcoCompute or implement automatic GPU energy monitoring. ClawHub review policy also keeps issues open for maintainer judgment.

## Best Possible Solution

Keep this issue open for maintainer triage. EcoCompute can remain available as the published ClawHub skill; if maintainers want to endorse it, use the existing Highlight or Mark official curation path. If automatic GPU monitoring or bundled runtime behavior is desired, track that as explicit OpenClaw core/plugin design work with security and maintenance review.
