## Summary

Keep this PR open. Current main still lacks the PR's intended manifest recovery: `cmdPublish` only publishes files returned by `listTextFiles`, and `listTextFiles` applies ignore rules before upload validation, so a broad markdown ignore can still hide the root `SKILL.md`. The PR is a focused draft implementation candidate for the linked open bug #1509, not work already shipped on main.

## Best Possible Solution

Keep this draft PR open for maintainer review. The likely path is to land or adapt the localized `cmdPublish` fallback, preserve normal `listTextFiles` ignore semantics, keep the broad-markdown-ignore regression test, and close #1509 only after the fix merges and ships.
