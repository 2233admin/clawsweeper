## Summary

Keep this draft PR open. Current main still does not implement the intended root-manifest recovery: `cmdPublish` still publishes exactly the files returned by `listTextFiles(folder)`, while `listTextFiles` applies ignore rules before publish validation. A broad `*.md` ignore can still remove the required root `SKILL.md`/`SKILLS.md`, and current tests do not cover the regression this PR adds for linked #1509.

## Best Possible Solution

Keep this draft PR open for maintainer review. The likely path is to land or adapt the localized `cmdPublish` root-manifest fallback, preserve shared `listTextFiles` ignore semantics, add the broad markdown ignore regression test, and close #1509 only after the fix merges and ships.
