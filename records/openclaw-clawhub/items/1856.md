## Summary

Keep open. Current main still does not implement the PR's intended fix: `cmdPublish` depends directly on `listTextFiles(folder)`, and `listTextFiles` applies root ignore rules before the publish command checks for a root `SKILL.md`/`SKILLS.md`. This draft PR remains a focused implementation candidate for the open linked bug #1509, not obsolete work already shipped on main.

## Best Possible Solution

Keep this draft PR open for maintainer review. The likely product path is to land or adapt the focused `cmdPublish` fallback, keep `listTextFiles` ignore semantics intact for normal discovery, add the regression test for broad markdown ignore patterns, and then close the linked bug after the fix merges and is released.
