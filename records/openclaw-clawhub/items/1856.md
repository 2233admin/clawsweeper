## Summary

Keep this PR open. Current main still does not implement the PR's intended fix: publish still validates only the files returned by normal discovery, while normal discovery can ignore root SKILL.md/SKILLS.md before publish checks for a manifest. The PR remains a focused draft implementation candidate for the linked open bug #1509.

## Best Possible Solution

Keep this draft PR open for maintainer review. The likely path is to land or adapt the focused cmdPublish fallback, keep listTextFiles ignore behavior intact for normal discovery, add the ignored-manifest regression test, and close #1509 only after the fix merges and ships.
