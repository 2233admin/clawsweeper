## Summary

Keep this issue open. The ClawHub cleanup policy for this review keeps issues open, and the substantive docs gap is still real: current main has generic reporting/moderation and admin remediation tooling, but no merged public guidance for trademark or bad-faith skill registration disputes. PR #1860 is the active draft implementation candidate and explicitly references this issue for closure.

## Best Possible Solution

Keep this issue open and review PR #1860 or an equivalent maintainer-approved docs/policy change. The best path is to document how to report bad-faith registrations, impersonation, and trademark-related skill disputes; specify useful non-sensitive evidence; warn against posting private legal materials in public reports; and explain how staff may use existing report, hide, hard-delete, owner-transfer, reclaim, and ban tooling after review.
