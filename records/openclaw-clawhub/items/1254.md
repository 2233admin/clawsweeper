Keep open. Current main still lacks the requested indexed normalized slug/display-name prefix recall path, and ClawHub cleanup policy keeps issues open unless they are PRs already implemented on main.

## What This Changes

Request an indexed lexical recall path so display-name and slug prefixes such as Midscene, Gaggiuino, and ScienceClaw are returned even when vector recall misses them.
