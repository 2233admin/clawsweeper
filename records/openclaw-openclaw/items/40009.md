## Summary

Keep open. Current main still renders the Control UI Raw config surface as a native textarea, with textarea-specific styling and tests. The requested structured JSON editor with folding, highlighting, validation, and formatting is not implemented on main, and the direct CodeMirror candidate PR #56401 plus adjacent Raw JSON5 readability PR #40732 are both still open rather than shipped.

## Best Possible Solution

Keep this issue open and review or consolidate #56401 and #40732 into one tested Control UI Raw config editor implementation. The fix should add JSON/JSON5 syntax highlighting, folding, validation/error display, full-document formatting, and editor-state coverage while preserving raw availability, sensitive redaction/reveal, reset-to-saved, raw diff, `onRawChange`, `config.set`, `config.apply`, and `baseHash` behavior.
