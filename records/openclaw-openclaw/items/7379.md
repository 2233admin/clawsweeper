Keep open. ## What This Changes

Request to add GitHub Copilot Premium/Chat quota-window percentages to the per-response `/usage` footer when `/usage tokens` or `/usage full` is enabled.

## Best Possible Solution

Add a generic reply-footer path that, when `/usage tokens` or `/usage full` is enabled and the active provider exposes usage windows, appends the existing `formatUsageWindowSummary` output to the footer while leaving provider-specific auth and fetching inside plugins. Cover the reply-runner/footer path with tests and update the token-use docs plus changelog.
