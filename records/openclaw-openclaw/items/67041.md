## Summary

Keep this PR open. Current main still lacks shared default redaction for Basic auth headers, standalone JWTs, and x-pomerium-jwt-assertion values on the logs.tail path, and the linked bug remains open. The PR is a focused implementation candidate, but it needs the existing review comments addressed before merge.

## Best Possible Solution

Keep this PR open as the implementation candidate for #66832. Ask for a small amendment: fix the Basic auth boundary so padded credentials are fully masked, add focused regression tests for padded Basic auth, standalone JWTs without a Bearer prefix, x-pomerium-jwt-assertion, and the logs.tail/redactSensitiveLines path, then run the targeted logging tests before merge.
