## Summary

Keep this issue open. Current main has runtime disable toggles for Bedrock, GitHub Copilot, Hugging Face, and Ollama discovery, but public docs still only document the Bedrock discovery config path and there is still no typed global model-discovery disable control. The related implementation PR #33365 is closed unmerged, so there is no open or merged canonical fix that resolves the remaining docs/global-control gap.

## Best Possible Solution

Keep this issue open. The low-risk next step is to document the shipped plugin-owned toggles in public docs for GitHub Copilot, Hugging Face, and Ollama, and to clarify the legacy models.*Discovery.enabled aliases while they remain supported. Separately, maintainers should decide whether to add a typed global model-discovery disable control and how it should interact with models.mode and plugin-owned discovery.
