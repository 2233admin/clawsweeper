## Summary

Keep open. Current main still clears the Control UI composer draft during session switches, and this PR head is incomplete against one visible session-selection path while also overlapping open PR #54655. The branch is useful prior work, but it needs maintainer direction rather than automatic cleanup.

## What This Changes

The PR adds an in-memory per-session chat draft cache and wires save/restore helpers into Control UI session and agent switching callbacks.

## Best Possible Solution

Ship one rebased Control UI change that preserves composer drafts in the centralized session-switch helper, removes stale draft entries for empty or deleted sessions, keeps existing per-session queue behavior intact, and adds focused tests for draft restoration across session and agent switches.
