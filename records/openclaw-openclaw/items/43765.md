The PR modifies heartbeat system-event handling, Feishu runtime health reporting, exec/process terminal status semantics, ACPX Codex command resolution, ACP spawn guidance, reply first-visible diagnostics, config/docs, lockfile entries, and related tests.

## Best Possible Solution
