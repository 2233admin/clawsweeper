## Summary

Keep open. Current main still drops preserved deleted-agent transcript directories from `sessions.usage`, so the PR is addressing a real gateway usage bug, but the current patch has a CI-blocking test type issue and the PR head is not green.

## What This Changes

The PR adds usage-only disk-backed session discovery for preserved agent transcript directories, lets transcript discovery accept an explicit `sessionsDir`, and expands `sessions.usage` regression coverage.

## Best Possible Solution

Land this PR or a narrow successor after fixing the type/CI failure and accepting the usage-only disk discovery design, so preserved transcript files continue to count in usage while normal agent-list visibility remains configured-agent scoped.
