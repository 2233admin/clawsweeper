## Summary

Keep this PR open. Current main still lacks the PR’s typed manifest-load failure reason and still records every manifest load failure as an error, so the narrower workspace missing-manifest behavior is not implemented. The supplied PR remains a focused candidate after the earlier closed #60701 attempt.

## Best Possible Solution

Review and land this PR or an equivalent change that adds a typed manifest-load failure reason, downgrades only workspace-origin missing-manifest candidates to warnings, preserves errors for non-workspace and unsafe/parse/invalid failures, and keeps the focused regression tests.
