## Summary

Keep this PR open. Current main still lacks WSL/font guidance in the Feishu and WhatsApp terminal QR login paths, and the channel docs do not already document the MS Gothic workaround. The PR is focused and does not broaden security-sensitive surfaces, but it needs maintainer follow-up because the current diff imports the deprecated broad `openclaw/plugin-sdk/infra-runtime` surface and has no targeted WSL/non-WSL stdout coverage.

## Best Possible Solution

Keep the PR open for normal review. The best path is a small channel-owned revision that adds WSL-only terminal QR guidance for Feishu and WhatsApp, uses an approved focused plugin SDK import for WSL detection or adds one, and includes targeted tests for WSL and non-WSL output. After that, land the PR rather than closing it as obsolete.
