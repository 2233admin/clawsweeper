## Summary

Keep this PR open. Current main still lacks WSL/font guidance in the Feishu and WhatsApp terminal QR login paths, and the relevant channel docs do not already document this failure mode. The PR is a focused, recent channel UX fix with no apparent security or supply-chain expansion, but it still needs normal maintainer review for the SDK import surface and regression coverage before merge.

## Best Possible Solution

Keep the PR open for normal review. The best path is a small channel-owned revision that adds WSL-only terminal QR guidance for Feishu and WhatsApp, uses an approved focused SDK/runtime import for WSL detection or adds one, and includes targeted tests for WSL and non-WSL output. After that, land the PR rather than closing it as obsolete.
