Land a tightened version after maintainer review: define whether the prompt file must be absolute, outside the workspace, or snapshotted; update Zod/types plus generated schema/help/labels/docs/baseline; and keep coverage for both CLI and embedded prompt composition without changing unrelated prompt mechanisms.
