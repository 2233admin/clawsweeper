## Summary

Keep this PR open. Current main still truncates cosine-similarity norms to the shared vector length in the active memory-host SDK helper, so the reported bug remains. The submitted PR is still useful conceptually, but its patch targets the removed `src/memory/internal.ts` path and needs a rebase or maintainer port to the current SDK implementation before it can land.

## Best Possible Solution

Keep this PR open and ask for a rebase or maintainer port of the same norm calculation fix to `packages/memory-host-sdk/src/host/internal.ts`, with the mirrored `src/memory-host-sdk/host/internal.ts` kept aligned if it remains, plus focused regression coverage for zero-padded different-length vectors. After that refreshed diff is reviewed, it should be mergeable if it stays scoped.
