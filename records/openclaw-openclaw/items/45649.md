## Summary

Keep this open. Current main still carries the same Control UI readability path described in the issue: collapsed tool summaries are the default visible surface, the text is small/muted with ellipsis truncation, the shell remains capped on wide layouts, and the UI-specific detail formatter can still add the awkward connector wording discussed in comments.

The issue asks for Control UI v2 tool bubbles to be readable, less prematurely truncated, less click-heavy, and cleaned up from awkward detail wording.
