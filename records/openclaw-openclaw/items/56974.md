## Summary

The requested refactor is still actionable on current main. Shared helpers exist, but several local async file/path existence helpers named by the issue are still present, and the marketplace pathExists duplicate is also still local.

## Best Possible Solution

Keep this issue open and implement a scoped refactor PR. A conservative path is to replace access-based read-only checks with the existing src/utils.ts pathExists where imports are already appropriate, replace the stat-based duplicate with an existing or newly neutral shared async helper, leave src/infra/state-migrations.fs.ts alone, and add or adjust narrow tests only where helper semantics are part of observable behavior.
