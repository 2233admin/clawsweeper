## Summary

Keep open. Current main still has cron descriptions in the shared job contract and Control UI edit flow, and it renders the actual system-event or agent-turn prompt text, but the cron job card/detail renderer does not visibly display the saved description as the requested human-readable summary.

## What This Changes

Show a cron job's prompt/task and human-readable summary from the Web UI cron dashboard without requiring users to inspect config files.

## Best Possible Solution

Render saved cron descriptions as a visible Summary in the Control UI cron job card or selected-job detail area for both systemEvent and agentTurn jobs, while keeping the existing cron storage/RPC contract, payload display, and run-history selection behavior intact.
