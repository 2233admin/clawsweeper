## Summary

Keep this PR open. Current main still does not implement the requested channel icons in the Sessions list or channel card titles, so the contribution is not obsolete. It should not be merged as-is because the supplied PR diff and review comments still show a dm-key peer extraction bug, duplicated icon mapping, and incomplete/stale session row contract handling.
