## What This Changes

The issue reports that Docker-based `openai-codex` OAuth requests fail with Cloudflare HTML challenges, missing public Responses OAuth scope, and a possible onboarding auth-store wipe side effect.

## Best Possible Solution

Keep this issue open as the Cloudflare/headless-container tracker.
