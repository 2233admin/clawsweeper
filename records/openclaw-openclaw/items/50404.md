## Summary

Keep open. Current main still has only partial session-management support: Gateway session APIs, a chat-header session selector, and a separate Sessions table. It does not yet provide the requested native Control UI left sidebar with chronological conversation history, search, inline rename/context actions, archive behavior, and ChatGPT-style switching. The open #73725 PR covers the narrower New Chat create-and-switch slice, while related duplicate issues have already been consolidated toward this more focused sidebar tracker.

## What This Changes

Request to add a Control UI session sidebar for saved conversation history, new-chat creation, renaming, deletion/archive actions, switching, search, and persistence.

## Best Possible Solution

Keep this as the focused native Control UI sidebar/session-history tracker. Review and land or close #73725 for the immediate New Chat create/switch behavior, then define the sidebar UX for search, rename, delete/archive, active highlighting, derived titles/previews, and mobile behavior before implementing it against the existing Gateway session APIs and updating the Control UI docs.
