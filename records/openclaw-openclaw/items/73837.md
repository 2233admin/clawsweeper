## Summary

Keep open. Current source and the v2026.4.26 source tag contain installer hardening for noninteractive apt and gum ioctl fallback, but the documented hosted installer still serves an older v22-era script that matches the report. There is also an open unmerged PR, #73855, with closing syntax for this issue, so this issue should not be closed before that implementation candidate is reviewed/landed or closed.

## Best Possible Solution

Keep this issue open until maintainers redeploy or sync https://openclaw.ai/install.sh from the hardened v2026.4.26/current source, then revise and land a hosted-installer drift guard such as #73855 without blocking legitimate installer PRs. A final verification should prove the hosted script hash or required markers match source, and ideally cover WSL2 Ubuntu with no preinstalled Node so the installer does not report Node success unless node is actually available and satisfies the required version check.
