## What This Changes

Add a Control UI session dropdown filter that lets users hide or show main, subagent, dreaming, cron, isolated, and related session kinds instead of showing every session in one noisy list.

## Best Possible Solution

Keep this issue open until PR #73311 is reviewed and either landed or closed.
