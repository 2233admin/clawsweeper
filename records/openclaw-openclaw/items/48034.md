## Summary

Keep this PR open for maintainer follow-up. Current main still does not implement the intended rw-mode skill sync, and the submitted PR is not safe to close as obsolete or merge as-is: the sync remains unreachable in the normal rw path, the PR diff imports the bundled skill resolver from the wrong module, and it includes an unrelated locale change that current main already covers.

## Best Possible Solution

Revise or replace this PR with a focused sandbox fix in `src/agents/sandbox/context.ts`: prepare a sandbox-readable skill mirror for rw sessions, make prompt skill locations resolve to readable sandbox-workspace copies, preserve current skill eligibility/filtering behavior, and add a targeted rw-mode regression test. Leave the already-implemented locale dropdown change out of the sandbox PR.
