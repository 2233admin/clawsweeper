## Best Possible Solution

Land the narrow Matrix-owned transport mitigation if Matrix maintainers accept the temporary synthetic response semantics, keep the upstream matrix-rust-sdk issue as the durable OTK ID-tracking fix, and require the focused Matrix tests plus the changed-lane gate before merge.
