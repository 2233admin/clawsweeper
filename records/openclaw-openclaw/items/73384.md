Keep open. The PR is recent, active, and stacked with same-author follow-up #73526, so it needs maintainer API/security review rather than conservative cleanup closure.

Keep this PR open as the canonical consolidated workflow follow-up to merged #72287. Coordinate that decision with stacked follow-up #73526.
