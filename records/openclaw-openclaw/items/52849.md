## Summary

Keep open. Current main has a trusted bundled tool-result middleware that can rewrite results before model consumption, but the public plugin hook requested here is still absent, and external plugins are explicitly blocked from the existing middleware seam. The related #62123 remains useful context, but it has not been merged or established as the canonical replacement for this issue.

## Best Possible Solution

Keep this open and either consolidate it with #62123 under one maintainer-owned design thread, or implement a documented, versioned public hook/trust policy for pre-model tool-result mutation. The design should cover allow/replace/block semantics, async failure behavior, provenance metadata, size bounds, Pi and Codex runtime ordering, SDK docs, and contract tests.
