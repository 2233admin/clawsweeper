Keep this PR open. Current main still has the old iOS/macOS shared chat composer attachment UI: a toolbar paperclip plus a small horizontal capsule strip. The PR's main user-facing change, thumbnail tiles with a persistent add-more tile in the attachment area, is not implemented on main and the PR is a plausible focused implementation candidate.
