Keep this PR open. Current main still has the old chat composer attachment UI: the PhotosPicker lives only in the toolbar, the toolbar is hidden when the iOS composer is focused/compacted, and the attachment row remains a capsule strip without the persistent trailing add-more tile. The PR is still the active implementation candidate rather than obsolete cleanup.

## Best Possible Solution
