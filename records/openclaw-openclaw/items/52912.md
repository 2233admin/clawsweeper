## Summary

Keep this PR open for maintainer review, but do not merge it as-is. Current main still lacks the intended Anthropic token sibling sync, while the PR broadens sibling writes to every OAuth credential and only patches the plugin auth-choice path, leaving the linked CLI token surfaces unresolved.

## What This Changes

The PR routes plugin auth-choice profile writes through a new sibling-sync helper, syncs Anthropic token and all OAuth credentials to sibling agent directories, adds Anthropic token sync coverage, and adds a DeepSeek provider contract-registry import.

## Best Possible Solution

Land a narrower maintainer-approved fix that centralizes token auth-profile persistence for Anthropic setup-token and manual-token contexts, covers interactive plugin auth, non-interactive setup-token, and paste-token where intended, keeps OAuth copying explicitly opt-in, and splits unrelated registry cleanup out of the auth PR.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Restrict OAuth auth-profile sync to intended contexts:** `src/plugins/provider-auth-helpers.ts:222-223`
  - body: The new sync policy returns true for every OAuth credential, so plugin OAuth logins would write refresh material into all sibling agent stores. That breaks the documented default that OAuth profiles are non-portable unless a provider explicitly opts in, and can overwrite credentials for agents that should stay isolated. Scope sibling sync to Anthropic token credentials or an explicit provider/context opt-in.
  - confidence: 0.92
- **[P2] Cover the models auth token paths:** `src/plugins/provider-auth-choice.ts:133-135`
  - body: This change only routes the plugin auth-choice runner through the new sibling writer. The user-facing `models auth setup-token` and `paste-token` paths persist through `src/commands/models/auth.ts`, and the Anthropic non-interactive setup-token path writes directly in the plugin, so the linked CLI failure can still reproduce after this PR merges.
  - confidence: 0.86
