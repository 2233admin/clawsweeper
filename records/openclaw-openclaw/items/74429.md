Keep open. Current main does not implement `openclaw config apply`, and this active PR adds a new config mutation command with docs/tests while still carrying human-review state, failed check runs, and concrete bot findings around stdin and `--replace-path` semantics.

The PR adds `openclaw config apply` for JSON5 config patches from file/stdin, recursive object patching, null deletes, explicit `--replace-path` replacement, shared validation/dry-run handling, nested SecretRef checks, tests, and CLI/exe.dev/Slack/Discord docs.
