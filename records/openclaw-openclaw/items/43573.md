## What This Changes

Land a narrow, rebased ACP-only fix that writes the initial spawn task to the child transcript before returning, dedupes only the seeded replay by provenance/marker, reports delivery truthfully, preserves current main's newer ACP behavior, and moves workflow policy changes to a separate maintainer-reviewed PR if still needed.
