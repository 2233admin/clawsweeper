- **[P2] Move session ingestion changes to the current SDK host files:** `src/memory/session-files.ts:3-7`
  - body: Current main no longer uses `src/memory/session-files.ts`; the live implementation is in `packages/memory-host-sdk/src/host/session-files.ts` with a mirror under `src/memory-host-sdk/host`.   - confidence: 0.9
- **[P2] Restrict refresh audit-log permissions:** `extensions/memory-lancedb/index.ts:703-711`
  - body: The new refresh audit log is created with default `mkdir` and `appendFile` modes. Replacement metadata such as memory IDs and operation times should not depend on the process umask; create the directory and file with private modes such as `0700` and `0600` or use an existing private state helper.
  - confidence: 0.84
- **[P2] Keep full memory IDs in forget candidates:** `extensions/memory-lancedb/index.ts:531`
  - body: Current main intentionally prints full UUIDs in `memory_forget` candidate lists so callers can pass the selected `memoryId` back to UUID-validated operations. This branch still renders only `r.entry.id.slice(0, 8)`, which leaves the displayed candidate ID unusable for delete/replace calls unless the caller inspects hidden details.
  - confidence: 0.86
