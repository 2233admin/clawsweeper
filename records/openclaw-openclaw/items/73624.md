## Summary

Keep open. Current main still has the reported prompt-composition failure: native subagent spawn carries the delegated `Your Role` task through `extraSystemPrompt`, while embedded, compaction, and CLI prompt assembly still short-circuit to `systemPromptOverride` before the builders that append that runtime prompt. The linked open PR #73637 remains the implementation candidate, so this issue should not be closed before that PR or an equivalent fix lands.

## Best Possible Solution

Review and land PR #73637, or an equivalent source change, so `systemPromptOverride` is preserved while runtime `extraSystemPrompt` is appended in embedded attempt, compaction, and CLI prompt assembly. Add regression coverage proving a subagent spawned under an agent with `systemPromptOverride` receives both the override text and the generated `Your Role` task block.
