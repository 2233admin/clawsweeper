## Summary

Keep this issue open. Current main has the backend pending exec approval listing API (`exec.approval.list`) and tests for it, but still does not expose the requested `openclaw approvals pending` CLI command, CLI tests, or CLI docs. The two related implementation PRs were closed unmerged, so there is no shipped user-facing CLI surface to close against.

## Best Possible Solution

Keep this issue open until OpenClaw adds `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts`, backed by the existing `exec.approval.list` gateway method. The first pass should include human-readable table output with command details, `--json` output for monitoring, focused tests in `src/cli/exec-approvals-cli.test.ts`, and docs updates in `docs/cli/approvals.md` plus any CLI index/nav reference. Treat `--watch` as a follow-up unless maintainers explicitly scope it into the first CLI surface.
