## Summary

Keep this PR open. Current main still has the two failing install paths reported by #74380: CodexBar is modeled as a plain brew formula, and summarize still points at the removed `steipete/tap/summarize` formula. The proposed branch is relevant and focused, but it needs normal maintainer review because it changes an installer metadata contract and moves one bundled skill from Homebrew tap install to npm install.

## What This Changes

This PR adds a `cask?: boolean` skill install metadata flag, uses it to build and label `brew install --cask` commands for CodexBar, and changes the bundled summarize skill installer from `steipete/tap/summarize` to the `@steipete/summarize` node package.

## Best Possible Solution

Land a reviewed fix that makes brew casks explicit in the skill installer contract, updates all intended metadata/type mirrors, changes the bundled CodexBar and summarize installers to currently valid install sources, and adds focused regression coverage for the installer command/status behavior.
