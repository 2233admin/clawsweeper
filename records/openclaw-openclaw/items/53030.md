## What This Changes

The PR expands the Japanese docs glossary and refactors the docs i18n glossary checker to auto-detect glossary languages, support `--lang`, and dynamically exclude translated docs directories from the English docs scan.

## Best Possible Solution

Land a refreshed docs-i18n update that keeps the useful ja-JP glossary backfill, makes the checker fail closed for selected glossary validation and missing glossary inventories, preserves existing zh-CN behavior, and adds focused regression coverage for multi-locale detection and explicit locale selection.
