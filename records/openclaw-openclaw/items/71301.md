## Summary

Keep open. Current main still only provides partial support: bundled authored docs, prompt guidance to read local docs, and config field lookup. The requested native, version-coupled docs retrieval surface for agents remains absent, and the reporter clarified they want upstream native search updated with deployments rather than an external scraper workaround.

## What This Changes

Ship a version-matched documentation bundle with OpenClaw releases and expose local-first docs retrieval tools for agent-guided onboarding.

## Best Possible Solution

Keep this open until OpenClaw has a release/package docs manifest or index, local-first docs search/get retrieval exposed to agents, explicit live-docs fallback behavior, and tests proving package-installed docs discovery plus agent/tool behavior for the installed version.
