## Summary

Keep open. Current main still waits on active descendant subagents with the full cron timeout and suppresses parent delivery without consulting `deliveryBestEffort`; latest release `v2026.4.26` shows the same code, and the related fix PR #46160 was closed unmerged.

## Best Possible Solution

Fix `src/cron/isolated-agent/delivery-dispatch.ts` so best-effort isolated cron text delivery does not wait for or suppress active fire-and-forget descendants, while preserving existing non-best-effort descendant synthesis/suppression behavior. Add a regression test for `deliveryBestEffort: true` plus active descendants, and update cron docs/changelog if the behavior surface changes.

## Clownfish Work Prompt

Fix the isolated cron best-effort active-subagent timeout bug tracked by openclaw/openclaw#44428. In `src/cron/isolated-agent/delivery-dispatch.ts`, best-effort isolated cron text delivery should not wait for active descendant subagents or suppress the parent cron output solely because fire-and-forget descendants are still running. Gate the `waitForDescendantSubagentSummary` call and the `activeSubagentRuns > 0` early return on `!params.deliveryBestEffort`, while preserving existing non-best-effort behavior and completed-descendant fallback/synthesis behavior. Add a regression test in `src/cron/isolated-agent/delivery-dispatch.double-announce.test.ts` where `deliveryBestEffort: true` and `countActiveDescendantRuns()` is positive: it must not call `waitForDescendantSubagentSummary`, must attempt outbound delivery of the parent output promptly, and must not return the active-descendant suppression result. Keep the existing non-best-effort active-subagent suppression test passing. Update `docs/automation/cron-jobs.md` and/or `docs/automation/tasks.md` if the best-effort exception needs user-facing documentation, and add a single-line changelog fix entry with proper attribution if the repo’s active changelog requires it. Do not change subagent runtime execution, scheduler timeout policy, or delivery behavior for non-best-effort jobs.
