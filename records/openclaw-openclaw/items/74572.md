## What This Changes

The issue reports that Active Memory lightweight embedded recall runs cannot see Memory Core plugin tools, causing the explicit memory tool allowlist to resolve empty and persist a failure placeholder turn.

## Best Possible Solution

Review and land or close #74592. The desired end state is a generic, extension-agnostic fix where Active Memory recall can materialize only explicitly allowlisted memory plugin tools before the final embedded-run filter, and unavailable memory tools do not leave a persistent user-visible placeholder in transcripts.
