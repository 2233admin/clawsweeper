## What This Changes

This PR refactors `mapSensitivePaths` in `src/config/schema.hints.ts` to delegate traversal to an in-place helper, documents the mutation contract, and avoids recursively spreading the hints map.

## Best Possible Solution

Land this narrowly scoped config-schema performance fix after maintainer review, rebase onto current main if needed, and rerun the targeted schema-hints test plus the normal changed gate; keep the documented in-place mutation contract because current callers consume the returned hints map as authoritative.
