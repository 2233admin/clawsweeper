Keep open. Current main still collects gateway health state but does not pass or render it in the Overview snapshot grid, and this PR targets the right UI boundary but is not merge-ready because it is stale against the current locale registry and required Overview test props.

The PR threads gateway health into the Control UI Overview renderer, adds a Health stat with OK/Issues/n/a display, and adds a `common.issues` translation key to several locale bundles.

## Solution Assessment
