## Summary

Keep open. Current main still lacks the Overview health stat, and this PR is still pointed at the right bug, but the patch is stale against the current Control UI i18n surface because main now ships nineteen locale bundles while the PR updates the older six-locale set.

## What This Changes

The PR threads gateway health into the Control UI Overview renderer, adds an OK/Issues/n/a Health stat, and adds `common.issues` translations to several locale files.

## Best Possible Solution
