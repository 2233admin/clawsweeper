## Summary

Keep this PR open. Current main still lacks the proposed system-event audience primitive and drain-time internal runtime-context wrapping, while this PR is a recent focused implementation candidate for the open canonical system-event leakage issue #69492.

## What This Changes

This PR adds an optional `SystemEvent.audience` / enqueue option, wraps internal-audience events in runtime-context delimiters during prompt drain, marks isolated-cron awareness events internal, and adds focused regression coverage.

## Best Possible Solution
