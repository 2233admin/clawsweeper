## Summary

Keep open. Current main has meaningful partial UX work, including Quick Settings, schema-backed config editing, and a Control UI ClawHub skills panel, but it still does not provide the requested WordPress-like guided configuration layer or dashboard-native ClawHub plugin discovery, installation, and configuration flow.
