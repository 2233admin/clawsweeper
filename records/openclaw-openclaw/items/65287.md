## Summary

Keep open. Current main still lacks Anthropic-native compaction params, Anthropic `context_management.edits` injection, compaction block round-trip handling, and the requested mixed 1h stable-prefix / 5m trailing-conversation cache policy. Related #10213 tracks server-side Anthropic compaction, but not the mixed TTL requirement, and the direct implementation PR #65288 was closed unmerged.

Keep this issue open unless maintainers explicitly fold its mixed Anthropic TTL requirements into #10213. The best implementation path is a focused Anthropic transport/cache-policy PR: add typed Anthropic compaction params, gate the required beta headers, inject `context_management.edits`, round-trip compaction blocks through history and streaming, and update payload-policy tests/docs so long retention uses 1h only for stable prefix material while trailing conversation remains 5m/default TTL.
