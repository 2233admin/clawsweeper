## Summary

Keep open. Current main still lacks the requested Anthropic-native compaction params, `context_management.edits` injection, compaction-block round trip, and mixed 1h/5m Anthropic cache TTL policy. Related #10213 tracks server-side compaction but does not preserve the mixed TTL acceptance criterion, and the direct implementation candidate #65288 was closed unmerged.

## Best Possible Solution

Keep this issue open, or explicitly fold its mixed Anthropic TTL acceptance criteria into #10213 before closing it as a duplicate. The likely implementation belongs in the Anthropic transport and payload policy: add typed run/model params, gate beta headers, inject `context_management.edits`, preserve compaction blocks through history and streaming, and update cache policy/tests so long retention applies only to stable prefix material while trailing conversation remains short/default TTL.
