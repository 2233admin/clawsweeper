## Summary

PR #72306 should stay open. Current main at 998e37fcb3d5b937fcf70d7509b677efba939f10 still has the gateway pricing-refresh lifecycle gap the PR targets: the refresh timer is not unref'd, upstream fetches are only timeout-aborted, and shutdown only clears a scheduled timer without aborting an in-flight refresh. The linked regression #72208 remains concrete and open, and the PR is a focused, test-backed gateway bug fix rather than obsolete, duplicate, stale, or ClawHub work.

## Best Possible Solution

Keep #72306 open for maintainer review and likely merge, or land an equivalent minimal fix in src/gateway/model-pricing-cache.ts that unrefs the scheduled pricing refresh timer, threads a shutdown AbortController through both LiteLLM and OpenRouter fetches, aborts it from the returned stop callback and test reset path, and preserves the new regression tests in src/gateway/model-pricing-cache.test.ts.
