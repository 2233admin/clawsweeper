## What This Changes

The issue requests a fix for `npm install -g openclaw@latest` crashing with a V8 fatal error during the official WSL2 Ubuntu 24.04 installer flow.

## Best Possible Solution

Reproduce the official installer on clean WSL2 Ubuntu 24.04.4 with Node 22.22.x, capture the npm debug log and V8 crash dump, and isolate whether the crash happens before, during, or after OpenClaw `postinstall`.
