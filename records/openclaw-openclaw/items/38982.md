## Summary

Keep this PR open. Current main still has startup-only TUI light/dark palette selection through OPENCLAW_THEME and COLORFGBG; it does not implement this PR's central runtime behavior: no /theme command, no /theme list, no runtime theme application API, no searchable theme picker, and no live-preview cancel/commit flow. The branch remains relevant, but it overlaps the open broader theme-system PR #44382 and still needs maintainer review plus cleanup of unrelated .secrets.baseline churn.

## Best Possible Solution

Keep this PR open for maintainer review and coordinate it with #44382. The best path is one focused TUI theme implementation that adds a runtime theme API, registers /theme and /theme list, opens a searchable picker with live preview, commits on select, reverts on cancel, repaints all TUI chrome consistently, documents the public behavior at https://docs.openclaw.ai/web/tui, includes targeted command/theme tests, and removes unrelated .secrets.baseline churn. If maintainers choose #44382 as canonical, close or supersede this PR only after that decision is explicit.
