## Summary

Keep this PR open. Current main still hardcodes the Control UI chat transcript as a polite live region during streaming, and the linked accessibility bug remains open. This PR changes the Control UI chat transcript to use `aria-live="off"` while sending or streaming and adds focused DOM assertions for idle, streaming, and sending states.
