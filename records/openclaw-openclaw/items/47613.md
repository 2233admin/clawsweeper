Keep open for maintainer review. Current main still has the callback-coupled Gateway reasoning-stream gap, and this PR is a focused implementation candidate, but its stale diff would drop the current reasoning visibility gate and overlaps other open PRs that need a canonical maintainer choice.

The PR changes `src/agents/pi-embedded-subscribe.ts` so `reasoningMode: "stream"` can broadcast `stream: "thinking"` agent events without requiring `onReasoningStream`, while still calling that callback when present.
