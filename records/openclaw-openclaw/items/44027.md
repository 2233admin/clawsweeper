## Summary

Keep this PR open. Current main still does not implement the PR’s intended `/verbose on` or `/verbose full` override for group text tool summaries; the dispatch gate suppresses ordinary non-forum group tool summaries without consulting verbose state. The branch is still a plausible bug-fix candidate, but it needs a rebase and a current-main design pass because group source delivery now defaults to `message_tool_only`, and the submitted diff would also suppress native command-source tool summaries that current tests expect to deliver.

## Best Possible Solution

Keep this PR open as a focused implementation candidate, but require a rebase and update against current main. The best path is to define the interaction between `/verbose on|full` and `messages.groupChat.visibleReplies`, then implement a narrow opt-in for eligible group text tool summaries while preserving native delivery, Slack non-direct suppression, forum behavior, sendPolicy/source-delivery policy, preview suppression, exec approvals, and media-only forwarding. Add regression tests for default group behavior, configured automatic group behavior, verbose off/on/full, and native command-source delivery.

## Risks / Open Questions

- Merging the PR unchanged would likely regress current native command-source behavior because the branch suppresses native text tool summaries while current main tests expect native text summaries and media to be delivered.
- A current implementation must account for `messages.groupChat.visibleReplies: "message_tool"`; otherwise `/verbose` may still appear broken in default group/channel turns, or it may unintentionally bypass a room-visibility safety policy.
- Verbose group tool summaries can expose tool arguments, URLs, plugin diagnostics, and data the model saw, so the fix should remain explicit opt-in and preserve Slack non-DM, sendPolicy/source-delivery, preview-suppression, exec-approval, and media-only behavior.
- The PR currently lacks focused current-main regression tests for group verbose off/on/full and adjacent delivery gates.
