## Summary

Keep this PR open. Current main has the broader MEDIA local-path documentation, and Feishu action sends now forward mediaLocalRoots, but the Feishu auto-reply dispatcher still lacks the PR's user-facing blocked-local-media hint and one-time final-reply fallback. Its reply-path media send also still omits mediaLocalRoots.

## Best Possible Solution

Keep this PR open for maintainer review. Rebase it onto current main, preserve the newer MEDIA documentation, and decide whether to land the narrow Feishu fallback/hint together with or after the related reply-path mediaLocalRoots plumbing.
