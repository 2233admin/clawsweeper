## Summary

Keep this PR open. Current main still lacks the Feishu reply-dispatcher mediaLocalRoots plumbing this PR adds, so the PR is not obsolete or implemented. The PR is also functionally relevant: Feishu media sending already requires scoped local roots for local file paths, while the reply dispatcher does not receive or forward them. A security pass found the PR patch is limited to two Feishu source files and does not touch workflows, dependencies, lockfiles, install scripts, secrets, or publishing metadata.

## Best Possible Solution

Keep this PR open for maintainer review, or supersede it with one canonical Feishu reply-path fix. The chosen patch should add mediaLocalRoots to CreateFeishuReplyDispatcherParams, compute roots with getAgentScopedMediaLocalRoots at both Feishu bot dispatcher call sites, forward them into sendMediaFeishu, and add a focused reply-dispatcher regression test for the forwarding.

- There are several overlapping open Feishu mediaLocalRoots PRs, so maintainers should choose one canonical patch to avoid divergent partial fixes.
