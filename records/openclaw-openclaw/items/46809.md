Keep open. Keep open and implement a focused Control UI memory surface that reuses the existing safe agent workspace file machinery where possible, adds a narrowly validated way to read/write today's `memory/YYYY-MM-DD.md`, and presents `USER.md`, `MEMORY.md`, and the daily log together for the selected agent.
