## What This Changes

The PR adds an internal Tree-sitter-backed shell command explainer under `src/infra/command-explainer`, with tests and new `web-tree-sitter`/`tree-sitter-bash` dependencies.

## Best Possible Solution

Keep the PR open for maintainer review, regenerate the lockfile with only the intended dependency changes, resolve parser WASM assets from package/module locations, and then validate the focused explainer test plus the changed gate before deciding how it should feed approval UX.

Yes for the review findings: the lockfile drift is directly reproducible from the PR patch, and the `process.cwd()/node_modules` asset resolution is visible in the new runtime. I did not execute tests because this review was constrained to read-only inspection.

## Solution Assessment

No. A Tree-sitter command explainer is a plausible direction, but this patch is not the best merge shape until supply-chain drift is removed and runtime asset resolution works outside repo-root cwd.

Overall confidence: 0.88

Full review comments:

- **[P1] Drop the registry-specific tarballs:** `pnpm-lock.yaml:1613`
  - body: The lockfile now pins hundreds of existing packages to `packages.atlassian.com` even though the feature only needs two new dependencies. Current `main` does not have these Atlassian tarballs, so this would redirect installs of unrelated native/runtime packages through a third-party mirror. Please regenerate the lockfile with the repo's normal registry so only the intended new dependency entries change.
  - confidence: 0.94
- **[P2] Resolve WASM assets from package locations:** `src/infra/command-explainer/tree-sitter-runtime.ts:6-10`
  - body: The runtime looks for both WASM files under `process.cwd()/node_modules`, but OpenClaw commands and the gateway can run with cwd set to a user workspace or packaged install location. In that case `explainShellCommand` will fail to load the parser. Resolve the assets via package exports, `createRequire(...).resolve`, or an `import.meta.url` anchored path instead.
  - confidence: 0.87
