## Summary

Keep this PR open. Current main still lacks the requested usage-panel behavior for long skill/tool/file names: the context breakdown names do not expose full values with `title` attributes, and current usage CSS has no scoped ellipsis rule for `.context-breakdown-item .mono`. The PR remains a focused UI fix, but its style change needs to be ported from the removed split usage-styles path into the current consolidated stylesheet.

## Best Possible Solution
