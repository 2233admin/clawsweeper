## Summary

Keep open. The PR targets a real unresolved current-main bug, but the proposed patch is incomplete for manifest suppressions that depend on configured provider context, and the branch also carries an unrelated ClawHub skills global-install feature that should not ride along with this model fix.

## What This Changes

The PR adds an inline-row suppression check and regression test for `openai-codex/gpt-5.4-mini`, while also adding `--global` ClawHub skills install/update CLI behavior, tests, and docs.

## Best Possible Solution

Land a narrowly scoped resolver change that mirrors the existing suppression call with provider config context, covers both unconditional and conditional suppression cases in tests, and moves the skills global-install feature to its own reviewed PR.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Pass suppression context for inline matches:** `src/agents/pi-embedded-runner/model.ts:597-600`
  - body: The new inline-row check calls `shouldSuppressBuiltInModel` with only `provider` and `id`. Manifest suppressions can depend on the configured provider API/baseUrl, such as the Qwen Coding Plan suppression, so inline rows with `api` can still bypass those suppressions and resolve. Mirror the existing check below by passing `baseUrl: providerConfig?.baseUrl` and `config: cfg`.
  - confidence: 0.88

## Security Review

Status: needs_attention

Summary: The model fix itself does not add secrets or network calls, but the branch also adds unrelated global ClawHub skill installation behavior that broadens third-party skill visibility.

Concerns:

- **[medium] Split the global ClawHub install surface:** `src/cli/skills-cli.ts:97`
  - body: `--global` installs or updates downloaded ClawHub skills under the shared managed skills directory, making them visible to all local agents by default. That expands the reach of third-party skill content and is unrelated to the model-suppression PR body, so it should be split or explicitly security-reviewed before merging this model fix.
  - confidence: 0.86
