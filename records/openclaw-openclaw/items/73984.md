## Summary

Keep this PR open. Current main still builds TUI autocomplete from the local `getSlashCommands(...)` list and has no `TuiBackend.listCommands` seam, so plugin command inventory from `commands.list` is not hydrated into TUI slash suggestions. The PR targets the right surface and remains a relevant implementation candidate for commands such as `/active-memory`, though it still needs maintainer review around refresh behavior before landing.

Keep this PR open as the focused implementation candidate.
