## Summary

Keep open: the narrow Control UI URL fix is still useful on current main, but this PR is not merge-ready because part of the URL fix targets stale status code and the branch also includes unrelated agent error-copy changes plus a bundled Drop Box document-intake skill with concrete security/privacy review concerns.

## What This Changes

The PR normalizes LAN Control UI display links in configure/status callers, changes embedded-agent error formatting and payload tests to add extra context and transport hints, and adds a bundled Drop Box document-intake skill with a PDF rendering helper.

## Best Possible Solution

Land a narrow current-main fix that centralizes display-only LAN-to-loopback normalization for browser Control UI URLs and applies it to dashboard, configure completion, status overview/status-all, daemon status, and setup finalization while preserving runtime bind, probe, custom-bind, tailnet, basePath, and TLS semantics. Split the embedded-agent error-copy work and Drop Box skill into separate reviews.
