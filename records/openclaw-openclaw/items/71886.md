## Summary

Current main still lacks the CSP allowance this PR adds. The Control UI documents and implements browser-local tweakcn theme imports, and the importer fetches `https://tweakcn.com/r/themes/<id>` directly, but `buildControlUiCspHeader()` still emits `connect-src 'self' ws: wss:` without `https://tweakcn.com`. The PR remains a narrow, relevant fix with matching regression coverage.

## Best Possible Solution

Review and likely land this narrow CSP allowance with the regression assertion, keeping it limited to the exact tweakcn `connect-src` origin required by the documented browser importer. Leave configurable CSP and Permissions-Policy behavior to #47565.
