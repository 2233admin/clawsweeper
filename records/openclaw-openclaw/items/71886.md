## What This Changes

The PR adds `https://tweakcn.com` to the Control UI `connect-src` CSP directive and adds a Gateway HTTP regression assertion for that header.

## Best Possible Solution

Land the exact-host CSP allowlist and regression assertion after maintainer review and CI confirmation; keep the configurable CSP and Permissions-Policy design tracked separately in #47565.

Yes. Current main can be checked statically: the importer fetches `https://tweakcn.com/r/themes/<id>` from the browser, while the served Control UI CSP omits that origin from `connect-src`.

## Solution Assessment

Yes. For the bundled tweakcn importer, adding one HTTPS origin to `connect-src` is the narrowest maintainable fix and does not grant script execution, image wildcards, or subdomain access.
