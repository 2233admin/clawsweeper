## Summary

Keep open. The report is fresh, corroborated by several users, and current main contains partial latency diagnostics and mitigations after v2026.4.26 but no complete, shipped fix for the broad 4.22/4.23 to 4.26 reply/reaction slowdown.

## What This Changes

The issue asks OpenClaw to fix a regression where upgrading from 4.22/4.23 to 4.26 makes Feishu reactions/replies and other chat surfaces noticeably slower with the same configuration.

## Best Possible Solution

Keep this as the broad upgrade-regression tracker until maintainers can compare v2026.4.22/v2026.4.23, v2026.4.26, and current main with the same redacted setup and per-phase timings, then split proven contributors into narrower fixes and close this only after user-visible latency is measurably back within budget or fully covered by canonical follow-ups.
