## Summary

Keep this PR open. Current main still treats malformed percent-encoded canvas paths as resolver exceptions that become 500 responses, and the proposed resolver-level fallback is a narrow, relevant fix with no blocking review findings found in this read-only pass.

## What This Changes

The PR wraps canvas file path normalization in a `URIError` guard that returns `null` from `resolveFileWithinRoot` and adds a regression test for malformed percent-encoded canvas paths returning 404.

## Best Possible Solution

Merge resolver-level malformed-encoding handling with regression coverage after normal maintainer review and CI, preserving the existing filesystem root, traversal, symlink, and A2UI serving contracts.
