## What This Changes

This PR hides tool-result/tool/function and tool-call-only assistant plumbing from the web chat history, and strips SECURITY NOTICE / EXTERNAL_UNTRUSTED_CONTENT markers during UI text extraction.

## Best Possible Solution

Keep this PR open for maintainer review or an updated contributor patch.
