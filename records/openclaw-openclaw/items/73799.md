Keep this PR open. Current main still does not treat `application/msword` or `application/x-cfb` as binary media, and the PR is a focused implementation candidate for the still-open legacy `.doc` attachment bug. The provided PR context also shows it carries forward useful prior work from #54380 and related attempts while adding the explicit regression coverage missing from main.
