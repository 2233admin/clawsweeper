## Summary

Current main does not implement this PR's intended ACP background-task terminal-summary fix. The progress cap is still 240 characters, ordinary non-blocked successful completions still fall through without a terminalSummary, and the delivery formatter only includes terminalSummary in the user-visible completion message. This PR remains a focused implementation candidate for #74070, with maintainer review and regression coverage still needed.

## Best Possible Solution

Keep this PR open as the focused implementation candidate for #74070.
