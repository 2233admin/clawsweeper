## Summary

Keep this PR open. Current main still does not implement the PR’s intended skill-loading diagnostics: malformed or incomplete SKILL.md files are dropped silently, missing skill roots return an empty result, and the proposed parse-failure/discovery/summary log messages are absent. The PR remains a plausible implementation candidate, but it needs updates for the review and security findings before it could land.

## Best Possible Solution

Keep the PR open or ask the contributor to refresh it against current main, preserving the useful skill-loading diagnostics while sanitizing or compacting path/error details, aggregating or capping repeated warnings, fixing discovery/duplicate metrics, and testing through the current subsystem logger string output. If maintainers prefer, reimplement the same diagnostics directly in src/agents/skills/local-loader.ts and src/agents/skills/workspace.ts with focused coverage in the existing skills tests.
