## Summary

Keep this PR open. Current main still rejects workspace bootstrap symlinks whose targets resolve outside the workspace and then renders those bootstrap files as missing. A maintainer replacement PR, #73401, is open but not merged, so it is not a basis to close this contributor PR yet.

## Best Possible Solution

Keep this PR open until maintainers either land #73401 or explicitly choose this branch. The best implementation belongs in `src/agents/workspace.ts`: allow only fixed top-level bootstrap symlinks when accepted, preserve hardlink, non-regular, dangling, oversized, fd identity, and target-identity protections, keep extra/glob bootstrap paths constrained to the workspace, and retain regression tests for accepted and rejected cases. After the chosen patch merges, close the overlapping PRs and reports against the merged implementation.
