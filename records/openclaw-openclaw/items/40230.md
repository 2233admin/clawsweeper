## Summary

Keep this PR open. Current main still has no symlink fallback in the workspace bootstrap reader, so fixed bootstrap files that resolve outside the workspace are rejected by the boundary layer and rendered as missing. The open maintainer replacement #73401 may be the better landing path, but it is not on main, so this contributor PR remains useful until a replacement merges or maintainers explicitly close it.

## Best Possible Solution

Keep this PR open while maintainers review or land #73401, or explicitly choose this branch. The best implementation should live in `src/agents/workspace.ts`, allow only fixed top-level bootstrap symlinks when accepted, preserve hardlink, non-regular, dangling, oversized, fd identity, and target-identity protections, keep extra/glob bootstrap paths constrained to the workspace, and retain regression tests for accepted and rejected cases.
