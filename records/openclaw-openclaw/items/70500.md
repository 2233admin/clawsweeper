## Summary

Keep open. PR #70500 is a focused docs-only change that adds an explicit warning about granting macOS Accessibility to a shared `node` runtime. Current main documents stable app-bundle TCC permissions and PeekabooBridge using OpenClaw.app permissions, but it does not contain the specific Node/CLI runtime warning or the proposed cross-link from Peekaboo docs.

## Best Possible Solution

Keep #70500 open for maintainer review. The likely best outcome is to merge or adapt the explicit Node/CLI Accessibility warning into `docs/platforms/mac/permissions.md` and keep a concise cross-link from `docs/platforms/mac/peekaboo.md`, aligned with the existing signed app / TCC permission model.
