## Summary

Keep this PR open. Current main still does not implement `agents.defaults.systemPrompt.identityLine` or `disabledSections`, and the existing `systemPromptOverride` is a full prompt replacement rather than an identity-line override. The PR remains a plausible implementation candidate for the open identity-preamble request in #43562, but it needs schema, docs, runtime wiring, tests, and product/security review before merge.

## Best Possible Solution

Keep this PR open as an implementation candidate. Align it with #43562, decide whether the public scope is only a safe identity-line override or also section disabling, then add typed config, strict schema, generated help/docs, runtime wiring through every prompt assembly path, focused validation tests, and end-to-end prompt composition coverage before merge.
