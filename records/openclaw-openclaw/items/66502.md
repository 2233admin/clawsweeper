## Summary

Keep this PR open. Current main still lacks the stable pnpm package-root fallback for a versioned `.pnpm/.../node_modules/openclaw/openclaw.mjs` entrypoint, so the PR is not obsolete or implemented on main. The diff is narrow and remains a plausible implementation candidate, though it should be adapted to current wrapper-aware daemon install code and should verify stable-path preference when both stable and versioned candidates exist.

## Best Possible Solution

Keep this PR open as a focused implementation candidate.
