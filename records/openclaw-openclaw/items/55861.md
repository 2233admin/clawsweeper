## Summary

Keep open. Current main still has the 78px collapsed Control UI rail and lacks this regression test, so the PR is not obsolete. The proposed one-line CSS change is useful for the linked Safari/macOS overlap bug, but it changes the shared base rail variable and therefore also widens the 769-1100px fixed tablet drawer rail, which is outside the stated desktop fix.

## What This Changes

The PR widens the Control UI collapsed nav rail from 78px to 96px and adds a browser regression test that compares collapsed content placement against the nav rail.
