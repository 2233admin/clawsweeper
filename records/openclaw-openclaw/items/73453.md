## Summary

Keep this PR open. Current main still has the WhatsApp group mention bug the PR targets: implicit self-chat mode is derived from `allowFrom` without checking whether the conversation is a group, so group `mentionedJids` can bypass identity-overlap detection when the linked self number is in `allowFrom`. The linked bug #49317 is still open, and this PR is a focused implementation candidate rather than obsolete cleanup.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land a focused WhatsApp plugin fix that confines implicitly derived self-chat suppression to non-group conversations, preserves explicit self-chat overrides and direct self-DM behavior, adds the LID/group mention regression test, keeps the changelog credit, and then closes #49317 after merge.
