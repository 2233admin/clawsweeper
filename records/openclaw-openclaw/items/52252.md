## Summary

Keep this PR open. Current main still has no Volta tools/image-to-shim rewrite, so the PR’s intended daemon fix is not implemented on main. The PR is focused, but the existing review finding about custom VOLTA_HOME and Windows Volta layouts should be addressed before merge.

## Best Possible Solution

Keep this PR open and update it, or land a replacement, to implement Volta tools/image path rewriting in `src/infra/stable-node-path.ts` with regression tests for default `~/.volta`, custom `VOLTA_HOME`, POSIX and Windows separators, and inaccessible-shim fallback. Then review/land it as the implementation candidate for #52184.
