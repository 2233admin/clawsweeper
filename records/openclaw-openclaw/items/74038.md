Keep open. Current main still gates gateway model-pricing bootstrap only on `models.pricing.enabled !== false`, so `models.mode: "replace"` does not skip the OpenRouter/LiteLLM pricing bootstrap yet. This PR is a focused implementation candidate for the open #74020 behavior, with matching tests/docs/config updates and no unexplained security-sensitive file changes, but it is not obsolete or implemented on main.
