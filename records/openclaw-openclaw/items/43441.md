## Summary

Keep this PR open. Current main still has the bundled-layout gap the PR is meant to fix: template fallback is a single module-relative `../../docs/reference/templates` path, cwd is checked before that fallback, and there is no packageRoot-null bundled `dist/` regression test. The PR remains a focused implementation candidate rather than obsolete work.

## Best Possible Solution

Keep this PR open as the implementation candidate. Maintainers should review the focused resolver change, keep package-root lookup first, add the bundled `../docs/reference/templates` module-relative fallback before cwd, preserve the packageRoot-null bundled-layout regression test, and land only after the relevant changed gate passes.
