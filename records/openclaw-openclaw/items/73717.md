## Summary

Keep this PR open. Current main still does not implement the intended `infer model run --model` canonicalization: the local path passes the raw override through, the gateway path splits and forwards raw provider/model strings, and current CLI tests still lack mixed-case catalog-ref coverage. The PR remains open and focused on open bug #73715; the diff is security-narrow and touches only CLI implementation plus tests.

## Best Possible Solution

Keep this PR open as the active implementation candidate for #73715. Review, rebase if needed, and land this PR or an equivalent fix that canonicalizes only unique case-insensitive catalog matches before both local and gateway `infer model run` dispatch, while preserving custom or ambiguous mixed-case refs and keeping the local/gateway regression tests.
