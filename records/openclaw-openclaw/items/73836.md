Keep this issue open. It is recent, actionable, and current main still has source-backed evidence for at least one reported symptom: outbound delivery mirroring reduces media URLs to filename text and appends a text-only delivery-mirror transcript entry, while Control UI image previews require structured attachment content. The broader active-run Control UI/gateway responsiveness and Telegram typing symptoms are not proven fixed, and the related active-run stall report #65522 remains open.

Keep this issue open as an actionable regression bundle.
