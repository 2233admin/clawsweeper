Keep this PR open. Current main still recreates `HEARTBEAT.md` for any workspace missing it because `ensureAgentWorkspace()` calls `writeFileIfMissing(heartbeatPath, heartbeatTemplate)` outside the `isBrandNewWorkspace` guard. The PR is still a focused fix candidate, but it is stale against current main because its test changes target `src/agents/workspace.e2e.test.ts`, while current coverage now lives in `src/agents/workspace.test.ts`. Keep this PR open as a focused bug-fix candidate. Rebase it onto current main, move/refit the regression coverage into `src/agents/workspace.test.ts`, set up the negative fixture with all starter files except `HEARTBEAT.md`, guard `writeFileIfMissing(heartbeatPath, heartbeatTemplate)` behind `isBrandNewWorkspace`, and run targeted workspace plus heartbeat validation before landing.
