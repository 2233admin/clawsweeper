## Summary

Keep this PR open. Current main and the latest release still use the old home-directory fallback chain with no Android/Termux PREFIX-derived fallback, while the PR remains a focused implementation candidate for the linked Termux crash and its tilde-expansion edge case.

## Best Possible Solution

Review and land this focused PR, or request a final guard adjustment if maintainers want stricter Termux detection. Preserve OPENCLAW_HOME, HOME, USERPROFILE, OPENCLAW_STATE_DIR, and OPENCLAW_CONFIG_PATH precedence, and keep the regression tests plus docs/changelog updates with the behavior change.
