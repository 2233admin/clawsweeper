Current main still does not implement this PR's intended gateway display-name stabilization. Gateway session rows can still derive `displayName` from stale `origin.label`, and the UI prefers that row value over its `Main Session` fallback; the PR remains a plausible focused fix but needs rebasing/adapting to the current shared row projection path.
