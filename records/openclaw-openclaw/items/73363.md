## Summary

Keep open. Current main still does not implement this PR's intended fix: the embedded runner creates `isReasoning` internally but drops it from returned payloads, and the cron isolated-agent summary/output pickers still filter only `isError`. The PR remains a focused implementation candidate for the #73186 Matrix/Feishu cron announce reasoning leak, with no security-sensitive diff surface apparent from the provided file list.

Keep this PR open and review it as the active focused implementation candidate for the #73186 cron announce reasoning leak.
