## Summary

Keep this PR open. Current main still rejects whitespace in Docker config/workspace paths, so the PR’s user-facing bug remains live. The PR is a plausible implementation candidate, but it is stale against the current script layout and still needs safer YAML emission before it can land.

## Best Possible Solution

Keep this PR open as a valid implementation candidate for #44599. The best path is to port the change to `scripts/docker/setup.sh`, add a small YAML scalar emitter that preserves spaces/backslashes and escapes apostrophes, cover config/workspace/extra mount paths with spaces and apostrophes in `src/docker-setup.e2e.test.ts`, and include the sandbox socket overlay in the quoting fix.
