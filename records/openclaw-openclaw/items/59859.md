## Summary

Keep this PR open. Current main still has no native Linux companion app, the public platform docs still describe Linux companion apps as planned, and the gateway native UI client allowlist still only covers macOS, iOS, and Android. This PR adds a C/GTK4/Libadwaita Linux companion under apps/linux with a GTK3 AppIndicator tray helper, native gateway/systemd/RPC/onboarding/diagnostics/management surfaces, Linux tests/docs, a LINUX_APP gateway client id, and a small agents.update default-agent contract fix.
