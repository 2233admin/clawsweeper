## Summary

Keep open. Current main still does not ship a native Linux companion app, docs still mark Linux companion apps as planned, and the gateway client/protocol paths do not include an `openclaw-linux` native client. The PR is tied to open maintainer product request #75 and overlaps with open alternative track #61576, so this needs maintainer product and security review rather than cleanup closure.

## Best Possible Solution

Keep this PR open and route it through maintainer product/security review for #75. The best path is to choose a canonical Linux companion direction, then require the selected track to settle gateway client ID/protocol contracts, lifecycle behavior, packaging/docs/labeler coverage, and the outstanding security and native-code review issues before merge.
