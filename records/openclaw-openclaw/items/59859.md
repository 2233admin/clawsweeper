## Summary

This PR should stay open. Current main still lacks a native Linux companion app, public docs still describe Linux companion apps as planned, and the gateway client registry/handshake paths do not recognize an `openclaw-linux` native UI client. The work is tied to open maintainer product request #75, with open alternative track #61576, so this needs maintainer product and security review rather than cleanup closure.

## Best Possible Solution

Keep this PR open and route it through maintainer product/security review for #75. The practical path is to choose whether this C/GTK implementation, the alternative #61576 Rust/GTK implementation, or a smaller staged implementation should become canonical, then require the chosen track to resolve gateway-contract, lifecycle, packaging, and security review issues before merge.
