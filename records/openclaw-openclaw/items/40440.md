## Summary

Current main still lacks a media-bearing Telegram pending-history path: no-mention group messages can record only text history, and later mention replay only builds media context from current and replied-to message media. The related fix PRs are both closed unmerged, so the reported bug remains valid and narrow enough for a replacement fix PR.

## What This Changes

Telegram group messages with media sent without mentioning the bot should retain usable media references in pending chat history so a later @mention can include those files in agent context.

## Best Possible Solution

Add an additive media-bearing pending-history contract, have the Telegram plugin attach bounded retained media references for mention-gated skipped group messages, merge those references into later MediaPaths/MediaTypes during mention replay, and cover the image-without-mention followed by @mention flow with focused Telegram and shared-history tests.
