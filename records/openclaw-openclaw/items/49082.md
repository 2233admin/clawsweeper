Keep this PR open. Current main still lacks the Feishu reply-path `mediaLocalRoots` propagation the PR targets, and the current PR head addresses the earlier review gap by wiring both Feishu bot dispatcher call sites. Overlapping PRs #51375 and #58591 exist, but there is no maintainer-selected canonical replacement in the provided discussion, so closing this PR now would discard a still-useful implementation candidate.
