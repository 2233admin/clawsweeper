## Summary

Keep open. Current main has useful flat session listing, session switching, labels/search, derived titles/previews, sub-agent primitives, and per-session overrides, but it still does not implement the requested project/topic directory, tags/archive/persistent pins/recommendations, side-by-side multi-model collaboration, role/consensus/meeting workflows, committee templates, task extraction, or project tracking. The issue discussion also has maintainer signal designating this as the canonical tracker after duplicate #27603 was consolidated here.

## What This Changes

Request broad WebChat conversation management with folder/tag/search/pinning/archive/summarization features plus multi-AI collaboration, roles, committee workflows, minutes, task extraction, and project tracking.

## Best Possible Solution

Keep this issue open as the canonical product tracker. Split it into milestones for session organization metadata and storage/protocol support, Control UI/WebChat directory navigation, and a separate multi-model collaboration design that reuses existing agents/sub-agents where possible. Treat #50404 and #73725 as narrower create/switch/sidebar slices, not replacements for this broader tracker.
