## Summary

Keep PR #69270 open. Current main still lacks several central behaviors from the PR: zero-token compaction accounting is skipped, subscribe-time compaction hooks still omit messageProvider, command and gateway reset hook contexts still omit messageProvider, and gateway manual compaction still leaves some stale cost/cache fields. The supplied PR discussion shows the author responded to review comments and expanded the patch into a targeted compaction/reset invariant fix, so this is still useful maintainer-reviewable work rather than obsolete cleanup.

## Best Possible Solution

Keep #69270 open for maintainer review. The best path is to either land an updated, reviewed version of this PR or split it into focused patches for hook messageProvider resolution and zero-token/cache accounting, preserving the supplied regression tests across agent, command, and gateway paths.
