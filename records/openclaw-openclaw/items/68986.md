For chunked or buffered final flushes, a last chunk ending with a literal `<fi`/`<think` prefix can be buffered and dropped before sanitization; current main fixes this by calling `stripBlockTags` in final mode for the last delivery chunk. Please carry that behavior forward when mapping `finalDelivery`.
