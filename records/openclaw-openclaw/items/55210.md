## Summary

Keep open. Current main already handles normal terminal stop chunks and usage-aware stream finalization, but the streaming error path still combines the error text with the stop choice, so the central compatibility bug is still real. The PR contains useful source work but is stale against current main and would regress the current finalization path unless reworked.

## What This Changes

This PR changes the Gateway OpenAI-compatible chat streaming helper and tests so non-terminal chunks carry nullable `finish_reason`, terminal stop chunks use empty deltas, and streaming errors emit error text before a separate stop chunk.

## Best Possible Solution

Carry forward the error-path split as a small current-main patch that emits the error text as a nullable non-final content chunk, lets the existing stop/finalization path emit the empty-delta stop chunk, preserves `stream_options.include_usage` behavior and disconnect cleanup, and updates focused Gateway streaming tests plus a changelog fix entry.

## Review Findings

Overall correctness: patch is incorrect

Full review comments:

- **[P2] Preserve the current stream finalizer:** `src/gateway/openai-http.ts:595-611`
  - body: The new `phase === "end"` branch closes the SSE response directly. On current main, lifecycle end can arrive before `agentCommandFromIngress` returns usage; `maybeFinalize()` waits for `finalUsage`, cleans up the disconnect watcher, writes the stop/usage chunks, then sends `[DONE]`. Closing here would drop the `stream_options.include_usage` final chunk and skip the normal cleanup path.
  - confidence: 0.9
- **[P2] Align the error test with nullable finish_reason:** `src/gateway/openai-http.test.ts:889`
  - body: The proposed implementation emits non-final content chunks with `finish_reason: null`, but the added error-content assertion requires the property to be absent. That makes the test inconsistent with the code under review; assert `null` for the error content chunk and keep the separate terminal stop assertion.
  - confidence: 0.93
