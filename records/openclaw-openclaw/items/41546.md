Keep this PR open. Current main still clears the Gateway Token while the Overview WebSocket URL is edited, and the linked bug #41545 is still open. The latest PR context shows an active, maintainer-repaired implementation candidate that preserves tokens only within the same normalized gateway scope while still clearing on real endpoint changes.
