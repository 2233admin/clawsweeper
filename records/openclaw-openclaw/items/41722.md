Keep this issue open. Fix the gateway-backed skill dependency installer path: classify Homebrew/Xcode CLT unsupported-macOS failures, return targeted remediation, avoid selecting or advertising brew when the host toolchain is known unusable, and retry only a viable alternate installer spec when metadata and policy allow it. Add focused regression tests for the classifier, brew selection, and fallback behavior.
