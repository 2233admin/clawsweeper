## Summary

Keep this PR open. Current main does not include the proposed `msteams-channel-archive` plugin, the `channel_deleted` plugin hook, or Teams `conversationUpdate/channelDeleted` dispatch/cleanup. Some adjacent Teams media/thread fixes have landed on main, but they do not replace the PR's central archive and deleted-channel lifecycle work.

## Best Possible Solution

Keep the PR open for maintainer review. Ask for a current rebase and a narrower split: first land a typed deleted-channel lifecycle event plus any documented, generic message metadata needed by plugins; then decide whether durable Teams channel archiving should be a bundled reviewed plugin or an external ClawHub/plugin package using those APIs.
