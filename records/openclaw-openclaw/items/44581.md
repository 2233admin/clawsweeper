Keep open. ## What This Changes

The PR branch adds OpenAI-compatible replay-time thinking-signature sanitization, history-aware signature preservation, mixed Gemini/Anthropic thought-signature cleanup, wrapped-Mistral strict9 detection, and targeted regression tests for those paths.

## Best Possible Solution

Keep this PR open and land a rebased current-main fix in the provider replay-policy architecture. The fix should preserve compatible provider-native thinking signatures until the replay target is known, strip unsupported signatures only for openai-completions including reasoning_details-era fields, keep Gemini thought_signature cleanup independent from Anthropic thinkingSignature preservation, and add scoped OpenRouter-wrapped Mistral strict9 handling without a broad global hint table.
