## Summary

Keep this PR open. Current main still performs shallow configured-root skill discovery: `loadSkillEntries()` iterates only direct child directories and skips a child when `skillDir/SKILL.md` is missing, so `skills/<group>/<skill>/SKILL.md` is not loaded. The PR remains a focused implementation candidate for the linked open bug #56915, and the provided discussion shows recent maintainer repair plus merge-ready labeling rather than an obsolete branch.

## Best Possible Solution

Review and land this PR, or an equivalent patch, as the canonical fix for #56915. The implementation should add bounded one-level grouped discovery in `src/agents/skills/workspace.ts`, preserve direct skill behavior, keep realpath/symlink and size/candidate limits intact, and ship the included regression coverage plus docs/changelog update.
