Keep the additive cron stable-id feature, but make generated Swift optional protocol fields source-compatible before merge, regenerate/check protocol artifacts, add the required `Unreleased` changelog entry with contributor credit, and then validate the cron service, gateway protocol, CLI, and Swift client surfaces together.
