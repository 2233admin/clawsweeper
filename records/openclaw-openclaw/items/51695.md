## Summary

Keep this PR open. Current main still has the minimal Tlon plugin README, so the intended README expansion is not implemented on main. The PR is coherent and docs-only, but it needs revision before merge because its setup command is invalid for the current CLI and its dependency list omits a declared, used runtime dependency.

## Best Possible Solution

Keep the PR open and ask for a small rework: expand `extensions/tlon/README.md` using current plugin terminology, say Tlon is bundled in packaged builds, use `openclaw plugins install @openclaw/tlon` only for older/custom installs, link https://docs.openclaw.ai/channels/tlon for full setup, and either include the complete runtime dependency list including `@aws-sdk/s3-request-presigner` or omit dependency documentation to avoid drift.
