## Summary

Keep this issue open. Current main implements several adjacent pieces, including sandbox routing, host exec approvals, compaction, JSONL session persistence, and hooks, but the specific OS-level exec isolation request is not fully implemented. An open draft PR, #62937, explicitly attempts to close this issue, so the right maintainer path is to review/land or close that PR before closing the issue.

## Best Possible Solution

Keep this issue open while #62937 is reviewed. If that PR is accepted, close this issue only after merge if it actually satisfies the central OS-level exec isolation request, or split the remaining permission-model/session-resume/compaction-hook leftovers into narrower issues. The best core direction is a generic, documented exec sandbox/permission contract that fits existing sandbox backends and approvals rather than a one-off macOS-only option.
