## What This Changes

The issue reports that after v2026.4.27 group/channel final replies default to private message-tool-only delivery across all channel plugins, with missing migration warning, misleading changelog wording, blocked agent config.patch access, and reload guidance concerns.

## Best Possible Solution

Keep the issue open until maintainers choose a coordinated upgrade path: all-channel release-note wording, operator-accessible visibleReplies configuration where appropriate, accurate hot-reload/restart guidance, and an explicit warning or migration/default decision that preserves the intended group-room safety posture.
