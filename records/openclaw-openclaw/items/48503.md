## Summary

Keep open. Current main still exposes only the smaller before_tool_call payload plus context fields; it does not surface the requested action classification, capability flags, or input provenance. The HELM OSS comment is useful prior-art context, and open PR #58079 is adjacent, but neither implements this request on main.

Keep this open for maintainer design and a coordinated additive SDK/runtime change: define stable action metadata or capability descriptors, thread a normalized and privacy-reviewed inputProvenance snapshot into before_tool_call, document the contract, add hook/runtime tests, and align the final shape with #48510, #48519, and #58079 before closing.
