## Summary

Keep this PR open. Current main still has the misleading `infer image describe` and `infer audio transcribe` fallback errors for the no-configured-provider path, and this PR is a focused implementation candidate for the open linked bug. The PR also appears to improve on the older duplicate implementation candidate #73576 by scoping the new test config mocks with `mockReturnValueOnce` and using `providerSupportsCapability`.
