## Summary

Keep this PR open for normal maintainer review. Current main still has the macOS LaunchAgent PATH behavior reported in #75233, and this branch is a focused implementation with source, test, docs, and changelog coverage rather than work that is already implemented on main.

## What This Changes

The PR changes macOS service PATH generation to use a canonical LaunchAgent PATH, stops merging existing plist PATH entries back into macOS install plans, and updates related tests, docs, and changelog.

## Best Possible Solution

Land this PR or an equivalent focused change after maintainer acceptance of the macOS system-only LaunchAgent PATH policy and final CI completion. Keep broader runtime Node path stabilization tracked separately by #52184/#52252 rather than mixing it into this PATH cleanup.

## Solution Assessment
