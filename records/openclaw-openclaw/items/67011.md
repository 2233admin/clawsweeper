## Summary

Keep this PR open. Current main still uses traversal-wide `WeakSet` tracking in diagnostic payload redaction, so shared acyclic sibling references are still misclassified as `[Circular]`. The PR targets the implicated sanitizer, and the discussion shows the author addressed the review concern about DAG traversal blowups with `WeakMap` memoization, but it still needs maintainer review and current-main cleanup before landing.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to rebase/clean it against current main, land active-recursion-path cycle detection with `WeakMap` memoization in `src/agents/payload-redaction.ts`, add focused regression coverage for shared sibling references and true cycles using current writer fixtures, and separately decide whether cache-trace digest fingerprinting should use the same semantics.
