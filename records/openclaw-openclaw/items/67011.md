## Summary

Keep this PR open for maintainer review. Current main still uses traversal-wide object tracking in diagnostic payload redaction, so the shared-reference false positive is still present, while the PR patch is narrow, addresses the prior memoization review comment, and has no definite code-review finding from this pass.

## What This Changes

This PR changes diagnostic payload traversal from traversal-wide `WeakSet` circular detection to active-path tracking with `WeakMap` sanitized-object reuse, and adds cache-trace regression coverage for shared object references.

## Best Possible Solution

Land or adapt the active-path plus memoized sanitizer fix with the focused cache-trace coverage, after classifying the unrelated-looking parity gate failure and preserving existing diagnostic redaction guarantees. Any digest-fingerprint alignment should be handled as a separate follow-up.
