## Summary

Keep this PR open. Current main still uses traversal-wide `WeakSet` tracking in diagnostic payload redaction, so shared acyclic sibling references are still misclassified as `[Circular]`. The PR targets the implicated sanitizer and discussion shows the author addressed the memoization concern, but the patch needs current-main cleanup before it can land.

## Best Possible Solution

Keep this PR open for maintainer review. Rebase it onto current main, implement active-recursion-path cycle detection with `WeakMap` memoization in `src/agents/payload-redaction.ts`, add focused regression coverage for shared sibling references and real cycles using current exported sanitizer names and `QueuedFileWriter` fixtures, then decide separately whether cache-trace digest `stableStringify()` should use the same semantics.
