## Summary

Keep open. Current main and the latest release still append the workspace-path hint to any heartbeat prompt that mentions HEARTBEAT.md, while this PR is an open implementation candidate for the still-open regression in #40255.

## What This Changes

This PR changes heartbeat prompt resolution so non-empty user-configured heartbeat prompts skip the workspace-path hint while default/internal heartbeat prompts keep it, and adds focused helper coverage.

## Best Possible Solution

Review and either land this PR or a narrow replacement that preserves user-configured heartbeat prompts while retaining default/internal prompt hinting and idempotency. The maintainer decision should explicitly cover whether custom prompts need an alternate non-mutating HEARTBEAT.md path guard before closing #40255.
