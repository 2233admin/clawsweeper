## Summary

Keep open. Current main and the latest release still append the heartbeat workspace-path hint to custom prompts despite docs saying custom heartbeat prompt bodies are sent verbatim; this PR targets the linked open regression, has green GitHub checks on its head SHA, and needs maintainer rebase/review because GitHub currently reports it as dirty against main.

## What This Changes

The PR exports the heartbeat workspace-hint helper for testing, adds an `isUserCustomPrompt` guard to skip hint injection for user-configured heartbeat prompts, and adds focused unit coverage for custom, default, unrelated, and already-hinted prompt cases.

## Best Possible Solution

Preserve user-authored heartbeat prompts verbatim while retaining workspace-path guidance for the built-in default and internally generated heartbeat prompts, then land this PR or an equivalent maintainer patch after mergeability refresh and validation.
