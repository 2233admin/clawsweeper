## Summary

Keep this PR open. Current main still has the earlier text-weight fallback for hybrid memory recall, but it does not implement this PR’s post-merge keyword-backed threshold behavior, and the regression coverage proposed here is not present on main.

## Best Possible Solution

Rebase and port the PR onto current main under extensions/memory-core/src/memory, preserve keyword-backed fallback hits using already-merged post-decay/MMR scores, resolve the remaining recall-threshold review concern, and add focused regression tests for BM25 textScore < 1 and temporal decay before running the targeted memory checks.
