## Summary

Current main still has the narrow server-side final-tag regexes in the sanitizer and streaming subscriber, so self-closing or attributed Gemini `<final>` variants can still leak. The issue is also paired with open PR #66225 using closing syntax, so it should remain open until that PR is reviewed/landed or closed.
