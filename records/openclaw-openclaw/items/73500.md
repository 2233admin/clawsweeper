## Summary

Keep this PR open. Current main still rejects gateway `agent` requests on `session.sendPolicy=deny` before checking whether the request actually asks for external delivery, while the runtime path already scopes that policy to `deliver === true`. The PR is a focused implementation candidate for the open #73381 bug and its diff keeps the outbound-delivery deny path intact.

## Best Possible Solution

Review and land this PR, or an equivalent maintainer patch, by moving the gateway `resolveSendPolicy` deny check under `request.deliver === true` while retaining the runtime `agent-command.ts` guard. Keep both regression cases: non-delivery allowed under deny policy, and `deliver: true` rejected under deny policy. After merge, let the linked #73381 issue close through normal GitHub closing syntax.
