The issue reports `openclaw onboard` failing to install bundled skill dependencies for `model-usage` and `summarize` on macOS because the current installer uses broken Homebrew formula paths.

## Best Possible Solution

Keep this issue open until PR #74408 or an equivalent narrow fix lands, with bundled metadata selecting a macOS cask-aware CodexBar install and the maintained npm installer for `summarize`, plus focused installer/status/security-scan coverage.
