## Summary

Keep open. Current main still parses Feishu `root_id` and `thread_id`, but direct-message reply routing remains group-gated: P2P replies do not enable `replyInThread`, do not target the thread root, and strip reply metadata before dispatch. The issue is also paired with open fixing PRs #73972 and #74001, so it should remain open until one canonical fix merges.

## Best Possible Solution

Keep this issue open and resolve it by reviewing and landing #73972 or #74001 as the canonical Feishu patch.
