Current main ships only the narrower TUI light/dark readability fix; it does not implement this PR's five named themes, `--theme` flag, runtime `/theme` command, or setter/getter theme API, and the PR still needs maintainer review because it mixes the TUI feature with unrelated protocol/package-resolution changes.
