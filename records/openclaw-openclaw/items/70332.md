## Summary

Keep open. Current main has related dreaming cleanup, but it does not implement #70332's core changes: heading-level-aware stripping of managed dream blocks, diary-style request-scoped fallback prose, body-only diary dedupe/duplicate-append suppression. The PR is still potentially useful, but it also has a concrete review finding around italic diary lines that should be fixed before merge.

## Best Possible Solution

Keep #70332 open as salvageable memory-core bug-fix work. The best path is to rebase onto current main, fix the italic timestamp/body fingerprint bug, keep heading-level stripping and duplicate-append suppression covered by focused tests, and coordinate any request-scoped fallback cleanup with the narrower #70523 inline daily-file fix.
