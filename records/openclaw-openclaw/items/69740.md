## Summary

Keep this PR open. Current main still lacks the proposed centralized production debug guard, and the PR branch only applies the guard to one ingress timing call site while mixing unrelated Control UI session-limit and install-smoke changes into a security-motivated PR.

## What This Changes

The PR adds a new `src/agents/debug-env.ts` helper for production-blocked debug flags, routes model-catalog timing through it, and also changes Control UI session-limit typing/input behavior plus install-smoke timeout expectations.

## Best Possible Solution

Keep this PR open and request changes.
