## Summary

Keep this PR open. Current main has shipped stronger runtime diagnostics for invalid thread-bound `sessions_spawn` combinations, but it still does not fully implement the PR’s stated prompt, live tool description/schema, and public docs fallback guidance for unsupported requester channels.

## Best Possible Solution
