## Summary

Keep this PR open. Keep this PR open as salvage material, but ask for a rebased and narrowed macOS PR: wire status-item left-click to the existing persistent NSWindow path, add a macOS-only resizable composer without changing iOS layout, drop or reuse the already-shipped drag/drop and Talk crash fixes as appropriate, address the callback/review issues, and split Gemini/image-generation work into the existing provider/gateway image_generate path for separate review.
