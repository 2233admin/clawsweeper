## Summary

Current main still does not implement the requested dashboard-wide keyboard navigation or a Control UI New Agent wizard. It has Cmd/Ctrl+K for the command palette and lower-level agents.create RPC support, but the dashboard UI does not expose the requested shortcut set or creation workflow.

## Best Possible Solution

Keep this issue open, or split it into two maintainer-tracked dashboard issues. Keyboard work should add a dashboard-level shortcut dispatcher with focus guards and tests for chat/navigation actions. Agent creation work should add a Control UI New Agent wizard that reuses existing agents.create, agents.files.*, config, and session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and updates Control UI docs/tests.
