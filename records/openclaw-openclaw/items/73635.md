The PR adds shared extraction for embedded/direct HTML HTTP errors, routes 403 HTML provider failures to auth-specific copy before rate-limit or transport rewrites, and adds regression coverage for formatter, sanitizer, and runtime classification paths.

## Best Possible Solution
