- The PR is stacked on open prerequisite PRs #73769, #73770, and #73771, and #73773 depends on this Gateway RPC layer, so closing this PR would remove the active implementation candidate before the stack is resolved.
- The provided diff removes existing surfaces such as plugins.uiDescriptors, sessions.pluginPatch, NodePresenceAlive*, and NodeEventResult while adding memory RPCs; that looks unrelated to the stated feature and could regress Control UI, plugin-host, session plugin patching, or native clients if merged as-is.

- Greptile's review comments identify a likely long-lived Gateway memory leak in indexJobs and incomplete sourceRefs pruning in the proposed handler.
