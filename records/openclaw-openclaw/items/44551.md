## Summary

Current main already has substantial Simplified Chinese Control UI support, matching @xiaoHEI-312's comment that most Web UI content can show Chinese when the browser/UI locale is Chinese. I would not close this issue yet because the specific requested Audit Log/tool operation labels are still not fully localized on current main, and related open PRs are still trying to clean up hardcoded Control UI strings.

## Best Possible Solution

Keep this issue open and narrow it to the remaining gap: localize Control UI audit/session-log/tool-call operation labels and any still-hardcoded tool-card chrome, or explicitly document which tool identifiers must stay raw technical IDs. The most practical path is to review and consolidate the relevant parts of #42065, #39692, and #61104 into a small PR that adds locale-backed display labels for exec/read/cron/subagents/browser and regression coverage for zh-CN rendering.
