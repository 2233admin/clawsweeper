## Summary

Current main still does not implement WhatsApp native outbound @mentions. The WhatsApp send path builds text and media payloads without outbound mention metadata, the typed send options cannot carry mention JIDs, tests still assert plain outbound text payloads, and the related implementation PR #56863 is still open/unmerged in the provided context.

## Best Possible Solution

Keep #39879 open until the bundled WhatsApp plugin ships native outbound mention support. A good implementation should add a typed outbound mention path, safely extract or accept @+digits/@digits, resolve group/LID participant mappings where available, pass Baileys-supported mention metadata for text and supported media captions, handle the separate visible-text send used for voice notes, and add focused unit coverage. PR #56863 can remain the implementation vehicle if maintainers choose to revive or merge it.
