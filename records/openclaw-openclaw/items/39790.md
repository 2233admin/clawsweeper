## Summary

Keep this PR open. Current main at a820a307dfeb950fa633c0ae6fb2386a35a91991 still does not implement either intended behavior: gateway session title extraction does not strip inbound metadata, and derived session titles cannot prefer the computed outer row displayName. The linked #39722 remains open/actionable, and open #48942 overlaps the title-test/refactor area but has not superseded or landed this fix.

## Best Possible Solution

Keep this PR or a successor open as the implementation candidate for #39722. The best fix is to strip inbound metadata before gateway session title/preview text extraction, pass the computed row displayName into deriveSessionTitle or equivalent precedence logic, and add focused regression tests for both behaviors while coordinating with #48942 if that title-test refactor proceeds.
