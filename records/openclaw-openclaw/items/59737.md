## What This Changes

## Best Possible Solution

Keep the issue open and resolve it through the existing implementation candidate if it cleanly adds `.github/PULL_REQUEST_TEMPLATE/quick_fix.md`, `.github/PULL_REQUEST_TEMPLATE/full_change.md`, a sensible fallback/default, and accurate contributor guidance for selecting the right template.
