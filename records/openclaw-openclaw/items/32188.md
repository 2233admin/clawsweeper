## Summary

Keep this issue open. Current main has adjacent session/model controls, usage reports, /status and /usage documentation, live session metadata, and threshold-based context warnings, but it still does not provide the requested persistent Control UI status area showing the active agent/model plus token, quota, and subscription context during operations.

## Best Possible Solution

Keep the enhancement open for maintainer product/UX review.
