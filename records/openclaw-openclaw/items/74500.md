## Summary

Keep this open. Current main has the intended canonical Anthropic plus `claude-cli` runtime path, auth scoping, plugin auto-enable coverage, and fallback telemetry hooks, but that does not prove the reported packaged 2026.4.26 cold-start Telegram route to OpenRouter is fixed. ## What This Changes

The issue asks OpenClaw to honor a configured `claude-cli` agent runtime on cold-start installs and emit visible diagnostics instead of silently routing turns to the OpenRouter fallback model.
