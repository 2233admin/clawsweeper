Keep open. Current main has plaintext SKILL.md loading, skill snapshot persistence, watcher refresh, ClawHub scan metadata, and install/audit scanning, but it does not implement the requested persist/trustedSources contract or a verified/signed/encrypted commercial Markdown wrapper. The prior ClawSweeper comment correctly linked open PR #74035 as an unmerged closing implementation candidate, so this issue should remain open while maintainers decide the product and security model.

## What This Changes

## Best Possible Solution
