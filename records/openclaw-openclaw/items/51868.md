## Summary

Keep open. Current main does not implement this PR’s half-block QR rendering fix: there is no half-block detector/class, fenced and indented code blocks still get only the generic `code-block-wrapper`, raw tool sidebar output is only JSON-formatted, and the current code CSS still uses generic code-block styling. The provided PR is a focused UI-only change with useful screenshots/review discussion and no security-sensitive file changes in its diff, so it should remain available for maintainer review or replacement by an equivalent tested implementation.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land this change or a maintainer-owned equivalent that adds half-block-art detection/wrapping in the markdown/tool-output paths, preserves QR quiet-zone whitespace, adds chat/sidebar CSS that tiles correctly without clipping, and includes focused tests for normal code blocks plus fenced and raw multiline QR output.
