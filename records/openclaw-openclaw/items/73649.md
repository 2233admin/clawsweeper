The PR updates `.github/ISSUE_TEMPLATE/bug_report.yml` by keeping shell rendering on a renamed `Logs` textarea and adding a separate non-rendered `screenshots_evidence` textarea for screenshots, recordings, and other evidence.

## Best Possible Solution
