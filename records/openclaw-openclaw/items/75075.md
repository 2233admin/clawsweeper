## Best Possible Solution

Land a maintainer-approved version that keeps the behavior opt-in, consumes the existing sanitized run-scoped tool-start events, preserves client-tool delegation semantics, and ships with schema, docs, changelog, and JSON/SSE ordering coverage.

## Solution Assessment
