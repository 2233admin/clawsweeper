## Summary

Keep open. Current main still has the stale session-runtime cache path and no transport-close eviction, so this PR is not obsolete. The PR is a focused candidate for a real MCP runtime bug, but its current head has an unresolved functional review concern around intentional transport closes during partial startup failure. Security review found no CI, dependency, lockfile, release, secret, or package-resolution changes.

## Best Possible Solution

Keep this PR open. The best path is to update the patch so transport-close eviction only runs for unexpected closes, preserve the SIGKILL eviction regression test, add coverage for multi-server partial startup failure, then run the targeted agents test file and the normal changed gate before landing.
