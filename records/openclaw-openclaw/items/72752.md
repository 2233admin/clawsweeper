## Summary

Keep this PR open. Current main still builds Telegram model button fallback text from the provider-local model id alone, so an OpenRouter ref parsed as provider `openrouter` plus model `openai/gpt-5.4-mini` still displays `openai/gpt-5.4-mini`. The PR is a narrow, relevant fix for #67792 with colocated regression tests; maintainer review should decide whether the long-label truncation edge called out in review belongs in this PR or a follow-up.
