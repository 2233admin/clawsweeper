## Summary

Keep this PR open. Current main still lacks Slack `allowBots: "mentions"` support: Slack config and runtime are boolean-only, while Discord already has the tri-state `off | all | mentions` behavior this PR is trying to mirror. The PR is still a focused implementation candidate, but it needs to be ported from the old `src/slack` layout to the current `extensions/slack` plugin layout.

## Best Possible Solution

Keep this PR open and ask for a port to the current Slack plugin layout. The implementation should update `extensions/slack/src/monitor/message-handler/prepare.ts`, `extensions/slack/src/monitor/channel-config.ts`, Slack config types and Zod schemas, config UI/generated metadata, Slack docs/help text, and focused Slack tests; then maintainers should run the Slack changed lane or `pnpm check:changed` before merge.
