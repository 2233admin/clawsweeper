## Summary

Keep #50943 open. Current main has a newer Feishu-owned conversation/binding model that already covers much of the sender-scoped topic affinity problem, but the PR is not fully obsolete: current main still does not implement the PR's explicit quoted/inline reply behavior in the Feishu bot path, and the PR should be revised rather than auto-closed.

## Best Possible Solution

Keep the PR open for maintainer-guided revision. The best path is to rebase onto current main, drop the generic Feishu session-key thread parsing and unrelated formatting churn, then focus only on any still-desired explicit quoted reply and outbound media `replyInThread` behavior with tests that fit the current Feishu conversation-binding model.
