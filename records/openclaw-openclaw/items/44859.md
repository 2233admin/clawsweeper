## Summary

Keep this PR open. Current main still does not implement cron multi-target delivery: cron delivery types, gateway schema, delivery planning, isolated-run dispatch, CLI options, and docs all remain single-target. The PR is still a plausible implementation candidate, but it needs rebase/review against the current delivery-plan split and explicit resolution of fan-out semantics and outbound-delivery security behavior before merge.

Keep this PR open as the active implementation candidate.
