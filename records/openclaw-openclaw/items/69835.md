## Summary

Keep this PR open. Current main at 6186ed2c074be70928ca4351ca6fc2fb87e8a142 still does not implement the PR's two central changes: `workspaceDir` is not accepted by the gateway agent protocol, dispatch still only forwards stored spawned-session workspace metadata, and missing bootstrap files are still emitted as `[MISSING]` prompt entries. ## Best Possible Solution

Keep the PR open for revision rather than closing it as implemented. The best path is to gate any explicit gateway `workspaceDir` override behind a trusted internal/admin path or validated spawned-session metadata, and to handle missing bootstrap noise with full test updates while coordinating with #40210 and the maintainer-labeled #73401 bootstrap work.
