## Summary

Keep open. Current main still has adjacent task status/detail helpers, task snapshots, CLI summary columns, /tasks rendering, session_status task lines, TaskFlow blocked summaries, and terminal notification strings, but it does not yet have the requested shared compressed operational-summary contract. There is also an open same-author implementation PR, #73746, whose body and diff explicitly include compressed operational summaries, so this issue should stay open while that PR is reviewed rather than spawning a duplicate cleanup path.

## Best Possible Solution

Keep this issue open and route it through review of #73746 or an equivalent maintainer-approved implementation. The best fix is a shared deterministic operational-summary builder in the task subsystem, reused by CLI task list/show, /tasks, /status/session_status task lines, task-domain/plugin views where appropriate, and task notification rendering, with focused tests and docs updates for any visible contract changes.
