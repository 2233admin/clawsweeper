## Summary

Keep this PR open. Current main still uses the old non-Windows Homebrew prompt guard in skills onboarding, so the FreeBSD/unsupported-platform bug remains present; this PR is still the active focused fix path for #68893.

## Best Possible Solution

Keep this PR open for maintainer review and landing, or cherry-pick the same focused allowlist plus regression tests. The product fix belongs in `setupSkills`: recommend Homebrew only on `darwin` and `linux`, keep unsupported Unix hosts quiet, and retain regression coverage for both unsupported and supported platforms.
