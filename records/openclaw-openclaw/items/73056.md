## Summary

Keep open. Current main still has the raw-session outer compaction pre-check that can return `no real conversation messages` before SDK preparation supplies turn-prefix context, and open PR #73092 is the active closing implementation candidate.

## What This Changes

Manual `/compact` should not let the outer raw-session idle pre-check skip tool-heavy over-threshold sessions as if they had no real conversation messages.

## Best Possible Solution

Keep the idle-session skip for genuinely empty or below-threshold sessions, but pass fresh observed token usage into manual compaction and bypass the outer no-real-message pre-check when the session is already at or above the compaction threshold.

## Reproduction Assessment

Yes. A high-confidence source-level reproduction is a limited message slice whose meaningful user turn is more than 20 entries behind recent tool results, plus over-threshold token usage; current main can return the outer skip before SDK preparation adds turn-prefix context.

## Solution Assessment

Yes. The threshold-bypass direction is the narrowest maintainable fix because it avoids changing the shared message classifier while preserving the original idle-cron protection for below-threshold sessions.
