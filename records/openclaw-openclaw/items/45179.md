## Summary

Keep open. Current main has useful temp-path hardening and scoped artifact metadata, but it does not implement the central request: a general provenance/classification registry for OpenClaw-created temp artifacts, cleanup/doctor awareness, or broad runtime-critical temp-path guardrails.

## What This Changes

Add default provenance and classification for OpenClaw-created temp artifacts, plus durable-path and runtime-critical `/tmp` or `/private/tmp` guardrails.

## Best Possible Solution

Define a small core temp-artifact contract for OpenClaw-created files: classify artifacts at creation, persist minimal provenance in state, expose safe helper APIs to plugins/tools, teach doctor/security/cleanup to distinguish labeled and unlabeled artifacts, and add targeted service/runtime checks for temp-backed critical paths.
