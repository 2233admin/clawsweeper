## Summary

This should stay open. Current main still does not export `openclaw/plugin-sdk/discord`, npm still reports `openclaw` latest as `2026.4.26` and `@openclaw/discord` latest as `2026.3.13`, and the published Discord plugin entry imports the missing subpath. The checked-in Discord plugin has already migrated to `channel-entry-contract`, but that source migration has not reached the package users get from `openclaw plugins install @openclaw/discord`.

## Best Possible Solution

Keep this issue open until maintainers either publish a compatible `@openclaw/discord` package and verify `openclaw plugins install @openclaw/discord` resolves to it, or add a temporary `openclaw/plugin-sdk/discord` compatibility export with a documented deprecation path and regression coverage for loading `@openclaw/discord@2026.3.13` against the current core.
