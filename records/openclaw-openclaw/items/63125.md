## Summary

Keep open. Current main has CLI timeout mitigations, gateway event-loop diagnostics, and several performance fixes, but it does not prove the 2026.4.8 gateway CPU/RPC-stall umbrella is fixed or shipped; the Discord voice symptom is still covered by the same-author open report #63098 and the current gateway CPU/status-timeout family remains open in #72338.

## Best Possible Solution
