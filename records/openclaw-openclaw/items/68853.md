## Summary

Keep this PR open. Current main still lacks the PR's central `triggerOpenClawRestart()` SIGUSR1 fast path and still routes Windows schtasks-supervised respawn through `triggerOpenClawRestart()`. The merged #69056 fixes a related Windows SIGUSR1 fallback, and open #73889 tracks broader Windows scheduled-task health, but neither supersedes this focused two-part fix; the discussion also shows a maintainer repaired this branch to preserve the contributor path.

## What This Changes

This PR adds a `triggerOpenClawRestart()` SIGUSR1 fast path, changes Windows schtasks-supervised respawn/update handoffs to call `relaunchGatewayScheduledTask()` directly, widens the restart attempt method type, adds restart regression coverage, and records a changelog fix.
