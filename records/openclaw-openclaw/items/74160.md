## Summary

Keep this active PR open. Current main does not implement the PR's intended persisted tail-drop repair: session-file repair still rewrites trailing empty errored assistant entries to the sentinel, existing tests still assert that trailing rewrite/no-op behavior, and docs only describe outbound Anthropic prefill stripping plus Bedrock rewrite-on-disk.
