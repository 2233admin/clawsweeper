## What This Changes

The PR adds a `--global` flag to `openclaw skills install` and `openclaw skills update`, routes that scope to `CONFIG_DIR`, rejects combined `--global`/`--agent` targeting, and updates skills docs and CLI command tests.

Review and land this PR if validation stays green, preserving workspace installs as the default while adding an explicit shared managed `--global` path for ClawHub skill install/update operations.
