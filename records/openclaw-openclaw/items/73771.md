Current main still lacks the PR’s central agent-namespace isolation in the builtin memory SQLite index, query filters, write/delete paths, result provenance, and QMD mcporter daemon state. It is also an active stacked implementation candidate with related open PRs and unresolved review feedback, so it is not conservative cleanup to close.
