## Summary

Keep open. Current main still has the reported wizard default path, and this PR is a relevant implementation candidate, but the patch is over-broad for custom shared workspace defaults and overlaps other open fixes that need maintainer selection.

## What This Changes

This PR adds and exports a peer-workspace suggestion helper, uses it for new interactive `agents add` workspace defaults, and adds agent-scope coverage for default and configured workspace cases.

## Best Possible Solution

Land one narrow, regression-tested fix that suggests the documented peer-level workspace for brand-new agents in the setup/default workspace case, preserves existing-agent updates and intentional shared-base configurations, then close the duplicate implementation PRs after the selected fix lands.

## Reproduction Assessment

Yes. Source inspection on current main is enough: setup persists `agents.defaults.workspace`, the wizard passes `resolveAgentWorkspaceDir(cfg, agentId)` into the `Workspace directory` prompt, and the resolver joins non-default ids under that fallback.

## Solution Assessment

No. The wizard-only direction is sound, but this implementation is not the best version yet because it converts custom shared workspace defaults into sibling paths; a safer fix should preserve #59789 shared-base behavior outside the documented default setup case.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.84

Full review comments:

- **[P2] Preserve custom shared workspace defaults:** `src/agents/agent-scope-config.ts:204`
  - body: When `agents.defaults.workspace` is a custom shared base, current main intentionally resolves non-default agents under that base (`/shared-ws/main`, covered by the #59789 regression test). The new helper always returns a sibling of the default agent workspace, and `agents add` then persists that value, so accepting the prompt can silently bypass the configured shared base. Gate the peer-level suggestion to the setup/default workspace case, or defer to `resolveAgentWorkspaceDir` for custom shared roots.
  - confidence: 0.84
