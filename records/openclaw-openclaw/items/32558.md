## Summary

Current main still uses direct Microsoft Graph `/content` uploads for Teams OneDrive and SharePoint file sending, with no upload-session threshold, chunk loop, or `nextExpectedRanges` handling. The PR branch adds Microsoft Graph upload-session fallback logic and tests for large Microsoft Teams OneDrive and SharePoint file uploads.
