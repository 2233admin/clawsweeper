## What This Changes

This PR adds optional SearXNG `img_src` parsing, includes it in returned web_search result items, and adds a regression test for image-result preservation.

## Best Possible Solution

Land this small SearXNG plugin change if maintainer review agrees with exposing the upstream field name; any broader cross-provider image-result schema can be handled later without blocking this passthrough.
