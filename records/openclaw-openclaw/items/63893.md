Land a rebased version of this migration that rebuilds legacy `flow_runs` tables after the additive schema steps, preserves existing rows, removes `owner_session_key`, recreates the current indexes, and keeps focused regression coverage for the legacy hybrid schema plus successful managed-flow insertion.
