## Summary

Keep open. The latest patch is still a renderer-side slice rather than an end-to-end live reasoning stream fix: current main still emits and stores live chat deltas as text-only strings before the PR's structured stream branch can run, and the PR also changes the documented reasoning visibility fallback by deriving visibility from the thinking level. The overlapping open work in #59086 should be consolidated or compared by a maintainer rather than closed as obsolete.

## What This Changes

The PR widens Control UI stream item/rendering code to accept structured assistant stream payloads, passes reasoning visibility into streaming groups, renders reasoning-only assistant bubbles, and adds grouped-render/buildChatItems regression coverage.

## Best Possible Solution

Land one consolidated implementation that carries structured assistant payloads from gateway chat events through controller/app state, stream segments, item building, and rendering, while preserving /reasoning visibility semantics and falling back to the reading indicator when hidden reasoning-only streams have no visible text.

## Reproduction Assessment

Yes. A high-confidence static reproduction exists by tracing a live assistant event through emitChatDelta, handleChatEvent, ChatState.chatStream, ChatProps.stream, buildChatItems, and renderStreamingGroup; the current path collapses the message to text before the PR's object-stream handling can run.

## Solution Assessment

No. The PR is not the best complete fix because it widens renderer inputs without plumbing structured live payloads through the state pipeline, and it uses thinking effort as a reasoning visibility fallback instead of preserving the documented /reasoning fallback of off.

  - confidence: 0.88

  - body: /think controls model effort, while /reasoning controls whether thinking blocks are displayed and falls back to off. Falling back from an absent reasoningLevel to props.thinkingLevel can show live reasoning for sessions that should inherit hidden reasoning visibility.
  - confidence: 0.86
