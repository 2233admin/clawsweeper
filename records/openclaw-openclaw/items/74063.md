## Summary

Keep open. Current main has a macOS companion app, iOS/Android node apps, and a browser Control UI/PWA, but the requested official all-platform desktop app is not implemented. The platform docs explicitly say Windows and Linux companion apps are planned, so this is a real product/roadmap request rather than stale, duplicate, or already shipped work.

## Best Possible Solution

Keep this open for maintainer product planning. The best path is to decide whether OpenClaw wants native Windows/Linux companion apps, a cross-platform desktop shell, or improved PWA packaging, then split the decision into narrower platform-specific design and implementation issues that fit the existing Gateway, Control UI, and macOS app boundaries.
