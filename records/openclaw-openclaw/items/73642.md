## Summary

Keep this PR open. Current main still lacks the requested `openclaw infer model run --thinking <level>` plumbing, and the PR is a focused implementation, but the patch leaves the command metadata stale so CLI capability introspection would not advertise the new flag.

## What This Changes

The PR adds a `--thinking <level>` option to `openclaw infer model run`, normalizes and forwards it to local simple completions and Gateway raw model runs, and updates infer docs plus CLI/runtime tests.

## Best Possible Solution

Land this PR or an equivalent patch after adding `--thinking` to the `model.run` capability metadata used by `infer list/inspect`, keeping the local and Gateway forwarding behavior, and preserving focused docs plus local/gateway/invalid-input coverage.
