## Summary

Keep this PR open. Current main still does not implement `references/` auto-load support, and the PR is useful source work, but the latest patch only makes merged reference content available through sandbox skill sync. This PR adds skill metadata parsing, workspace loading, tests, docs, and a changelog entry for auto-loading markdown files from a skill-local `references/` directory.

Keep this PR open and request a focused revision. Public docs and changelog should land only once normal reads, sandbox sync reads, and `workspaceAccess: rw` reads all see the same supported behavior.
