## What This Changes

The PR adds missing-path guidance to the read tool, pre-dispatch unsupported-action validation to the message tool, and targeted regression tests for both paths.

## Best Possible Solution

Land a focused version of this PR after the supported-action display is deduplicated and the PR head has green relevant CI, keeping the guardrails in the existing read/message tool wrappers and preserving pass-through behavior when discovery is unavailable.

## Reproduction Assessment

## Solution Assessment
