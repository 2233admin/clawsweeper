Keep open. Current main still leaves the default generated AGENTS.md operational safety/tool sections outside a 1500-character bootstrap budget, and current head/tail truncation does not preserve them either; the adjacent tiered-bootstrap work changes file selection, not priority inside AGENTS.md.

## What This Changes

The issue asks OpenClaw to preserve critical tool-use and safety guidance when the auto-generated AGENTS.md is truncated by agents.defaults.bootstrapMaxChars.

## Best Possible Solution

Move the essential safety and tool-use rules to the top of the generated AGENTS.md template, add a regression proving those sections survive a 1500-character bootstrapMaxChars budget, and document how existing workspaces can update their local AGENTS.md without overwriting personal content.
