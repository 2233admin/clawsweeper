## Summary

Keep #59086 open. Current main still does not implement the PR's central behavior: reasoning streaming is callback-gated, chat delta/final payloads remain text-only, the UI stream state/rendering is text-only with streamed reasoning disabled, and the local-before-global event ordering change is absent. The related #68982 work overlaps but is not clearly a canonical superseding item for this broader PR.

## Best Possible Solution

Keep the PR open for maintainer judgment or contributor follow-up. The best path is still an end-to-end structured reasoning stream implementation: emit thinking independently of UI callbacks, map actual agent run ids to client runs before global fan-out, forward thinking through chat deltas, carry structured stream messages through UI state/buildChatItems, render streamed reasoning according to the existing thinking visibility setting, and cover hidden-reasoning, thinking-only, thinking+text, tool-interleaving, silent/heartbeat, and final-resync cases. Maintainers should decide whether #59086 or the narrower #68982 is the landing vehicle rather than auto-closing either as cleanup.
