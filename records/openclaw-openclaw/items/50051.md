## Summary

Keep this PR open. Current main still does not implement the central ExecuTorch/Parakeet Talk Mode STT backend, the bundled executorch plugin/runtime, the macOS Talk STT backend selector, transcript-only Talk prompt display, or the PermissionManager notification .app guard. The PR contains useful work and author follow-up on review blockers, but it is broad, security-sensitive, and better handled by maintainer review or splitting than by conservative cleanup closure.

## Best Possible Solution

Keep this PR open for maintainer triage, but split or rework it. The best path is to land small focused pieces if maintainers want them: the PermissionManager .app notification guard, transcript-only Talk prompt display, any generic local/keyless media-provider seam still needed after current plugin APIs, and a deliberate macOS Talk Mode STT backend selector. The ExecuTorch provider/runtime itself should receive a focused bundled-plugin security review or move to ClawHub if maintainers do not want it in core.
