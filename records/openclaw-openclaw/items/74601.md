## Summary

Keep open: current main remains workspace-scoped for Skill Workshop writes, review context, and proposal storage, and no open PR appears to implement or link this request. The requested expansion is plausible, but it changes shared-skill write authority and cross-workspace review boundaries, so it needs maintainer product/security design rather than a cleanup close or automated repair lane.

## What This Changes

Add opt-in Skill Workshop support for proposing shared-skill updates, promoting workspace skills to shared skills, and detecting cross-workspace skill patterns.

## Best Possible Solution

Stage this inside extensions/skill-workshop: add explicit target metadata, shared-root validation, target-aware proposal storage, manual-approval-only shared applies, docs, and focused tests first; treat cross-workspace semantic promotion as a later design after privacy and evidence limits are agreed.
