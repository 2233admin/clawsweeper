## Summary

Keep open. Current main still scopes Skill Workshop proposals, review context, state, and writes to workspace skills only, while this request asks to broaden a bundled experimental plugin into shared-skill writes and cross-workspace review; that is not implemented and needs maintainer product/security direction before an automated fix lane.

## What This Changes

Add opt-in Skill Workshop support for proposing shared-skill updates, promoting workspace skills to shared skills, and detecting similar proposals across workspaces.

## Best Possible Solution

Stage this inside the Skill Workshop plugin: first design an explicit shared-target model with separate proposal metadata/storage, root validation, docs, and manual-approval-only shared applies; then treat cross-workspace semantic promotion as a follow-up after maintainers define evidence, privacy, and safety limits.
