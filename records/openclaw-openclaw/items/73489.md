## Summary

Keep this PR open. The latest head also addresses the Greptile review direction by using `fetchWithSsrFGuard`, `Readable.fromWeb()` plus `pipeline`, an explicit `content-length` null check, and a streamed size-cap transform, but it still needs maintainer review and focused transport tests before landing.
