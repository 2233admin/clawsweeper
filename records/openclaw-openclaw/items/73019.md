## What This Changes

Add xAI Realtime Voice Agent support to the `/voiceclaw/realtime` gateway surface, ideally through a shared OpenAI-Realtime-compatible adapter usable by xAI and future OpenAI Realtime support.

## Best Possible Solution

Use the paired draft PR as the canonical implementation path.
