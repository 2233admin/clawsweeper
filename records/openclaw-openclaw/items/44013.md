## Summary

Keep this PR open. Current main still does not ship Linux, Windows, or unified Tauri desktop companion apps, and the project docs/vision continue to treat Windows and Linux companion apps as planned first-party platform work. The submitted implementation has real build, runtime, workspace, packaging, and security blockers, but no replacement has landed on main, so this should remain a maintainer/product/security review item rather than be auto-closed.

## Best Possible Solution

Keep this PR open for maintainer/product/security review or author follow-up. The best path is to coordinate it with #75 and the newer desktop-app work, choose one shared Linux/Windows desktop companion architecture, fix the Tauri build/runtime/security issues, add workspace, lockfile, docs, labeler, CI, release, and dependency-maintenance wiring, then validate Linux and Windows builds before merge consideration.
