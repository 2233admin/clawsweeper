## Summary

Keep #72179 open. Current main still lacks the proposed public `cron_lifecycle` plugin hook, hook-runner method, gateway cron hook emission, and docs. The PR is the active focused implementation for the cron audit/telemetry gap from #8209; older PR #8855 is closed unmerged and does not supersede this one.

## Best Possible Solution

Keep #72179 open as the canonical implementation path for #8209. Maintainers should review the typed `cron_lifecycle` event shape as a public plugin API, ensure gateway emission covers both `started` and `finished` for `agentTurn` and `systemEvent`, keep focused tests/docs, and merge after normal validation.
