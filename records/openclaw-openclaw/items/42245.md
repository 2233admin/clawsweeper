## Summary

Keep this PR open. Current main still lacks a missing-`--agent` warning for `openclaw cron add --message`, the linked issue #42196 remains open, and the PR branch is a narrow CLI/test/changelog implementation with successful check-runs on the current head.

## What This Changes

This PR adds a stderr warning when `cron add` creates an agent-turn job without a nonblank `--agent`, updates cron CLI tests for warning and JSON stdout behavior, and adds a changelog fix entry crediting the contributor.

## Best Possible Solution

Keep this PR open as the canonical implementation path for #42196.
