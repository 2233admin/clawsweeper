## Summary

Keep open. Current main still lacks the Twitch token writeback and OpenProse control-flow fixes, but this PR should not merge as-is because the proposed writeback can persist a literal refreshed OAuth token over an authored `${...}` config reference.

## What This Changes

The PR adds Twitch refreshed OAuth token writeback for config-backed accounts with tests, and changes an OpenProse fast-loop example so terminal output branches are exclusive.

## Best Possible Solution

Revise the Twitch writeback to persist only literal config-backed token fields, preserve or skip authored env-reference token fields, and add regression coverage for base-level, `accounts.default`, and named-account `${...}` accessToken configs. The OpenProse exclusivity change can stay if it still applies cleanly after rebase.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.9

Full review comments:

- **[P1] Preserve env-referenced Twitch tokens before writeback:** `extensions/twitch/src/token-writeback.ts:105-108`
  - body: This helper writes a resolved config snapshot after changing `accessToken`. If the user authored `channels.twitch.accessToken: "${OPENCLAW_TWITCH_ACCESS_TOKEN}"`, the refreshed token differs from the read-time env value, so `writeConfigFile` will persist the literal OAuth token instead of restoring the env reference. Detect authored env-reference token fields and skip or preserve them.
  - confidence: 0.9

## Security Review

Status: needs_attention

Summary: The PR introduces a concrete secret-handling regression for Twitch configs that store `accessToken` as an env reference.

Concerns:

- **[medium] Env token refs can become literal OAuth tokens:** `extensions/twitch/src/token-writeback.ts:105`
  - body: `readConfigFileSnapshotForWrite()` exposes resolved config, and the PR mutates that value before `writeConfigFile()`. Because env refs are restored only for unchanged or still-matching values, a refreshed token can be written as a literal secret over an authored `${...}` reference.
  - confidence: 0.9
