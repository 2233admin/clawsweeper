## Summary

Keep open. Current main still lacks the core Gemini CLI OAuth env-proxy change and still closes dispatchers gracefully before destroying them, so this PR is not obsolete. The PR changes Gemini OAuth guarded fetch mode and dispatcher/proxy-agent teardown behavior, adds proxy-path tests, and also modifies Discord proxy fallback, LINE setup, Nostr bus/constants, and plugin SDK tests.

## Best Possible Solution
