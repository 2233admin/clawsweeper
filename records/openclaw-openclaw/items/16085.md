## Summary

Keep this PR open. Current main still implements the bundled Signal plugin against native signal-cli JSON-RPC plus SSE only: there is no `apiMode`, no adapter/container client, no `/v2/send`, and no `/v1/receive/{account}` WebSocket path. This PR remains the canonical implementation candidate for the container REST/WebSocket support requested by #10240 and later reported again in #73553; #73086 is adjacent native SSE timeout work, not a replacement.
