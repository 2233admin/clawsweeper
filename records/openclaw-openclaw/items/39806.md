## Summary

Keep PR #39806 open. Current main still has the Debug Snapshots UX gap from #39777: disconnected refresh returns before visible state is set, and the Snapshots card has no body-level loading or disconnected callout. The PR is not merge-ready as-is, but it remains a focused attempt at a real unimplemented UI fix rather than something obsolete or closeable.
