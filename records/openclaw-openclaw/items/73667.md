## Summary

Keep this draft PR open for maintainer follow-up. Current main has newer partial Active Memory/QMD reliability work, but it does not fully implement this PR's lower default timeout, zero setup grace, or lower-latency docs guidance; the PR branch is also not merge-ready because its fixed hook timeout would truncate valid custom recall budgets and the branch is currently non-mergeable against main.

## What This Changes

The PR lowers Active Memory's default recall timeout and setup grace, shortens the registered before_prompt_build hook timeout, adds jittered parallel QMD startup initialization, and updates Active Memory documentation and tests for lower-latency defaults.

## Best Possible Solution

Revise or replace the branch around current main: preserve the hook envelope invariant for resolved custom timeouts, make an explicit maintainer decision on lower default or nonblocking Active Memory behavior, reconcile QMD startup with the current lazy/startup mode contract, and update docs plus deterministic tests for high timeoutMs and multi-agent QMD startup behavior.

## Reproduction Assessment

Yes for the main review finding: the PR branch itself shows a 4000ms hook envelope while still accepting config.timeoutMs values up to 120000ms. A config with timeoutMs above 3000ms exercises the mismatch without needing a live backend.

## Solution Assessment

No. The direction addresses a real reliability report, but this patch is not the safest solution because it breaks valid custom recall budgets and is behind current QMD startup changes; the safer path is a rebased fix that sizes the hook timeout from the resolved config or cap and uses the current startup policy seams.

## Review Findings

Overall correctness: patch is incorrect

- **[P1] Size the hook timeout from the configured recall budget:** `extensions/active-memory/index.ts:2434`
  - body: The branch still accepts `config.timeoutMs` up to 120000ms and the recall watchdog waits `config.timeoutMs + setupGraceTimeoutMs`, but this registration caps the hook envelope at the default budget plus 1s.
