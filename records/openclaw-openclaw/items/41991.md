## Summary

Keep this PR open. Current main still collapses Gemini CLI OAuth client-config extraction failures into one generic setup error, while this PR is a narrow diagnostics-and-tests change against the active bundled Google plugin. The open security/product issue #54289 could supersede it only if maintainers decide to remove or gate automatic extraction.

## Best Possible Solution

Keep this PR open for maintainer review. If automatic Gemini CLI credential extraction remains supported, land a narrow diagnostics fix in extensions/google/oauth.credentials.ts with regression coverage around resolveOAuthClientConfig() for missing binary, missing credential source, parse failures, and read/extraction exceptions. If #54289 removes or gates extraction, close this PR as superseded by that security/product decision.
