## Summary

Keep this PR open. Current main still collapses Gemini CLI OAuth credential extraction failures into the same generic setup error, while this PR is a narrow diagnostics/test update against the active bundled Google plugin. The related security/product issue #54289 could supersede it only if maintainers decide to remove or gate automatic extraction.

## Best Possible Solution

Keep this PR open for maintainer review. If automatic Gemini CLI client-config extraction remains supported, land a narrow diagnostics fix in extensions/google/oauth.credentials.ts with regression coverage around resolveOAuthClientConfig() for missing binary, missing credential source, parse failures, and read/extraction exceptions. If #54289 removes or gates extraction, close this diagnostics-only PR as superseded by that security/product decision.
