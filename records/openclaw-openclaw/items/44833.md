## Summary

Keep open. Current main still appears to have the reported Telegram group activation bug: captionless photos produce media placeholders and downloaded media, but group mention activation is computed from empty Telegram text/caption, and the shared mention helper does not let catch-all patterns evaluate against empty text before the group turn is skipped. The directly linked fix PR #44883 is closed unmerged, so this is still actionable.

## What This Changes

The issue requests that captionless Telegram group photos trigger image understanding consistently with Telegram DMs when normal group mention policy is satisfied.

## Best Possible Solution

Update shared mention matching or Telegram inbound activation so intentionally configured catch-all mention patterns can authorize captionless media-only group turns, while preserving specific pattern behavior, sender and group authorization, native mentions, reply-to-bot implicit mentions, control-command bypass, audio preflight, and media attachment delivery.
