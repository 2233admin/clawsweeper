## Summary

Keep open: current main still has the reported iOS Contacts/Calendar first-use permission gap, but the PR branch still has merge-blocking cancellation behavior in the invoke permission path and includes an unrelated installer execution-path change that needs separate review.

## What This Changes

This PR adds iOS Contacts/Calendar privacy usage strings, changes Contacts and Calendar first-use authorization paths to request OS permission prompts, adds Calendar permission-box tests, and modifies Windows installer failure handling.

## Best Possible Solution

Land a scoped iOS-owner fix that adds the missing usage-description keys, makes Contacts and Calendar first-use permission requests cancellation-safe, preserves least-privilege Calendar write-only behavior, and validates the fresh-install node invoke flows; split the installer change into its own reviewed PR and keep #51011 as the companion settings UX.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Make Contacts permission waits cancellation-aware:** `apps/ios/Sources/Contacts/ContactsService.swift:105-109`
  - body: This branch awaits `CNContactStore.requestAccess` with a bare continuation. `invokeWithTimeout` cancels the invoke task after a timeout, but this continuation has no cancellation handler, so locked or headless first-use invokes can leave the detached task suspended until the user responds. Use a cancellation-aware helper or keep a fail-fast noninteractive path.
  - confidence: 0.9
- **[P2] Make Calendar prompt startup atomic with cancellation:** `apps/ios/Sources/Calendar/CalendarService.swift:182-183`
  - body: `box.install`, the `isResolved` guard, and `start` are separate operations. If cancellation fires after the guard but before `start`, the continuation has already resolved false and this code can still launch an EventKit prompt after the invoke timed out. Gate install-and-start atomically so cancelled invokes cannot start late OS prompts.
  - confidence: 0.84
