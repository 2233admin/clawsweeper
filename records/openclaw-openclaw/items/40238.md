## Summary

Keep open. Current main still has the inline skill tool-dispatch policy gap, and this PR moves in the right direction, but the proposed patch is not merge-ready because it does not forward the full target-session, sender, group, and plugin-binding context that the normal command path uses, and it would reintroduce an eager coding-tool import in a module current main keeps lazy.

## What This Changes

The PR changes inline skill `command-dispatch: tool` handling to select tools through `createOpenClawCodingTools` and adds regression coverage for non-owner messaging-profile dispatch.

## Best Possible Solution

Inline skill `command-dispatch: tool` should resolve callable tools through the same policy-resolving factory used by normal command sessions, behind a lazy runtime wrapper, while forwarding target-session, route, sender, model, group, and plugin-binding context with focused regression tests for profile-disallowed exec-style tools and target-session group policy.

## Review Findings

Overall correctness: patch is incorrect

Overall confidence: 0.86

Full review comments:

- **[P2] Forward target-session and sender policy context:** `src/auto-reply/reply/get-reply-inline-actions.ts:198-224`
  - body: The new coding-tools call still does not mirror the normal command path: it reads group fields from `sessionEntry` and omits `allowGatewaySubagentBinding` plus `senderName`/`senderUsername`/`senderE164`. Those fields feed plugin subagent binding and group tool policy, so inline tool-dispatch skills can resolve a different tool set than the session that exposed the command.
  - confidence: 0.88
- **[P3] Keep coding tools behind a runtime import:** `src/auto-reply/reply/get-reply-inline-actions.ts:2`
  - body: Current main keeps inline action tool loading behind a runtime import so the inbound inline-actions module does not eagerly load the full tool stack. Replacing that with a static `pi-tools` import reintroduces the eager dependency; add a small runtime wrapper or lazy loader when rebasing.
  - confidence: 0.78
