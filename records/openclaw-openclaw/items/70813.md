Full review comments:

- **[P2] Use the resolved account for edit formatting:** `extensions/whatsapp/src/send.ts:228`
  - body: `editMessageWhatsApp` resolves the active listener using the selected/default account, but `resolveMarkdownTableMode` still receives `options.accountId`. Capture the resolved account id from `requireOutboundActiveWebListener` and pass it through here.
  - confidence: 0.88
