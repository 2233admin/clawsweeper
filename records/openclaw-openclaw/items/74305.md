## What This Changes

The issue reports that ACPX Codex worker sessions fail with AcpRuntimeError when the configured Codex ACP command includes model and model_reasoning_effort overrides.

## Best Possible Solution

Land a focused ACPX Codex command-normalization fix that preserves configured adapter args but skips appending runtime model/thinking keys that are already present, with tests for full overlap, partial overlap, wrapper commands, and the configured-command scenario from the report.

## Reproduction Assessment

## Solution Assessment
