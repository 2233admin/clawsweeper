## Risks / Open Questions

- The proposed hook would expose the full post-policy tool list and provider-facing parameter schemas to loaded plugins, including tools never called; maintainers should confirm this fits the trusted-plugin boundary.
- The PR body/diff describe passing `parameters` by reference. If handlers can mutate schema objects before provider/session use, the implementation should clone, freeze, or fire after those objects are no longer mutable in the run path.
- Fire-and-forget per-attempt emissions can repeat on retries; consumers may need a documented fingerprint or dedupe strategy.
