## Summary

Keep open. Current main still lacks the PR’s intended Control UI streaming dedupe behavior, so the PR is not obsolete or implemented on main. It is also not merge-ready as provided: the PR includes an unrelated binary workspace artifact, has a review finding about whitespace-only deltas being dropped, and overlaps with still-open PR #47377 without a merged canonical fix.

## Best Possible Solution

Keep this PR open until maintainers either fix and land it or choose a corrected canonical implementation such as #47377. The best path is a narrow current-main patch that tracks committed stream length, stores only newly streamed suffixes at tool boundaries, renders only the live suffix after committed segments, preserves whitespace-only separators, removes the unrelated binary artifact, and adds regression coverage for cumulative text before and after one or more tool calls.
