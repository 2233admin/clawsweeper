## Summary

Keep open. Current main still lacks the stream-segment dedupe behavior this PR is trying to add, but the proposed patch is not merge-ready because it loses whitespace-only streamed suffixes, includes an unrelated binary artifact, and needs to be rebased onto the current chat item builder path.

## What This Changes

The PR adds Control UI chat stream offset state, slices committed and live stream text by that offset, updates related UI test fixtures, adds a chat-view regression test, and adds an unrelated workspace PNG.

## Best Possible Solution

Land one corrected current-main implementation that stores and renders only uncommitted streamed suffixes across tool interruptions, preserves whitespace separators, removes the local artifact, and covers both `app-tool-stream` segment commits and `buildChatItems` live rendering with focused regression tests.
