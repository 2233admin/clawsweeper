## Summary

Keep this issue open. Current main partially satisfies the request with agent scoping, JSON output, and recent-session filtering, but it still does not implement the requested duration-style older/newer filters, size-threshold filter, or list-row byte-size metadata needed for `--larger-than`.

## Best Possible Solution

Keep this issue open and implement the remaining core CLI/protocol/tool work deliberately: add duration parser-backed `--older-than` and `--newer-than`, define and expose the size metric for `--larger-than`, carry the metadata through CLI JSON, gateway `sessions.list`, and `sessions_list`, then update `docs/cli/sessions.md` and focused CLI, protocol, gateway, and tool tests.
