## Summary

Keep open. Current main has the system-event primitive and partial Microsoft Teams/task-delivery fallbacks, but the requested cross-channel, model-visible delivery-failure notification is still missing from the normal shared reply path and from Feishu, Telegram, and Discord final reply delivery.

## Best Possible Solution

Implement a shared delivery-failure notification contract for normal channel reply delivery. Reuse the existing system-event queue, add a compact sanitized helper for channel/target/thread/error/retry hint/message preview, and wire the shared dispatcher or channel onError paths for Feishu, Telegram, Discord, and Microsoft Teams with focused regression coverage. Keep plugin hooks as observer instrumentation rather than treating them as the agent notification.

## Clownfish Work Prompt

Implement cross-channel message delivery failure notification for normal reply delivery. The observable bug is that Feishu, Telegram, Discord, and shared reply delivery can fail after the agent has generated a reply, but the agent only sees logs/hooks/fallback behavior and does not get a next-turn delivery-failure system event. Reuse the existing system-event primitive and the Microsoft Teams partial implementation as the reference. Add a shared helper that builds a compact sanitized event with channel, target/thread/session context when available, a bounded error summary, retry hint for transient/throttled/network errors when feasible, and a bounded message preview. Wire the shared reply dispatcher and/or the channel-specific onError paths so normal Feishu, Telegram, Discord, and Teams reply failures enqueue the event to the right session. Preserve existing message_sent hook semantics; do not treat hooks as the agent notification. Do not add raw secrets, full message bodies, tokens, or unrelated retry-policy changes. Add focused tests for the helper and for Feishu/Telegram/Discord/Teams reply failure paths. Add a user-facing changelog entry with appropriate reporter credit, likely Thanks @sunxzycn, if behavior changes.
