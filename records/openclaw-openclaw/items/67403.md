## Summary

Keep this PR open. Current main still uses full normalized-query string matching in the memory-wiki digest candidate gate and full-page scorer, and it has no token-overlap helpers or regression tests for the multi-term wiki search behavior this PR targets. The PR remains a plausible implementation candidate, but its current diff also includes unrelated plugin-loader/trust and test-environment changes that need maintainer review or splitting before merge.

## Best Possible Solution

Keep this PR open for maintainer review.
