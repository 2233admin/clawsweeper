## Summary

Keep open. Current main still lacks Copilot `/models` discovery, so the PR is relevant, but the latest branch is not merge-ready because the new discovery cache drops agent-scoped auth context, reuses one plan catalog across credential contexts, and regresses the existing `xhigh` fallback when live discovery is unavailable.

## What This Changes

The PR adds GitHub Copilot runtime `/models` discovery, API-to-model mapping, dynamic catalog augmentation, thinking-profile derivation, and focused unit tests, with small adjacent plugin SDK/test cleanups.

## Best Possible Solution

Land a refined GitHub Copilot plugin implementation that fetches authenticated Copilot model capabilities through provider-owned hooks, scopes cached results to the credential/baseUrl/agent context, preserves existing fallback thinking behavior when discovery fails, and covers auth-profile-only, multi-agent/profile switching, cache invalidation, and known-model fallback cases with tests.

## Reproduction Assessment

Yes. Source inspection gives a high-confidence reproduction path: an agent-local Copilot auth profile can be found by catalog preflight, but the PR's shared fetch helper then looks outside that agent store; separately, disabling or failing discovery makes the new thinking fallback omit current-main `xhigh` support for known Copilot GPT/Codex models.

## Solution Assessment

No, not as written. Dynamic `/models` discovery is the right product direction, but the implementation should carry `agentDir` through discovery, key or invalidate the cache by credential context, and preserve current fallback thinking levels when API discovery is unavailable.

  - confidence: 0.84
- **[P2] Keep xhigh fallback for known Copilot models:** `extensions/github-copilot/index.ts:463-464`
  - body: When `/models` discovery fails or has not populated the cache, this fallback now returns only the generic profile and drops the current hardcoded `xhigh` support for known Copilot GPT/Codex models such as `gpt-5.4` and `gpt-5.2`. That makes offline or transient-discovery paths less capable than current main.
  - confidence: 0.86
