## Summary

Keep this PR open. Current main still does not implement the requested global `agents.defaults.reasoningDefault` contract: the config schema/types/docs omit the global key, reply and status resolution only honor per-agent `agents.list[].reasoningDefault`, and direct status can still fall through to model default reasoning. The PR remains the active implementation candidate for #39094, but the latest discussion shows maintainer follow-up, unresolved final security audit context, and an unrelated cron runtime change that should receive maintainer review before merge.

## Best Possible Solution

Keep this PR open for maintainer review.
