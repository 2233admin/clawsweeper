## Summary

Keep open. Current main still does not expose the five requested active-run steer/abort helpers from `openclaw/plugin-sdk/agent-runtime`, even though the internal runner defines them and the package export path exists. The related closed issue #49624 does not make this PR obsolete because the observable SDK gap remains. A security pass found no dependency, workflow, script, or package-resolution changes in the PR, but the existing `abortEmbeddedPiRun` all-runs overload is a real maintainer decision point before landing.

## Best Possible Solution

Review this PR as a still-relevant SDK proposal. The best path is to expose a supported channel-plugin active-run control surface for all five helpers, preferably with a maintainer decision on whether `abortEmbeddedPiRun` should be exported whole or wrapped to the single-session contract, plus SDK docs/baseline coverage. If maintainers prefer a narrower subpath over the broad `agent-runtime` barrel, ask the contributor to retarget the PR rather than closing it as obsolete.
