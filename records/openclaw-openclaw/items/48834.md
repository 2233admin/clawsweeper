## Summary

Keep open. Keep this PR open for a focused rebase or replacement against current main: use the existing generic conversation-binding and `conversation-runtime` surfaces, fix non-streaming Feishu ACP block delivery with regression tests, add Feishu `threadBindings` schema/config metadata and lifecycle/update hooks through `conversationBindings`, update `/focus` and `/unfocus` command/docs copy, and split or separately review the ACP environment change against the current ACP spawn path.
