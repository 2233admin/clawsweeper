## Summary

Keep this PR open. Current main still lacks the proposed Pi tool adapter conformance harness, and the adapter still does not pre-check already-aborted execute signals before agent-tool hook execution or client-tool delegation. The PR remains useful, but should be revised before merge to address the abort-ordering review feedback, preserve current client-tool parameter coercion, and use the repo-declared TypeBox package import.

Keep this PR open for contributor or maintainer revision.
