- **[P2] Reuse the selected routed provider config:** `extensions/speech-core/src/tts.ts:1830`
  - body: After `resolveFirstRoutableText` has already selected a routable provider, this line re-resolves config for `effectiveProvider` outside the preflight try/catch. If the primary provider config throws or was skipped but a fallback was selected, TTS can still abort or route with the wrong model gate; carry `{ provider, providerConfig }` from the selected route instead.
  - confidence: 0.86
- **[P2] Preserve line-start markdown reference pairs:** `src/shared/text/emotion-tags.ts:196`
  - body: The reference-link guard only returns true when text precedes the first label on the line, so a valid line-start pair like `[fast][1]` is treated as emotion tags and stripped to `[1]` in `off`/`on` display paths. The branch already documents this in #72821, but the sanitizer should preserve numeric/non-emotion reference labels or use document-level reference definitions before merge.
  - confidence: 0.78
