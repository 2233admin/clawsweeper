## Summary

Current main still does not fully cover the Gemini plain-text/localized reasoning leak described here. It handles provider-marked Google thought parts, tagged `<think>/<final>` cleanup, explicit `isReasoning` payloads, and some exact `Reasoning:` channel handling, but ordinary Gemini text with broader reasoning preambles such as `Thinking Process:`, `Analysis:`, `Tankegang:`, `Analyse:`, or `Begrunnelse:` is not classified before delivery.

## Best Possible Solution

Keep this issue open as the Gemini plain-text reasoning leakage tracker. Review and land PR #60551 or a replacement that adds a conservative shared outbound/message-tool sanitizer with tests for untagged planning prose, appended reasoning blocks, multilingual reasoning headers, tagged fragments, code/prose pass-through, and normal `/reasoning` visibility; coordinate with #72363 for shared leak handling.
