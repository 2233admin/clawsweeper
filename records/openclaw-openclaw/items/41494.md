## Summary

Current main still does not fully solve the reported Gemini plain-text reasoning leak. It handles provider-marked Google thought parts and XML-style reasoning/final tags, but there is no merged sanitizer for unmarked or localized reasoning preambles such as `Thinking Process:`, `Analysis:`, `Tankegang:`, or the reported SOUL/rule narration. Open PR #60551 is still open and directly targets this remaining gap, so the issue should stay open until that PR or a replacement lands.

## Best Possible Solution

Keep this issue open as the broader Gemini/plain-text reasoning leakage tracker. Review and land PR #60551 or a replacement that adds a conservative shared sanitizer for untagged reasoning/planning preambles before outbound delivery, with regression tests for the reported Gemini examples, localized headers, tagged fragments, code/prose pass-through, normal `/reasoning` visibility, and Telegram/message-tool delivery paths. PR #48593 can remain a narrower adjacent tag-fragment fix, but it does not close the broader preamble leak by itself.
