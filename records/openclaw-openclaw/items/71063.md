The PR updates `createChildAdapter` to guard non-writable stdin, consume initial stdin pipe errors, catch initial write/end failures, and add unit tests for non-writable stdin plus shell-wrapped async `EPIPE`.

## Best Possible Solution

## Solution Assessment
