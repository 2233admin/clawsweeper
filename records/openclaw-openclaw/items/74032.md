## Summary

Keep open. Current main has topic-aware delivery for Telegram native command replies, but it still registers and clears Telegram bot commands only in the default command scope, while the related open PR #6457 targets the remaining group-scope/topic-command behavior described in the report.

The issue reports that Telegram `/status` native slash commands do not reliably answer inside group forum topics even when normal mention-triggered replies work with `requireMention: false`.

## Best Possible Solution
